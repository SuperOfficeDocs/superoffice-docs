# CEP 10.1.3 update

## Chat – Configuration options for styling chat widget

When setting up a chat channel, it's important to be able to fine tune different fonts, colors and buttons. With these options you can fine tune the chat layout:

* Add more options to configure styling for Chat widget.
* Cursor: pointer when hovering over chat widget.
* Possible to have different colors on header and chat messages.
