# Mobile CRM 11.1.0 update

## New mobile experience

**SuperOffice Mobile CRM** puts your complete customer relationship system in your pocket. Access customer information, manage activities, and close deals - **from anywhere**.

Following our web interface update, we have enhanced the mobile experience to make daily CRM tasks more **efficient** while maintaining all the functionality you rely on.

![New design for Mobile CRM - the menu wheel -app-screen][img1]

### Key improvements

* Scroll to access data, replacing tabs with compressed views.
* Split company/contact views to match the SuperOffice CRM experience.
* Thumb-friendly navigation for quicker access to key information.
* New screen for adding participants to meetings
* Improved readability and spacing to reduce eye strain.
* Updated colors and icons aligned with our latest design.
