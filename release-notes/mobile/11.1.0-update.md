# Mobile CRM 11.1.0 update

## New mobile experience

**SuperOffice Mobile CRM** puts your complete customer relationship system in your pocket. Access customer information, manage activities, and close deals - **from anywhere**.

Following our web interface update, we've now enhanced the mobile experience to make daily CRM tasks more **efficient** while maintaining all the functionality you rely on.

![New design for Mobile CRM - the menu wheel -app-screen][img1]

### Key improvements

* Reduce eye strain with improved readability and spacing.
* Access data by scrolling instead of hidden behind tabs.
* Split company/contact views matching SuperOffice CRM experience.
* Find information faster with intuitive thumb-friendly navigation.

![Company screen with scroll option -app-screen][img2]

### Visual improvements

We've upgraded the overall visual experience with:

* Compressed views instead of tabs
* Thumb-friendly navigation
* New screen for adding participants
* Visual improvements:
  * Improved readability and spacing for less eye strain
  * Split company/contact views matching the updated web experience
  * New colors and icons consistent with our latest design
