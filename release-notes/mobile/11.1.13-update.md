# Mobile CRM 11.1.13 update

## Request - View request relations

There can be both hierarchical and side-by-side relationships (related, parent, or child), that helps you to understand dependencies, track progress and manage complex service scenarios with confidence. This gives you greater control, better visibility and a smoother way to work with interconnected requests.

In the tab for Related Request you can see the relation type and the related request(s).

![Request with related requests shown as hierarchy or side-by-side -app-screen][img1]

> [!NOTE]
> You can only view request relations in Mobile CRM, to edit, add or delete request relations you need to go to SuperOffice CRM.
