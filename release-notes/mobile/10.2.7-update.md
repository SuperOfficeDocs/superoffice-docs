# SuperOffice Mobile CRM - 10.2.7 update

## Custom tasks to run a CRMScript in Mobile CRM

Custom tasks running CRMScript that are defined in SuperOffice CRM can also be triggered from the **Task** menu in Mobile CRM if **Shown in Mobile CRM** is checked ON for the [Task menu list][2] in Settings and maintenance.

![Task menu on Appointment with 2 custom tasks -app-screen][img1]

Make sure to log out and in again in Mobile CRM to make sure all your caches are updated after you have created a new Task menu definition in Settings and maintenance.

Read more in the [Core CRM][1] section and how to set it up in [Settings and maintenance][2].
