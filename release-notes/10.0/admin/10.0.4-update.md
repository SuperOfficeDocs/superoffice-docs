# Settings and maintenance 10.0.4 update

## Document list – Template variable for semi-long date (10.0.4)

A new template variable is added for long dates from the gregorian calendar. `[dat1]`, format **d.mmm yyyy**.

![Template variable for semi-long date -screenshot][img1]

The month will of course be written in the local language, so it will be correct for each language.
