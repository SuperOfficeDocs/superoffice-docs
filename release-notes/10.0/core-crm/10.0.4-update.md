# SuperOffice Core CRM 10.0.4 update

## Dashboards – Navigation preview with own dashboards

When you have created a lot of different dashboards, and some are either not pinned or set as favorites, it's hard to remember what you called them.

![Navigation preview with own dashboards -screenshot][img1]

On the navigation for Dashboard, you have a section for Own dashboards, and of course, selecting that will show your own dashboards. It's easy to navigate and easy to update the different dashboards if you want.

[Read more about Dashboards on our Community][1].
