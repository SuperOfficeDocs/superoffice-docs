# Service 11.6 update

## GUI Web panel - Option for always reload content on tab navigation

If you create a GUI web panel and configure it to be visible in Panel - Request screen section tab > the 'Always reload content on tab navigation' is available.

## Trigger - Load request message - Make it possible to evaluate how message was created

You now have more control of how to use "load request message" trigger.

## Admin Request Preferences

This means that in Settings and maintenance you can assign a preference for requests on our standard logic for preferences based on a System/User group/User level from within Settings and maintenance.
