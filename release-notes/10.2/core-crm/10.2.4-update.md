# Core CRM 10.2.4 update

## Remove of Old Dashboard

For the last months all users have seen a message that the old dashboard will be removed in end of March 2023, and now the time has come. The old dashboards are no longer available, and are removed for all users.

From now on all users has to use the existing dashboard functionality. Read more about [Dashboard][1] in Learn.
