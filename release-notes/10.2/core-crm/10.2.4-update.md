# Core CRM 10.2.4 update

## Remove of Old Dashboard

For the last months, all users have seen a message that the old dashboard will be removed in end of March 2023. Now, the old dashboards have been removed for all users and are no longer available.

From now on, all users should use the existing dashboard functionality. Read more about [Dashboard][1] in Learn.
