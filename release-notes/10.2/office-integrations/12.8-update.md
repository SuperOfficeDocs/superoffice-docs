# WebTools for SuperOffice update

When you open and edit a document/file/e-mail  in SuperOffice, your browser gives you a dialog with a **security question** about the file download. To open the file you need to select the **Open/Allow** -button.

There is a setting for **Document editing** in the **Local settings** dialog found under **Personal settings**. Read more about it for the [changes on Core CRM][1].

Any issues using WebTools, go and see these **FAQs**:

* [Browser security question pop-up][2]
* [Can't open a document or an e-mail][3]

## WebTools for Windows 12.8 update
