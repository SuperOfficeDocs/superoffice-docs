# Office integration 10.2.10 update

## Wizard for SharePoint integration

If you are using Sharepoint as a document handler, you can set up SuperOffice to store all files in your own SharePoint Online repository.

It's easy to set up the integration with the [five-step configuration ​wizard][1]. Read more about the [SharePoint integration][2] for the SuperOffice document library.
