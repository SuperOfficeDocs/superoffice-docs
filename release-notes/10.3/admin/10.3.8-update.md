# Settings and maintenance 10.3.8 update

## CRMScript with Monaco editor

When writing CRMScripts, getting extra support can be crucial.

The Monaco editor is user-friendly and provides enhanced IntelliSense for both JavaScript, TypeScript, and CRMScript, making it easier to write and debug your code.

![CRMScript - Intellisense with the Monaco editor -screenshot][img1]

Learn more about [CRMScript Intellisense][1].
