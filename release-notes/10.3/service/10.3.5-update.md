# Service 10.3.5 update

## Request types - Fine tuning of requests for different types

To make the use of request types more versatile you can enable some different settings for each request types.

To be able to add the request types to the **+New** menu in SuperOffice CRM, just click ON the Show in new check box.

You also have Yes/No settings for:

* Exclude signature
* Exclude email recipients
* External access level as default
