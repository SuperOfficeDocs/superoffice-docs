# Core CRM 10.3.1 update

## Dashboard - New Requests dashboard and tiles

When you are working with Service and requests, it's often needed to have some overview of your response and closing time. With the new default dashboard for **Requests** you get that overview with multiple request tiles to show you analytics for your request status.

Go to Dashboard and search for Requests:

![Dashboard - Search for Requests dashboard -screenshot][img1]

And get the overview of your request status.

![Dashboard - Default Requests dashboard with -screenshot][img2]

## Dashboard - New standard Requests tiles

To make sure you have the tools to create a good request dashboard, we have added new standard tiles:

* My average response time (1x1)
* My average closing time (1x1)
* Average response time by category (2x2)
* Average closing time by category (2x2)
* Average closing time by user (2x2)
* Requests created this month (2x2)

![Dashboard - Standard Request tiles -screenshot][img3]
