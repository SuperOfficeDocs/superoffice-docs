# superoffice-docs

**This repository is frozen as of Friday, July 10, 2026.**
Documentation content is moving to [SuperOfficeDocs/docs](https://github.com/SuperOfficeDocs/docs), our new Mintlify-based platform. This repo is kept read-only for history and reference — please do not open new issues or PRs here.

Note that the new repo is not yet open for external issues or PRs either; that opens at go-live (targeted September 1, 2026).

Contains core content that makes [https://docs.superoffice.com](https://docs.superoffice.com), the place for all technical documentation for SuperOffice products.

All content source is organized into an intuitive folder structure. This is intentional to help facilitate collaboration from both SuperOffice employees and customers.
