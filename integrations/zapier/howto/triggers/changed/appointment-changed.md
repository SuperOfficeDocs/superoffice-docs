# Create a SuperOffice CRM Appointment changed trigger

[!include[note](../../includes/note.md)]

## Steps

[!include[create zap step](../../includes/create-zap-intro.md)]

1. Click on **Trigger** step to choose the event that will start a Zap. You will be taken to a window from which you can select the application you want to use. Search for and click on SuperOffice.

    ![Zapier: Create a trigger -screenshot][img2]

1. Search for and click on SuperOffice.

    ![Zapier: Select a trigger -screenshot][img3]

1. A side menu will open, in the **Trigger event** section, select **Appointment Changed**, then click the **Account** section and login using SuperOffice account.

    ![Set up a trigger -screenshot][img4]

1. [!include[test](../includes/test.md)]

    ![Test an action -screenshot][img5]

1. Click on **Action**. You will be taken to a window from which you can select the application you want to use. It can be an action in SuperOffice or within any other application Zapier supports.

    ![Select an action -screenshot][img6]
