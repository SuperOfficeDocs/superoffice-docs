# Known limitations in SuperOffice for Teams

* **Bot scope:** The bot only handles sign-in and sign-out. It doesn't answer general questions.
* **File selector:** Supports single-file selection per save. Multi-select is planned.
* **No overwrite:** Saving the same file again always creates a new document.
* **Long chats:** Very long chat histories can impact performance. Limits may be introduced.
* **Private and shared channels:** Not supported due to Microsoft Teams platform restrictions (subject to change by Microsoft).
* **Unmanaged users:** Archiving is not supported in 1:1 chats with personal/unmanaged accounts.
* **File archiving in chats:** File uploading and attaching is disabled in chat contexts by SuperOffice for Teams.

> [!NOTE]
> This documentation reflects the current pilot version of SuperOffice for Teams. Features and behavior may change based on feedback and future releases.
