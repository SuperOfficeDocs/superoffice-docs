# Requirements

Before you can use SuperOffice for Teams, make sure the following requirements are met.

## User requirements

* Microsoft Teams (desktop or web).
* A SuperOffice Online user account with permission to create activities and documents.

## Administrator requirements

* A Microsoft 365 or SuperOffice administrator must register the tenant and grant admin consent before users can archive content.

See [Deploy SuperOffice for Teams][1] for the full setup steps.
