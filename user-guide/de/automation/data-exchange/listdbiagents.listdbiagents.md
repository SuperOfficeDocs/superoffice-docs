# Agents

This screen contains a list of any existing Data Exchange (database integrator) agents. SuperOffice Service can be connected to one or more agents in order to synchronise data in SuperOffice Service with other databases.

* **New agent**: Click here to create a new agent.
* **Name**: Displays the agent name.
* **Protocol**: Displays the communication protocol used to connect to the agent.
