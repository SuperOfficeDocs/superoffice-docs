Om SuperOffice CRM for Web är installerat på en HTTPS-sida fungerar inte webbpaneler som använder webbadresser med HTTP. Du måste använda webbadresser med HTTPS.

Du kan öppna webbadressen i felsäkert läge i webbläsaren, men då är sidan inte säker längre.

Om SuperOffice CRM for Web är installerat på en HTTP-sida använder du "vanliga" webbadresser med HTTP.
