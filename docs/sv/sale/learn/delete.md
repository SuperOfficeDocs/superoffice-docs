# Ta bort en försäljning

Det kan hända att du vill ta bort en försäljning om det inte längre är aktuellt att spara den i databasen.

1. Gå till den önskade försäljningen.
2. Klicka på knappen **Uppgift** (![ikon][img1]).
3. Välj **Ta bort "försäljningsnamn"**.
4. Bekräfta med **Ja**.
