# Uppskjutna, sålda och förlorade försäljningar

I fältet **Fas** i fönstret Försäljning, kan du ange att försäljningen är **Såld** eller **Förlorad**. Du kan också markera kryssrutan **Uppskjuten** under det här fältet om du vill pausa försäljningsprocessen. Om du gör detta, ändras fälten nedan tillsammans med ikonen längst upp till vänster i försäljningsfönstret.

För enskilda försäljningstyper visas även en lista med faser,[!include[SM](../../learn/includes/as-defined-sm.md)] [Läs mer om säljguider][1].

## Uppskjuten

Om du ställer in försäljningen på **Uppskjuten**, visas följande fält:

* **Återöppningsdatum**: Här kan du ange datum då försäljningen ska återöppnas.
* **Orsak (uppskjuten)**: Här kan du ange orsaken till uppskjutandet.

![Uppskjuten försäljning -screenshot][img5]

Dialogrutan **Bokning** öppnas när du klickar på **Spara**. Här kan du boka kommande möten eller samtal för att följa upp den uppskjutna försäljningen.

## Sålt/Förlorat

Om du ställer in försäljningen på **Såld** eller **Förlorad**, kan du ange datum, eventuell konkurrent och varför försäljningen lyckades eller misslyckades.
