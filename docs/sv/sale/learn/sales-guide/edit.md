# Redigera säljguide

[!include[Must be admin](../../../learn/includes/req-admin.md)]

1. Klicka på knappen **Arbetsflöde** i navigatorn (![ikon][img1]).
2. Välj fliken **Säljguide**.
3. Välj önskad försäljningstyp i listrutan **Försäljningstyp**.
4. Klicka på **Redigera**.
5. [Redigera de enskilda faserna][3].
6. Redigera händelser och dokument genom att dubbelklicka på dem i listorna längst ned.

    * [Lägg till en händelse för en fas][1]
    * [Lägg till ett dokument för en fas][2]

    Ändringarna sparas automatiskt.
