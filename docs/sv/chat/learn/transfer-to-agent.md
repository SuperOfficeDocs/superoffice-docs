# Överföra ett chattsamtal till en annan agent

Ibland kan en handläggare inte fortsätta att chatta med kunden och lösa det aktuella problemet. I sådana situationer kan chatten enkelt överföras till en annan handläggare som sömlöst fortsätter att kommunicera med kunden och ger svar utan fördröjning.

Du kan överföra chattsamtal till andra agenter och ta emot överförda chattsamtal från andra agenter.

> [!NOTE]
> En chef kan också ta över chatten eller överföra den till en annan handläggare. På så sätt kan en person med administratörsbehörighet se till att kunden får en handläggare som är tillgänglig och som kan ge kunden rätt hjälp.

## Så överför du en chatt

1. Klicka på <i class="ph ph-list" aria-label="Task menu"></i> och välj **Överför**.
2. Välj en agent i listan **Överför till**. Endast agenter med status **Närvarande** visas här.
3. Klicka på **Överför**. Chattsamtalen markeras i listan. Nu händer något av följande:
    * Om den mottagande agenten accepterar det överförda chattsamtalet går du in i lysningsläge. Du kan visa det pågående chattsamtalet men du kan inte svara på meddelanden. Du stänger chattsessionen genom att klicka på <i class="ph ph-list" aria-label="Task menu"></i> och välja **Stäng fönster**.
    * Om den mottagande agenten avvisar det överförda chattsamtalet visas det inte längre som markerat i sessionslistan. Du kan fortsätta chattsamtalet eller [skapa ett ärende][1] som ska hanteras av någon annan.
    * Om den mottagande agenten inte svarar klickar du på **Avbryt överföring**. Du kan fortsätta chattsamtalet eller [skapa ett ärende][1] som ska hanteras av någon annan.

> [!TIP]
> Om du har behörigheten **Kan lyssna** kan du gå till **Dashboard**-fliken och dubbelklicka på en chattsession för att visa den.

## Acceptera ett chattsamtal som har överförts från en annan agent

Om en agent överför ett chattsamtal till dig så visas samtalet automatiskt i chattpanelen och markeras även i sessionslistan.

1. (om relevant) Välj det överförda chattsamtalet i sessionslistan för att visa den.
2. Klicka på **Acceptera överföring** (nedanför chattsamtalen) om du vill acceptera samtalet. Om du inte vill acceptera chattsamtalet klickar du på **Avvisa överföring**.
