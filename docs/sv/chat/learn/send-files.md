# Skicka filer i chatt

Du kan skicka SuperOffice-dokument och andra filer till kunden.

1. Klicka på <i class="ph ph-paperclip" aria-label="Attachments"></i> på nedre delen av chattpanelen.
2. Gör något av följande om du vill skicka en fil:
    * Klicka på **Välj ett CRM-dokument** och sök efter och välj SuperOffice-dokumentet som du vill skicka.
    * Klicka på **Ladda upp filer (eller dra och släpp här)**, välj filen och klicka på **Öppna**.
    * Dra och släppa filer från Windows Explorer till **Ladda upp filer (eller dra och släpp här)**. Kontrollera att **→ Flytta** visas bredvid muspekaren när du släpper filen (inte **+ Kopiera**).

Filen/dokumentet skickas till kunden på en gång.
