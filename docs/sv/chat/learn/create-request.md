# Skapa ärende i en aktiv chatt

Om du inte kan lösa problemet via chatten kan du skapa ett ärende som hanteras av en annan specialist.

1. Klicka på ![ikon][img1] och välj **Skapa ärende**.

    ![Klicka på åtgärdsknappen och Skapa ärende så sparas all information i chatten i ärendet -screenshot][img2]

2. Fyll i relevanta fält i dialogrutan.
3. Chattloggen läggs till i ärendet, men du kan lägga till mer information här om det behövs.
4. Klicka på **Spara**.
5. Ett kort meddelande som innehåller ärende-ID läggs till i textfältet i chatten. Klicka på **Skicka** för att skicka meddelandet till kunden.
