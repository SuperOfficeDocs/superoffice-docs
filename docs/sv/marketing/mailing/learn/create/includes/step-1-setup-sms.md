Det första steget för att skapa ett SMS-utskick är att **konfigurera** utskicket. Här kan du ange SMS-utskickets namn, avsändaren och arkiveringsalternativ för utskicket.

1. Ange ett namn i fältet **Namn för utskick**.

2. Ange avsändarens namn eller mobiltelefonnummer i fältet **Från**. Om du vill att mottagarna ska svara på SMS-meddelandet måste du ange ett telefonnummer.

3. Välj en mapp som du vill placera utskicket i från listan **Mapp**.

    [Hur skapar jag en utskicksmapp?][19]

4. I fälten **Urval** och **Projekt** kan du associera utskicket med ett urval och/eller ett projekt

    [!include[Start typing to search](type-to-search.md)]

5. Klicka på **Nästa** när du vill gå vidare till nästa steg.
