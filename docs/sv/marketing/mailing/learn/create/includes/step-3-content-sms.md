I steget **Innehåll** kan du ange SMS:ets innehåll.

1. Ange texten i fältet **SMS-text**. Du kan använda [mallvariabler för kontakter][1] här. Detta fungerar på samma sätt som för e-postutskick. Se [Använda mallvariabler i meddelanden][2].

2. Klicka på **Nästa** när du vill gå vidare till nästa steg.

> [!NOTE]
> Antalet tecken (inkl. mellanrum) visas i fältet **Antal tecken**. Teckenbegränsningen för 1 SMS-meddelande är 160 tecken. Om ditt meddelande överskrider 160 tecken skickas det som 2 eller fler meddelanden.
