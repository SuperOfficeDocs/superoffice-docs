# Analysera dina utskicksresultat

För att avgöra om ett utskick var framgångsrikt behöver du siffror att jämföra med. Lär dig hur du analyserar din e-poststatistik i den här guiden.

Visste du att den genomsnittliga öppningsfrekvensen för ett utskick i alla branscher är 21,33 procent?

Det innebär att generellt öppnar och läser 1 av 5 mottagare ditt utskick. För att se om ditt utskick ligger över eller under det här genomsnittet kan du analysera din utskicksstatistik per enskilt utskick.

## Analysera din statistik

SuperOffice samlar all statistik och alla resultat för varje utskick du skickar. Genom att jämföra resultaten av olika utskick kan du se vilken ämnesrad, typ av innehåll och länkar som fungerar bäst för att engagera och intressera dina mottagare. Den här statistiken ger dig den information du behöver för att förbättra dina framtida utskick.

SuperOffice Marknadsföring mäter effekten av ditt utskick direkt. Du kan se öppningsfrekvensen, klickfrekvensen för länkar och vilka e-postmeddelanden som har returnerats.

## Bestäm ditt benchmark för framgång

När är ditt utskick framgångsrikt? Den här frågan kan vara svår att svara på om du inte har skickat utskick till dina kontakter med en professionell e-postlösning. Därför är det en bra idé att skicka ut några "testutskick" från din e-postlösning för att fastställa ditt benchmark.

För att förbättra dina framtida utskick måste du ta reda på vad som fungerar för dina kontakter. Det gör du enklast genom att dela upp målgrupperna för dina första utskick i två eller flera grupper. Skicka sedan samma utskick, där du ändrar ämnesraden, din uppmaning eller andra delar av utskicket.

Statistiken från varje utskick visar vilket utskick som gav bäst resultat och ger dig ditt benchmark. Sedan kan du jämföra resultatet för eventuella framtida utskick med det benchmark du har ställt upp.

## Steg

Om du vill se statistik om ditt utskick kan du gå till fliken **Utskick** och klicka på utskicket i listan. Skärmen **Visa utskick** visas.

> [!TIP]
> Klicka på knappen **Uppdatera** om du vill uppdatera utskicksstatistiken.

När ditt utskick är klart kan du se hur vilket resultat ditt utskick gav på fliken **Sammanfattning** .

![När utskicket har skickats kontrollerar du fliken Sammanfattning för att få en översikt över hur bra utskicket gick -screenshot][img2]

## Flikar med utskicksstatistik

* **Sammanfattning**: Här hittar du olika typer av statistik för utskicket:
  * **Antal skickade**: Hur många utskick som skickades.
  * **Andel öppna**: Hur många utskick som öppnades.
  * **Klickfrekvens**: Hur många länkar som klickades.
  * **Avvisningsfrekvens**: Hur många utskick som [returnerades][1].

* **Innehåll**: Här visas innehållet i utskicket.

* **Rapporter**: Fyra diagram visar statistik för utskicket. (Håll musen över om du vill se exakta siffror).
  * Alla länkar som klickades på och hur många mottagare som klickade på varje länk du använde. Hur många mottagare som öppnade ditt utskick under de första 48 timmarna och när de flesta läser det.

  * Dessutom visar SuperOffice Marknadsföring antalet mottagare baserat på deras status. Det innebär att du bland annat kan se hur många mottagare som läst utskicket, hur många som utskick som returnerats och hur många dubbletter ditt utskick innehöll.

* **Mottagare**: Denna flik visar en [mottagarlista][2] för utskicket. Utskickets status (skickar, väntar, misslyckades, returnerat, läst osv.) för varje individuell mottagare visas här. Du kan filtrera översikten genom att markera kryssrutorna längst ned på skärmen. Du kan även välja och lägga till mottagare i urval och projekt med hjälp av menyknappen.

* **Spårade länkar**: Denna flik visar [antalet klick på varje spårad länk][3] i utskicket. Klicka på en länk för att gå till fönstret **Visa länk**. Du kommer att se en lista över mottagarna som har klickat på länken och när de klickade på den. Använd denna lista för att göra ett urval eller ett projekt för vidare uppföljning.

* **Hantera studsar**: En översikt över alla returnerade e-postmeddelanden. Använd åtgärderna för att [rensa dina returnerade e-postadresser][1], antingen genom att ställa in kontakterna som tidigare anställd eller ta bort e-postadresserna .

> [!NOTE]
> Flikarna **Rapporter** och **Spårade länkar** kräver användarlicensen Marknadsföring. En del flikar finns inte tillgängliga för SMS-utskick och dokumentutskick.
