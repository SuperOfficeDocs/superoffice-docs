# Spåra formulärinlämningar och visa statistik

Om du vill se statistik om ditt formulär kan du gå till fliken **Formulär** och klicka på formuläret i listan. Skärmen **Visa formulär** visas.

Du kan hitta formulärstatistik på följande flikar.

## Sammanfattning

Här kan du hitta grundläggande information om formuläret, inklusive en förhandsgranskning av formuläret, och visa inlämningsstatistik:

* **Totalt antal inlämningar**: Hur många formulär har lämnats in.
* **Sista inlämning**: När det sista formuläret lämnades in.
* **Ifyllda de senaste 7 dagarna**: Inlämnade formulär per dag de senaste 7 dagarna.
* **Ifyllningsdatum**: Status för inlämnade formulär.
* **Utskick**: Det här fältet visas om du har aktiverat **E-postsvar** för det här formuläret. [Visa information och statistik om formulärsvaret][2].

## Rapporter

Dessa diagram visar statistik för formuläret. (Håll musen över om du vill se exakta siffror.)

## Formulärinlämningar

Här kan du visa och behandla alla inlämnade formulär. En röd ikon på fliken anger nya formulärinlämningar ![ikon][img1].

Använd filteralternativen längst ned i listan om du bara vill [visa inlämningar med specifik status][3].
