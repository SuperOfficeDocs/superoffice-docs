# Visa formulär

På fliken **Formulär** kan du hitta alla befintliga formulär.

Om du vill visa ett befintligt formulär klickar du på ett formulär i listan. Skärmen **Visa formulär** visas.

## Visningsalternativ

* **Senaste**: Visar de formulär som du nyligen har arbetat med.
* **Mina formulär**: Visar de formulär som du har skapat.
* **Aktiva formulär**: Visar bara aktiva formulär. Formulär kan inaktiveras manuellt eller inaktiveras automatiskt ett visst datum eller när ett visst antal inlämningar tagits emot.
* **Alla formulär**: Visar alla tillgängliga formulär.
* **Sök i formulär**: Ange en sökfras och tryck på **ENTER** eller klicka på sökikonen ![ikon][img2].

## Fönstret Visa formulär

Överst på skärmen finns följande alternativ:

* **Inaktiv**/**aktiv**: Klicka på skjutreglageknappen om du vill aktivera eller inaktivera formuläret.
* **Uppdatera**: Klicka på denna knapp för att uppdatera skärmen.
* **Redigera**: Klicka på denna knapp för att redigera formuläret.
* ![ikon][img1]: Klicka på denna knapp för att ange formuläret som [favorit][1].

## Flikar

På skärmen **Visa formulär** finns följande flikar:

### Sammanfattning

Här kan du hitta grundläggande information om formuläret, inklusive en förhandsgranskning av formuläret, och visa inlämningsstatistik:

* **Totalt antal inlämningar**: Hur många formulär har lämnats in.
* **Sista inlämning**: När det sista formuläret lämnades in.
* **Ifyllda de senaste 7 dagarna**: Inlämnade formulär per dag de senaste 7 dagarna.
* **Ifyllningsdatum**: Status för inlämnade formulär.

Bredvid **Utskick** finns en länk till det formulärsvarsutskick som används med det här formuläret. Klicka på länken om du vill visa information och statistik om formulärsvaret. Se [Spåra utskick och visa statistik][2].

### Publicera

Här hittar du länken och skriptet för [publicering av formuläret][3] på en webbsida eller i ett utskick.

Klicka på knappen **Öppna i nytt fönster** om du vill förhandsgranska formuläret.

**Visa med förifyllda fält**: Välj det här alternativet om du vill lägga till fältnamn till länken och JavaScript. Genom att lägga till mallvariabler i formulärlänken i utskick kan formulärfälten [fyllas i med befintliga uppgifter som namn och e-post][4] för registrerade kontakter.

Till vänster på skärmen kan du välja olika typer av publiceringsalternativ för formuläret.

### Rapporter

Diagram visar statistik för formuläret. (Håll musen över om du vill se exakta siffror.)

### Formulärinlämningar

Här kan du visa och [behandla alla inlämnade formulär][5]. Använd filteralternativen längst ned i listan om du bara vill visa inlämningar med specifik status.

### Svars-e-post

Fliken visas om **E-postsvar** är aktiverat för det här formuläret (se [Skapa ett nytt formulär – Åtgärder][6]). Här kan du förhandsgranska formulärsvaret (utskicket) som associeras med det här formuläret.
