# Arbeta med formulär

Formulärfunktionen är en kraftfull motor för att få information från ett formulär till SuperOffice utan alltför mycket integrationer och krångel. Med hjälp av formulär blir det lättare att interagera med kunder och prospekt via din webbplats eller via kundcentret. En stor mängd data kan läggas till i ett formulär (inklusive filöverföring) och du kan lägga till automatiserade åtgärder som ska utföras när formuläret lämnas in och behandlas.

Det är enkelt att skapa ett formulär och lägga upp det på en webbsida som du vill att det ska visas på, t.ex. "Kontakta mig"-formulär, och det är lätt att omvandla alla dessa nya potentiella kunder till kontakter i SuperOffice.

Exempel på formulär: påbörja kostnadsfri provperiod, kontakta oss, uppdatera data, samtyckeshantering.

## Vad kan jag använda webbformulär till?

Du kan använda webbformulär för att samla in dina webbplatsbesökares uppgifter – deras namn, e-postadress, preferenser, kommentarer och återkoppling och eventuellt omvandla dem till kunder. De data du får in via webbformulär kan importeras automatiskt till din SuperOffice CRM-databas, vilket sparar tid och arbete.

* **Inhämta kontaktuppgifter:** Informationen som skickas in via webbformulär sparas i SuperOffice CRM så att du kan följa upp dem senare.

* **Få återkoppling från kunder:** När kunder eller webbplatsbesökare har en fråga, ett problem eller en förfrågan kan de kontakta dig direkt via webbformulär.

* **Inhämta samtycke:** När en person ger dig tillstånd att lagra information om dem i ett webbformulär vet du att de verkligen är intresserade av dina marknadsföringsmeddelanden. Det gör inte bara att dina e-postkampanjer blir mer framgångsrika, utan du kommer också att följa de regler som anges i GDPR.

* **Automatiska händelser:** Formulär hjälper dig att påskynda ditt arbete genom att automatisera manuella uppgifter, till exempel skapa uppföljningsaktiviteter, göra utskick, uppdatera kontakternas intressen och lägga till kontakter i ett urval eller ett projekt.

## Var hittar du formulär?

Det finns flera sätt att hitta formulär. Gå till fliken Formulär, formulärmallar eller fliken Formulärinlämningar.

![Det finns flera sätt att hitta formulär -screenshot][img2]

* **Formulär:** här kan du skapa nya formulär och se alla formulär som tidigare har skapats. På fliken Aktiva formulär visas vilka formulär som används för närvarande. Den röda bubblan i det övre högra hörnet i ett formulär visar antalet formulärinlämningar som inte har behandlats ännu.

* **Formulärmallar:** här hittar du alla tillgängliga formulärmallar. När du öppnar den här fliken första gången hittar du olika mallar i online-mallbiblioteket. Du kan också importera och använda mallar från online-mallbiblioteket eller så kan du använda dem som inspiration för att skapa egna. Klicka bara på den mall du gillar och tryck på Importera mall för att ladda ned den.

* **Formulärinlämningar:** visar all information om de kontakter som har skickat ett formulär och vad deras aktuella formulärstatus är.

## Arbetsflöde

När du skapar ett formulär följder du detta arbetsflöde:

1. [Skapa nytt formulär][2].
2. [Formulärsvar – Steg 1: Konfiguration][3] – ett e-postmeddelande kan skickas till en avsändare automatiskt när deras formulär behandlas.
3. [Publicera formulär][4] – använd en länk eller JavaScript om du vill publicera formuläret i ett nyhetsbrev eller på en webbplats eller ett kundcenter.
4. [Spåra formulärinlämningar och visa statistik][5] – visa svarsfrekvensen i realtid.
5. [Behandla formulärinlämningar][6] – uppdatera SuperOffice med inlämnade data, antingen manuellt eller automatiskt. I SuperOffice kan du identifiera befintliga kontakter baserade på e-postadresser eller skapa nya kontakter och företag.

## Använd dashboards för att få en översikt över formulärinlämning

Att ha en dashboard för formulärinlämning gör det enkelt att mäta och visualisera deras data. Med hjälp av dashboarden kan du hålla koll på resultat, trender och produktivitet. Läs mer om dashboards för formulärinlämning.
