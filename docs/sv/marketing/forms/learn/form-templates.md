# Arbeta med formulärmallar

## Använd mappar för att organisera saker och ting

[Hur skapar jag en utskicksmapp?][19]

## Använda formulärmallar

1. Välj fliken **Formulärmallar**.
2. Klicka på **Alla mallar**.
3. Klicka på <i class="ph ph-list" aria-label="Task menu"></i> bredvid den mall som du vill använda.
4. Välj **Kopiera till nytt formulär**. Mallen öppnas i dialogrutan **Redigera formulär**. Se [Skapa nytt formulär][1].

## Skapa en helt ny formulärmall

1. Gå till fliken **Formulärmallar**.
2. Klicka eventuellt på **Lägg till mapp** för att skapa en mapp för mallen.
3. Klicka på **Lägg till formulär**. Skärmen **Redigera formulär** visas.
4. Ange ett beskrivande namn på mallen i fältet **Formulärnamn**.
5. Fyll i dialogrutan **Redigera formulär** enligt beskrivningen i [Skapa ett nytt formulär][1].
6. Klicka på **Spara** när du är klar. Den nya formulärmallen visas då i mallistan.

## Skapa en mall utifrån ett befintligt formulär

1. Gå till fliken **Formulär**.
2. Klicka på <i class="ph ph-list" aria-label="Task menu"></i> bredvid det formulär som du vill använda som utgångspunkt för mallen.
3. Välj **Kopiera till mall**.
4. Ange i fönstret **Redigera formulär** ett beskrivande namn på mallen i fältet **Formulärnamn**.
5. Klicka eventuellt på **Mapp** för att välja en mapp för mallen.
6. [Gör de ändringar som behövs][1].
7. Klicka på **Spara**.
