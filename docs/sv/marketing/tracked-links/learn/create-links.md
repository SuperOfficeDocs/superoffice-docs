# Skapa spårade länkar

[!include[Requirement](includes/req-marketing.md)]

Du kan spåra webbadresser som infogats i e-postmeddelanden. Fliken **Spårade länkar** används för att administrera dessa spårbara länkar. De länkar som skapas här finns tillgängliga när du redigerar innehållet i ditt utskick.

> [!NOTE]
> Du kan även skapa nya länkar medan du redigerar innehållet.

**Steg:**

1. Välj fliken **Spårade länkar**.

2. Tillval: Välj en mapp eller skapa en ny mapp.

    [Hur skapar jag en utskicksmapp?][19]

3. Klicka på ![ikon][img1] knappen **Lägg till**. Du kommer då till skärmen **Egenskaper för länk**.

4. Ange ett beskrivande namn på länken i fältet **Namn**.

5. Under **Länkningsåtgärder** längst ned i fönstret kan du [välja vad som ska hända][1] när mottagarna klickar på länken i meddelandet.

6. Klicka på **OK** när du är färdig.

En liten prick på flikarna för länkningsåtgärderna anger vilka länkningsåtgärder som är aktiva. Flera länkningsåtgärder kan vara aktiva för en länk.
