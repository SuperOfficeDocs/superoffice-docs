# Klagomål om skräppost

Fliken **Klagomål om skräppost** visar alla mottagare som har rapporterat ett e-postutskick som skräppost. Ett klagomål om skräppost indikerar vanligtvis att en mottagare inte har gett sitt godkännande till att få den här typen av e-post från dig.

> [!NOTE]
> Ett klagomål om skräppost resulterar automatiskt i att eMarketing-samtycket tas bort.

## Varför rapporteras mina utskick som skräppost och vad kan jag göra?

Människor klickar på knappen skräppost av olika skäl. Mottagaren är av någon anledning osäker på vem du är, eller varför du mejlar till denne. I vissa fall rapporterar människor e-postmeddelanden som skräppost för att de inte hittar länken Avsluta prenumeration.

Om en mottagare använder Mailgun läggs e-postmeddelanden som rapporteras som skräppost till på listan "undertryckt". Detta innebär att våra användare inte längre kan skicka e-postmeddelanden till den personen. Detta leder i sin tur till att inga utskick skickas och att svar på pågående förfrågningar inte kan levereras, vilket kan leda till allvarliga problem.

### Checklista för att undvika klagomål om skräppost

* Är det ett specifikt utskick eller en viss utskicktsyp som våra prenumeranter reagerar på?
* Vem rapporterade utskicket som skräppost?
* Ser ditt utskick inte ut som det nyhetsbrev som personen prenumererar på?
* Går det att missförstå innehållet i utskicket?
* Är det inte uppenbart för mottagaren att utskicket kommer från dig?
* Skickade du utskicket till en lång lista med okända e-postadresser (som inte har registrerats i SuperOffice)?

## Exportera

Du kan exportera listan för vidare bearbetning.

1. Markera de e-postadresser som du vill exportera.
2. Välj **Åtgärder** > **Exportera**.
3. Välj ett exportformat. Exportfiltret visas längst ner i din webbläsare. Du hittar den även i mappen Hämtningar.
