# Lägg till kopplingsfält i meddelandet

Det finns inget bättre än att göra affärer med en personlig touch. Du kan anpassa din kommunikation genom att använda kopplingsfält (kallas även mallvariabler) för att infoga kundspecifik information i ditt utskick.

Kopplingsfält finns i många former: det kan vara kundens förnamn, deras företagsnamn, hälsningsfras, namnet på säljaren som kunden hade kontakt med etc. Med andra ord är kopplingsfält lättanvända verktyg för att anpassa innehållet för varje enskild kund.

![Anpassa utskicket genom att lägga till kopplingsfält i textfälten -screenshot][img2]

## Steg

1. Markera en text i meddelandet.

2. Skriv t.ex. "Hej!".

3. Placera muspekaren efter texten och klicka på **Kopplingsfält** i verktygsfältet.

4. I listan som visas väljer du **Kontakt** och **Förnamn**.

    ![Lägg till kopplingsfält i meddelandet -screenshot][img1]

5. Fortsätt med resten av texten och lägg in de kopplingsfält som du behöver. En lista finns [i variabelreferensen][2].

> [!TIP]
> Innan du skickar ett utskick bör du [kontrollera att kunduppgifterna är korrekt inlagda i meddelandet][1].
