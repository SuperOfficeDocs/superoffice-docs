* **Värd**: IP-adress eller värdnamn för den e-postserver som används för att skicka e-post. Servern måste vara tillgänglig från den SMTP-port som SuperOffice-programmen kör på.

* **Port**: Ange vilken SMTP-port SuperOffice-programmen kör på. Standardporten är 25.

* **Användarnamn** / **Lösenord**: Ange användarnamn och lösenord för servern.

* **Använd SSL**: Markera detta alternativ om du vill använda Secure Socket Layer vid inloggning.

* **Kontrollera SMTP-anslutning**: Om det här alternativet är markerat kommer systemet att försöka skicka ett e-postmeddelande till en adress på SuperOffice AS för att fastställa om SMTP-servern tillåter att du skickar ut e-post från din egen domän. Om du inte markerar detta alternativ görs inget försök.
