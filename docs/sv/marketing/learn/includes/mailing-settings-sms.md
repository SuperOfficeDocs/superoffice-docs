* **Modul**: Välj den SMS-leverantör som du har avtal med för att skicka SMS från SuperOffice Service.

* **Standardlandskod**: Ange en standardlandskod för det land som du befinner dig i. Till exempel +44 för Storbritannien. Denna landskod används för mobiltelefoni. Den används för utgående SMS (t.ex. när du inte anger landskoden för det mobiltelefonnummer som du skickar till) och inkommande SMS (så att SuperOffice Service kan känna igen kunden).

* **Avsändare**: Ange avsändaren av SMS-meddelandena. Denna text visas på mottagarens mobiltelefon.

* **Plugin-konfiguration**: Ange (eller klistra in) den valda modulens konfiguration (som kommer från din SMS-leverantör).
