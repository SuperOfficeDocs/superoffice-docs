# Lägg till favorit

Funktionen Favoriter är ett enkelt sätt för dig att komma åt dina favoritutskick eller formulär.

## Lägga till ett utskick som favorit

Om du vill lägga till ett utskick eller formulär som favorit öppnar du det och klickar på stjärnikonen ![ikon][img1] i det övre högra hörnet. Stjärnan blir gul ![ikon][img2] och skapa utskicket/formuläret läggs omedelbart till i dina favoriter.

I SuperOffice Service kan du nu snabbt komma åt posten genom att klicka på **Marknadsföring** och sedan ![ikon][img3] i navigatorn.

> [!TIP]
> Om du håller muspekaren över ett utskick eller formulär i favoritlistan visas en knappbeskrivning med ytterligare information om utskicket eller formuläret.

## Ta bort ett utskick eller formulär från favoritlistan

1. Öppna utskicket eller formuläret.
2. Klicka på favoritikonen ![ikon][img2] i sidhuvudet. Ikonen ändras till ![ikon][img1].
