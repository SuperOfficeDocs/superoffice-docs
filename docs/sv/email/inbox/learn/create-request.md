# Skapa ärende

Du kan skapa ett nytt ärende i SuperOffice Service baserat på avsändaren och innehållet i ett e-postmeddelande. Ärendet kopplas automatiskt till rätt företag och kontakt om de känns igen.

## Steg

1. Markera ett meddelande i inkorgen.

1. Klicka på knappen **Arkiv** (<i class="ph ph-archive" aria-hidden="true"></i>) och välj **Skapa ärende**.

1. Dialogrutan **Skapa ärende** öppnas med fält som är förifyllda utifrån e-posten:

    * Välj lämplig [kategori][4] för det nya ärendet.
    * Uppdatera ämne, prioritet eller andra fält vid behov.

    ![Dialogrutan Skapa ärende med e-postinnehåll infogat som meddelande -screenshot][img1]

1. Klicka på **Spara** för att skapa ärendet.

    * Om avsändaren känns igen kopplas ärendet automatiskt till rätt kontakt och företag.
    * Om inte, registreras avsändaren som en ny kontakt i SuperOffice Service.
