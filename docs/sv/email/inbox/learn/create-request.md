# Skapa ärende

Med den här funktionen kan du snabbt lägga till ett ärende i SuperOffice Service baserat på avsändaren och texten i e-postmeddelandet.

1. Gå till inkorgen.
2. Markera meddelandet i inkorgen.
3. Klicka på <i class="ph ph-archive" aria-hidden="true"></i> **Arkiv**.
4. Klicka på **Skapa ärende**.
5. Fyll i relevanta fält i dialogrutan.
6. Klicka på **Spara**.
    * Om avsändarens e-postadress känns igen kopplas ärendet automatiskt till rätt kontakt och företag.
    * Om avsändarens e-postadress inte känns igen registreras avsändaren automatiskt som en ny kontakt i SuperOffice Service.
