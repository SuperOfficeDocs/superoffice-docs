# Visa e-postmeddelanden

När du markerar ett meddelande i Inkorgen förhandsvisas meddelandet till höger i fönstret.

Fliken **Förhandsgranska** innehåller följande fält:

| Fält | Beskrivning |
|---|---|
| Ämne | Här visas e-postmeddelandets ämne. |
| Från | Visar avsändaren. Om avsändaren är registrerad i SuperOffice CRM kan du hålla muspekaren över namnet för att visa kontaktkortet. |
| Datum | Här visas datum och tid för meddelandet. |
| Till | Meddelandets mottagare. |
| Bifogade filer | Här visas filer som har bifogats i meddelandet. |

> [!NOTE]
> Gå till <i class="ph ph-user-circle" aria-hidden="true"></i> **Personliga inställningar** > **Alternativ för e-post**. Gå till fliken **Allmänt**. Om du markerar **Visa interna bilder** visas inbyggda bilder i förhandsgranskningsfältet.
