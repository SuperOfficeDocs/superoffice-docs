# Skriv ut ett e-postmeddelande

1. Gå till inkorgen.
2. Öppna meddelandet som du vill skriva ut.
3. Klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Task menu"></i> och välj **Skriv ut valt e-postmeddelande**. Meddelandet öppnas i ett förhandsgranskningsfönster i webbläsaren.
4. Klicka på utskriftsalternativet på menyn i fönstret.
