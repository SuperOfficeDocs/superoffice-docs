# Skapa uppgift

Med den här funktionen kan du snabbt lägga till en uppgift i SuperOffice CRM baserat på avsändaren och texten i e-postmeddelandet.

1. Gå till inkorgen.
2. Markera meddelandet i inkorgen.
3. Klicka på **Arkiv** ![ikon][img1].
4. Klicka på **Skapa uppgift**. Dialogrutan **Händelse** öppnas.
5. [Fyll i nödvändiga uppgifter i fälten][2].
