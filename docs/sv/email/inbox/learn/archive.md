# Arkivera e-postmeddelanden

## <a id="in"></a>Arkivera inkommande e-post

Du kan arkivera ett inkommande e-postmeddelande (inklusive eventuella bilagor) som ett dokument i SuperOffice CRM. Detta är praktiskt när du vill spara information som du har tagit emot via e-post tillsammans med andra uppgifter om företaget/kontakten i SuperOffice CRM.

1. Gå till inkorgen eller den aktuella mappen.
2. Markera det meddelande du vill arkivera.
3. Klicka på <i class="ph ph-archive" aria-hidden="true"></i> **Arkiv**.
4. Klicka på **Arkivera e-postmeddelande**. Dialogrutan [**Dokument** öppnas][1].

## <a id="out"></a>Arkivera utgående e-post

Om du vill skicka ett meddelande och arkivera det som ett dokument i SuperOffice CRM klickar du på **Skicka och arkivera** när du [skriver ett e-postmeddelande][3].

> [!TIP]
> Om du inte arkiverar meddelandet när du skickar det kan du fortfarande gå till mappen Skickat och arkivera meddelandet enligt beskrivningen ovan.

## <a id="attachment"></a>Arkivera eller öppna bilaga

1. Gå till inkorgen.
2. Markera meddelandet i inkorgen. Bilagorna visas under sidhuvudet för e-post, där namnet och filtypsikonen visas (om den känns igen).
3. Klicka på pilen på den bilaga du vill spara eller öppna.
4. Välj ett av följande alternativ:
    * **Aktivt**: Bilagan visas i din webbläsare. Klicka på bilagan för att öppna den eller spara den. Hur det fungerar beror på webbläsaren.
    * **Arkiv**: Dialogrutan [**Dokument** öppnas][1].
