# Inkorgen i SuperOffice <i class="ph ph-at" aria-hidden="true"></i>

Inkorgen i SuperOffice är en inbyggd e-postklient som gör att du kan hantera e-post direkt i SuperOffice CRM. Du kan hantera meddelanden, bilagor och kalenderinbjudningar samt koppla e-postmeddelanden till kontakter, försäljningar, projekt eller ärenden – allt på ett ställe.

![Förhandsgranskning av e-postmeddelande med valt meddelande och tillgängliga åtgärder -screenshot][img1]

## Öppna inkorgen

Välj <i class="ph ph-at" aria-hidden="true"></i> **Inkorg** i navigatorn för att öppna inkorgen i SuperOffice.

## Utseende för inkorg

Inkorgens skärmbild är uppdelad i två delar:

* **Vänster ruta:** Visar en lista med meddelanden i den aktuella mappen.
  * Klicka på **Nytt e-postmeddelande** för att [skapa ett nytt meddelande][3].
  * Klicka på mappnamnet (till exempel **INKORG**) för att välja en annan mapp från ditt anslutna e-postkonto.
  * En etikett som **5 nya** bredvid mappnamnet visar hur många olästa meddelanden som finns i mappen.

* **Höger ruta:** Visar det valda meddelandet och tillgängliga åtgärder, som svara, arkivera och mer.

## Ikoner och funktioner i inkorgen

| Symbol | Beskrivning |
|---|---|
| <i class="ph ph-magnifying-glass" aria-label="Search icon"></i> | Sök i aktuell mapp. |
| <i class="ph ph-sort-ascending" aria-label="Sort ascending icon"></i> / <i class="ph ph-sort-descending" aria-hidden="true"></i> | Sortera meddelanden efter datum, avsändare eller ämne i stigande eller fallande ordning. |
| <i class="ph ph-funnel" aria-label="Filter icon"></i> | Visa eller dölj meddelanden baserat på kriterier som avsändartyp, olästa eller mottagningsdatum. |
| <i class="ph ph-cloud-slash" aria-label="Not subscribed icon"></i> / <i class="ph ph-cloud-check" aria-label="Subscribed icon"></i> | Visar om den valda mappen är prenumererad och synkroniserad. Klicka för att växla prenumeration. Vissa mappar (som Inkorg och Skickat) kan inte avslutas. |
| <i class="ph ph-arrow-clockwise" aria-label="Refresh icon"></i> | Uppdatera meddelandelistan. |

> [!NOTE]
> Även om en mapp är prenumererad och uppdaterad visas endast de senaste meddelandena. Som standard hämtar SuperOffice e-post från de senaste 30 dagarna (Online) eller 180 dagarna (Onsite). Kontakta support om du vill ändra denna gräns.
