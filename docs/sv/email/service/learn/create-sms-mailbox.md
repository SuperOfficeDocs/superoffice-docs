# Skapa SMS-postlåda

SuperOffice kan importera SMS-meddelanden. För att det ska fungera måste du konfigurera en SMS-postlåda:

1. [!include[Go to email](includes/goto-email.md)]

1. Klicka på **Ny SMS-inkorg**.

    > [!NOTE]
    > Om det redan finns en SMS-brevlåda visas inte knappen **Ny SMS-brevlåda**.

1. I listrutan **Kategori** väljer du den standardkategori som ska tillämpas på ärenden som har skapats från SMS-meddelanden.

1. I listrutan **Prioritet** väljer du vilken prioritet som ärenden som tagits emot via SMS ska få.

1. [!include[Set customer language](includes/step-set-language.md)]

1. Klicka på **OK**. SMS-postlådan skapas.
