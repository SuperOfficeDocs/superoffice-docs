# SuperOffice Gmail Link

SuperOffice Gmail Link är en plugin-modul som kopplar samman Google Gmail-klienten med SuperOffice. Detta innebär att du bland annat kan:

* Arkivera inkommande och utgående e-postmeddelanden och bilagor från ditt Gmail-konto som dokument i SuperOffice.
* Skapa kalenderaktiviteter i SuperOffice baserat på e-postmeddelanden i Gmail.
* Skapa ärenden baserat på e-postmeddelanden i Gmail.
* Sök avsändare i SuperOffice: användarvänliga guider som hjälper dig att skapa ett nytt företag, en ny kontakt eller en ny e-postadress i din SuperOffice CRM-databas om avsändaren inte är okänd.
