# Aktivera SuperOffice Gmail Link

Titta på den här filmen eller följ anvisningarna nedan om du vill lära dig att konfigurera Gmail Link och koppla ditt Gmail-konto till SuperOffice (filmens längd: 2:53):

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/kiQyO9KwiEs]

## Steg

1. Gå till appsidan för Gmail Link i [SuperOffice App Store][1].

1. Klicka på **Installera** och välj vilken webbläsare du vill installera tillägget för.

    **Chrome:** Om du väljer Chrome omdirigeras du till Chrome Web Store. Där klickar du på knappen **Lägg till i Chrome** och bekräftar att du vill lägga till det här tillägget.
    * **Firefox**: Om du väljer Firefox öppnas installationsverktyget för Firefox och du ombeds att tillåta webbläsartillägget för Firefox. Bekräfta genom att klicka på **Fortsätt till installationen**.

1. Det sista steget för båda webbläsarna är att logga in på ditt SuperOffice CRM-konto. SuperOffice och Gmail detekterar automatiskt att tillägget har installerats.
