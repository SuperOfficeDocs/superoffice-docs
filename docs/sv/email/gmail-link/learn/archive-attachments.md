# Arkivera bifogade filer

Om den bifogade filen är den enda delen av e-postmeddelandet som du vill behålla för framtida referens kan du spara en e-postbilaga i SuperOffice CRM som ett dokument. Följ de här stegen:

1. Öppna e-postmeddelanden som innehåller en eller flera bifogade filer i Gmail-inkorgen.

2. Håll muspekaren över den bifogade filen som du vill arkivera.
    ![ikon][img1]

3. Klicka på SuperOffice-ikonen ![ikon][img2].

4. I dialogrutan **Arkivera bifogad fil som ett SuperOffice-dokument** fylls fälten **Företag** och **Kontakt** i automatiskt om avsändaren är registrerad i SuperOffice. Fyll i relevant information i följande fält:

    [!include[Archiving options for Gmail](includes/gmail-archiving-options.md)]

    Om avsändaren av e-postmeddelandet inte är registrerad i SuperOffice CRM visas en varning högst upp i dialogrutan. En varning visas också om avsändaren är registrerad på flera företag i SuperOffice CRM.

5. Klicka på **Spara**. Den valda bilagan arkiveras i SuperOffice CRM som ett nytt dokument. Därefter kan du dubbelklicka på e-postmeddelandet i detaljkortet **Aktiviteter** för att öppna det.
