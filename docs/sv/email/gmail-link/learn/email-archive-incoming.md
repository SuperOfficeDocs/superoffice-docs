# Arkivera inkommande e-post

Du kan arkivera ett inkommande e-postmeddelande (inklusive eventuella bilagor) som ett dokument i SuperOffice CRM. Detta är praktiskt när du vill spara information som du har tagit emot via e-post tillsammans med andra uppgifter om företaget/kontakten i SuperOffice CRM.

![Om du har installerat SuperOffice Gmail Link kan du arkivera e-postmeddelanden och skapa aktiviteter eller ärenden -screenshot][img2]

## Steg

1. Välj det e-postmeddelande du vill arkivera i Gmail-inkorgen.

2. Klicka på ![ikon][img1] i verktygsfältet för e-post.

    Om avsändaren av e-postmeddelandet inte är registrerad i SuperOffice CRM visas en varning högst upp i dialogrutan. En varning visas också om avsändaren är registrerad på flera företag i SuperOffice CRM.

3. I dialogrutan fylls fälten **Företag** och **Kontakt** i automatiskt om avsändaren är registrerad i SuperOffice. Fyll i relevant information i följande fält:

    [!include[Archiving options for Gmail](includes/gmail-archiving-options.md)]

4. Klicka på **Spara**. Det valda e-postmeddelandet (och eventuella bilagor) arkiveras i SuperOffice CRM som ett nytt dokument. Därefter kan du dubbelklicka på e-postmeddelandet på detaljkortet **Aktiviteter** för att öppna det.

Klicka på länken i popupmeddelandet för att öppna dokumentet i SuperOffice.

> [!NOTE]
> Du kan också [arkivera bilagor som dokument][3].
