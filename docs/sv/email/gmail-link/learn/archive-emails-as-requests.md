# Skapa ärende

[!include[Requirement](../../../../../common/includes/req-service-essentials.md)]

Du kan skapa ett ärende i SuperOffice Service baserat på ett e-postmeddelande i Gmail-inkorgen.

1. Öppna ett e-postmeddelande i Gmail-inkorgen.

2. Klicka på knappen **Mer** ![ikon][img1] i verktygsfältet för e-post.

3. Välj **Skapa ärende**.

4. I dialogrutan **Skapa ärende i SuperOffice baserat på e-postmeddelande** klickar du på den ärendekö där du vill att ärendet ska placeras. Varje kö har fått tilldelat en specifik kategori och prioritet.

5. Klicka på **Spara**. Ett bekräftelsemeddelande visas.
    * Om avsändarens e-postadress känns igen kopplas ärendet automatiskt till rätt kontakt och företag.
    * Om avsändarens e-postadress inte känns igen registreras avsändaren automatiskt som en ny kontakt i SuperOffice Service.

6. (valfritt) I bekräftelsemeddelandet väljer du **Klicka här för att visa det** om du vill titta närmare på ärendet.
