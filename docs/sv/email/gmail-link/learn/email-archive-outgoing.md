# Arkivera utgående e-post

När du skickar ett e-postmeddelande kan du arkivera det (inklusive eventuella bilagor) som ett dokument i SuperOffice CRM. Detta är ett användbart sätt att hålla reda på den utgående korrespondensen.

Följ de här stegen:

1. Klicka på **Skriv** i Gmail.

2. Skriv e-postmeddelandet.

    > [!NOTE]
    > Om du skickar e-postmeddelandet till flera mottagare aktiveras e-postmeddelandet bara för den första mottagaren.

3. Klicka på **Skicka och arkivera**. E-postmeddelandet skickas.

4. I dialogrutan **Arkivera e-postmeddelande i SuperOffice** fylls fälten **Företag** och **Kontakt** i automatiskt om mottagarens e-postadress är registrerad i SuperOffice. Fyll i relevant information i följande fält:

    [!include[Archiving options for Gmail](includes/gmail-archiving-options.md)]

    Om e-postmeddelandets mottagare inte är registrerad i SuperOffice CRM visas en varning högst upp i dialogrutan. En varning visas också om mottagaren är registrerad på flera företag i SuperOffice CRM.

5. Klicka på **Spara**. E-postmeddelandet (inklusive eventuella bilagor) arkiveras i SuperOffice CRM som ett nytt dokument. Därefter kan du dubbelklicka på e-postmeddelandet på detaljkortet **Aktiviteter** för att öppna det.

Klicka på länken i popupmeddelandet för att öppna dokumentet i SuperOffice.

> [!TIP]
> Om du inte arkiverar meddelandet när du skickar det kan du fortfarande [gå till mappen Skickat och arkivera meddelandet][3].
