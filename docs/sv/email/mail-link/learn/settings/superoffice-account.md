# SuperOffice-konto

De här alternativen visas i fönstret **SuperOffice-konto** i dialogrutan **Alternativ för SuperOffice CRM**.

| Alternativ | Beskrivning |
|---|---|
| Vald SuperOffice-klient | Vilken SuperOffice-klient som ska användas när du [söker efter avsändare][2]: **SuperOffice CRM** (Windows-klient), **SuperOffice CRM** (webbklient) eller **Service**. |
| Användarnamn | Ange ditt användarnamn för den valda klienten och klicka på **Testa koppling** för att bekräfta att du kan logga in i den valda klienten och därmed att kopplingen mellan e-postklienten och SuperOffice-klienten fungerar. |

## Avancerade inställningar

Om du klickar på nedåtpilen till höger om **Avancerade inställningar** visas följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Aktivera felsökningslogg | Det här alternativet bör aktiveras vid felsökning om det önskas av SuperOffices kundsupport. |
| Visa loggfil | Öppna loggfilen för SuperOffice Mail Link. Även relevant vid felsökning. |
| URL | Det här fältet innehåller adressen till SuperOffice-klienten där e-postmeddelanden arkiveras. Endast relevant för SuperOffice CRM for Web och SuperOffice Service. |
