# Dialogrutan SuperOffice CRM-alternativ

Öppna dialogrutan **Alternativ för SuperOffice CRM** genom att klicka på den lilla pilen till höger om **SuperOffice**-knappen i e-postfönstret eller inkorgen.

Du kan också välja **Alternativ för SuperOffice CRM** på menyn **Verktyg** i Outlook eller menyn **Åtgärder** i Anteckningar.

I den här dialogrutan kan du ange arkiveringsalternativ, generella alternativ och kontoalternativ samt visa versionsinformation om SuperOffice Mail Link.

## Tillgängliga alternativ

* [Arkiveringsalternativ][1]
* [Allmänna inställningar][2]
* [Inbjudningar][3]
* [SuperOffice-konto][4]

## Fönstret Om

Fönstret **Om** i dialogrutan innehåller information om den aktuella versionen av SuperOffice Mail Link och om installerade komponenter.
