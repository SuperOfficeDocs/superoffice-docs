# Skapa ärende från ett e-postmeddelande

Troligtvis får du de flesta kundförfrågningar via företagets info@- och support@-adresser. Ibland skickar kunderna dock personliga e-postmeddelanden med frågor eller problem som du behöver hantera.

Du kan arkivera ett e-postmeddelande i inkorgen som ett ärende i SuperOffice så att du inte glömmer att följa upp det. Hur detta fungerar beror på om du lägger till e-postmeddelandet till ett nytt eller ett befintligt ärende.

> [!NOTE]
> Kräver Microsoft Outlook på Mac eller PC, Gmail eller SuperOffice Inbox.

## Skapa ett nytt ärende

1. Markera det eller de e-postmeddelanden du vill arkivera. Om du väljer mer än ett e-postmeddelande upprepas proceduren nedan för varje e-postmeddelande.

2. Klicka på knappen **Arkivera e-post** i verktygsfältet och välj **Skapa ärende** i listrutan.

3. I dialogrutan **Arkivera ett ärende**, välj **Skapa nytt ärende**.

4. I fältet **Sätt ärendet i den här kön** väljer du den ärendekö du vill skicka ärendet till. Varje kö har fått tilldelat en specifik kategori och prioritet.

5. Markera kryssrutan **Ta bort från inkorgen** om du vill ta bort e-postmeddelandet från inkorgen och flytta det till mappen **Borttaget** i e-postklienten efter att det har arkiverats i SuperOffice Service.

6. Klicka på **OK**. Ett bekräftelsemeddelande visas.

    * Om avsändarens e-postadress känns igen arkiveras ärendet automatiskt under rätt kontakt och företag.
    * Om avsändarens e-postadress inte känns igen registreras avsändaren automatiskt som en ny kontakt i SuperOffice Service.

7. I bekräftelsemeddelandet kan du välja **Klicka här för att öppna ärende** om du vill titta närmare på ett ärende, eller på **Stäng** för att stänga bekräftelsemeddelandet.

## Lägg till ett meddelande till ett befintligt ärende

1. Markera de e-postmeddelanden du vill arkivera. Om du väljer mer än ett e-postmeddelande upprepas stegen nedan för varje e-postmeddelande.
2. Välj **Arkivera e-post** > **Skapa ärende** i verktygsfältet.
3. I dialogrutan **Arkivera som ärende** väljer du **Lägg till meddelande till befintligt ärende**.
4. I sökfältet skriver du ID:n eller de första bokstäverna i ärendets rubrik.
5. Tryck på ENTER eller klicka på knappen med förstoringsglaset för att starta sökningen. Ärenden som matchar dina sökvillkor visas i fältet **Välj ärende som detta meddelande ska bifogas**.
6. Välj det ärende som du vill lägga till e-postmeddelandet till.
7. Markera kryssrutan **Ta bort från inkorgen** om du vill ta bort e-postmeddelandet från inkorgen och flytta det till mappen **Borttaget** i e-postklienten efter att det har arkiverats i SuperOffice Service.
8. Klicka på **OK**. Ett meddelande visas som bekräftar att ärendet har uppdaterats. Om ärendet tidigare var stängt, återaktiveras det nu.
9. I bekräftelsemeddelandet kan du välja **Klicka här för att öppna ärende** om du vill titta närmare på ett ärende, eller på **Stäng** för att stänga bekräftelsemeddelandet.
