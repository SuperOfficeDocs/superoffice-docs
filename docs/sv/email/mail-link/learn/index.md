# SuperOffice Mail Link

SuperOffice Mail Link är en plugin-modul som kopplar samman e-postklienten Microsoft Outlook med SuperOffice CRM och/eller SuperOffice Service. Detta innebär att du bland annat kan:

* Arkivera e-postmeddelanden och bilagor till SuperOffice CRM.
* Arkivera ett e-postmeddelande som ett dokument eller en uppgift i SuperOffice CRM.
* Arkivera ett e-postmeddelande som ett ärende i SuperOffice Service.
* Sök efter ett e-postmeddelandes avsändare.
* Konfigurera funktioner för standardarkivering.
* Använd SuperOffice-mallar eller fördefinierade e-postmallar när du skapar e-postmeddelanden.
* Uppdatera informationen om företag och kontakter i SuperOffice CRM baserat på e-postmeddelanden som du får.
