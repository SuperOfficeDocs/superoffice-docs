# Ändra standardklienten för e-post i SuperOffice

1. Öppna SuperOffice CRM.
2. Gå till <i class="ph ph-user-circle" aria-hidden="true"></i> **Personliga inställningar** > **Inställningar**.
3. Välj **E-post**.
4. Välj önskad e-postklient i fältet **E-postklient**.
5. Klicka på **Spara**.
6. Nästa gång du klickar på **Inkorg** i navigatorn i SuperOffice CRM öppnas den e-postklient du har valt.

> [!TIP]
> I SuperOffice CRM for Web kan du också ändra standardklienten för e-post på inloggningssidan. När du klickar på **Inställningar** visas ett fält för **E-postklient**.
