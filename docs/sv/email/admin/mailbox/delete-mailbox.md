# Ta bort brevlåda

1. [!include[Go to email](includes/goto-email.md)]

1. Klicka på önskad brevlåda. Skärmen Egenskaper visas.

1. Klicka på knappen **Ta bort** längst ned på skärmen. Brevlådan tas bort.
