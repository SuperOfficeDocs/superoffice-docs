# SuperOffice for Outlook

Med SuperOffice för Outlook-tillägget kan du enkelt arbeta med både Microsoft 365 och din SuperOffice CRM-lösning genom att du får tillgång till viktiga uppgifter och funktioner från SuperOffice direkt i din e-post.

När du har anslutit SuperOffice for Outlook kan du spara nya kontakter, e-postmeddelanden, försäljningar, aktiviteter eller ärenden direkt från Outlook. På tilläggspanelen får du snabbt tillgång till viktig information om kontakter, försäljningar och ärendehistorik. Få överblick över kontakter och företag i e-postmeddelandet för att omedelbart sätta in den interagerar med i en kontext.

![På fliken Åtgärd kan du utföra uppgifter som att spara till e-post och skapa aktivitet i SuperOffice -screenshot][img1]

## Så installerar du SuperOffice for Outlook

Om du vill ansluta SuperOffice för Outlook går du till vår [App Store][1] och följer instruktionerna på **installationsfliken**. Ingen installation krävs, du behöver bara aktivera tillägget i din webbläsare.

> [!NOTE]
> Du måste ha en aktiv Microsoft 365-prenumeration för att ansluta tillägget till Outlook.
