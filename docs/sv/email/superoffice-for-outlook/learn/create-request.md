# Lägg till ärende eller meddelande

Du kan skapa ett [ärende][1] i SuperOffice direkt från Outlook. Om e-postmeddelandet rör ett pågående problem kan du bifoga meddelandet till ett specifikt ärende.

## Skapa ärende

1. Öppna e-postmeddelandet som rör ärende.

1. Välj **Skapa ärende i SuperOffice** i **SuperOffice for Outlook**-sidopanelen.

1. Välj rätt kö för det nya ärende.

1. Klicka på **Spara**.

## Lägg till meddelande

1. Välj **Lägg till meddelande till ärende** i **SuperOffice for Outlook**-sidopanelen.

1. Sök efter ärende med ID eller titel och välj det.

1. Klicka på **Spara**.

> [!TIP]
> Du kan också skapa en händelse eller försäljning direkt från e-postmeddelandet.
