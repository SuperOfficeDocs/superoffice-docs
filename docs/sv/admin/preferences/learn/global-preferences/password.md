# Lösenordspolicy för Kundcenter

Gå till ![ikon][img1] **Inställningar** i navigatorn och välj fliken **Globala inställningar**. Sektionen **Lösenordspolicy för Kundcenter** innehåller följande alternativ:

* **Dölj lösenord för kontakt i ärendehandläggningen**: Om du markerar det här alternativet visas lösenord som prickar.
* **Tillåt bara svåra lösenord när kontaktslösenord anges**: Om du markerar det här alternativet måste kontakternas lösenord uppfylla minimikraven nedan.
* **Minimalt antal tecken i lösenord**: Det minsta antalet tecken som tillåts i ett lösenord.
* **Minimalt antal siffror i lösenord**: Det minsta antalet siffror (0–9) som tillåts i ett lösenord.
* **Minimalt antal små bokstäver i lösenord**: Det minsta antalet små bokstäver som tillåts i ett lösenord.
* **Minimalt antal stora bokstäver i lösenord**: Det minsta antalet stora bokstäver som tillåts i ett lösenord.
* **Minimalt antal specialtecken i lösenord**: Det minsta antalet specialtecken som tillåts i ett lösenord. Specialtecken är tecken som varken är bokstäver eller siffror, t.ex. asterisker och frågetecken.
