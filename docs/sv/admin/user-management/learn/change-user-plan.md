# Ändra användarplaner för aktiva användare

När medarbetare tar på sig nya uppgifter eller får en ny tjänst på företaget kan du behöva ändra deras användarplaner.

Du kan ge användarna åtkomst till nya funktioner genom att ändra deras användarplaner under Inställningar och underhåll.

> [!TIP]
> Ge användarna den utbildning de behöver och följ upp hur det går så att de snabbare kommer igång med de nya funktionerna.

## Steg

Användarinställningar i SuperOffice CRM där du ändrar användarplaner Användarinställningar i SuperOffice CRM där du ändrar användarplaner

1. Klicka på **Användare** under Inställningar och underhåll.

2. Välj den aktiva användare vars användarplan behöver ändras i fönstret för medarbetare.

    Kontrollera att de har rätt Roll och tillhör rätt Primärgrupp.

3. Välj önskad användarplan i fältet för Användarplan.

4. Klicka på Spara. Nästa gång användaren loggar in i SuperOffice CRM kommer de nya funktionerna att vara tillgängliga.
