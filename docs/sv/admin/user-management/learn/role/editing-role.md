# Redigera en roll

[!include[Requirement](../includes/note-anon-req.md)]

1. [!include[Open Roles](includes/open-roles.md)]

2. Gå till fliken **Medarbetare** eller **Externa**.

    [Så redigerar du rollen för anonyma användare.][1]

3. Dubbelklicka på den [roll][3] du vill redigera i listan.

4. Ändra rollens namn och beskrivning i dialogrutan **Skapa ny roll** och klicka sedan på **Spara**.

5. [Redigera rollens rättigheter.][4]

6. Om du redigerar roller för medarbetare: Gå till fliken **Funktionella behörigheter** och redigera rollens funktionella behörigheter.

    [!include[How to set functional rights](includes/add-remove-right.md)]

    Ändringarna sparas automatiskt.
