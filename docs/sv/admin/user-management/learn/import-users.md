# Importera användare

Om ditt företag använder Active Directory eller någon annan autentiseringstyp kan du lägga till flera användare samtidigt:

> [!NOTE]
> Importfunktionen är inte tillgänglig om du använder SuperID-autentisering.

1. Öppna fönstret Användare och välj fliken **Medarbetare**.
1. Klicka på **Importera** längst ned i översikten.
1. Klicka på **OK** i dialogrutan **Välj autentiseringstyp**.
1. Välj den användargrupp användarna tillhör i listan **Användargrupp** i dialogrutan **Importera användare från ActiveDirectory**. Användarna i den aktuella gruppen visas i översikten **Potentiella användare**.
1. Markera de användare du vill lägga till.
1. Välj i vilket företag du vill lägga användarna i listan **Lägg till användare till företag**.
1. Välj en roll och primärgrupp för användarna.
1. Klicka på **OK**. Namnen på de nya användarna visas nu i listan.
1. [Ange information om användaren][1].
