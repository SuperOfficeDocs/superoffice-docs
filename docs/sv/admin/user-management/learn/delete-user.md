# Ta bort medarbetare

1. [!include[Click Users](includes/goto-users.md)]

2. Välj fliken **Medarbetare**.

3. [!include[Select user](includes/select-user.md)]

4. Klicka på knappen **Ta bort**. Du kan inte ta bort ditt eget användarnamn, som du använde när du loggade in.

5. Markera ett alternativ i dialogrutan **Ta bort användare**. Klicka på ett alternativ för att visa en beskrivning av vad som händer när du tar bort användaren.

    * Ta bort användarlicenser
    * Markera medarbetaren som tidigare anställd
    * Ta bort medarbetare \[login ID\]
    * Ta bort medarbetare \[login ID\] och kontakt \[full name\]

    > [!NOTE]
    > De här alternativen är rangordnade efter effekt.

6. Klicka på **OK**.
