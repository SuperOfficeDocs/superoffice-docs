1. Klicka på **Next**. **Förhandsvisning**-fönstret visas.

1. I **Förhandsvisning** kan du granska importdata innan importen startar.

    | Ikon | Beskrivning |
    |:-:|---|
    | ![Ikon – nytt företag][img1] | Företaget finns inte och kommer att läggas till. |
    | ![Ikon – företaget ändras][img2] | Företaget finns redan och kommer att uppdateras. |
    | ![Ikon – ny person][img3] | Kontakten finns inte och kommer att läggas till. |
    | ![Ikon – personen ändras][img4] | Kontakten finns redan och kommer att uppdateras. |
    | tom | Ingen uppdatering. |
