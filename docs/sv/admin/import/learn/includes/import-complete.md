När importen är klar visas två listor:

* Nya företag och kontakter som har importerats
* Befintliga företag och kontakter som har ändrats genom importen

Ovanför varje lista finns en länk till urvalet som innehåller de nya/ändrade företagen/kontakterna.

> [!TIP]
> Om du har importerat nya företag/kontakter och problem uppstår med de importerade uppgifterna kan du öppna urvalet i SuperOffice CRM och ta bort de nya företagen/kontakterna.
