**Många kontakter visas utan ikon. Betyder det att de inte kommer att importeras?**

Rader utan ikoner importeras inte. Om du ser att många kontakter inte importeras kan det bero på att de inte är kopplade till ett företag. I dessa fall kan du göra något av följande:

* Om kontakten verkligen är kopplad till ett företag går du tillbaka till importkällan (Gmail, Outlook eller Excel) och anger ett företag för kontakten. Starta sedan om importen.

* Om kontakten inte är kopplad till ett företag utan ska registreras under sitt eget namn i SuperOffice lägger du in personen som ett företag:

    1. Klicka på **Föregående** och **Konfigurera importinställningar**.
    2. Klicka på listan **Om företagsnamn är tomt** under **Kontakt**.
    3. Välj **Skapa företag med kontaktens namn**.
    4. Klicka på **Spara**.
    5. Klicka på **Nästa** och kontrollera att [ikonen Nytt företag][img1] visas bredvid kontakten. Personen registreras som ett företag i SuperOffice CRM med sig själv som enda kontaktperson.
