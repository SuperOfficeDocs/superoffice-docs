# Ändra en listas namn och beskrivning

[!include[Find specific list](includes/tip-find-list.md)]

1. Öppna fönstret **Listor**.
2. [!include[Click arrow and select](includes/expand-list.md)] den aktuella listan.
3. Klicka på knappen **Redigera lista**.
4. Skriv in ett nytt namn på listan i fältet **Namn**.
5. Redigera beskrivningen av listan i fältet **Beskrivning** om det behövs.
6. Klicka på **OK** så att ändringarna sparas under Inställningar och underhåll.
