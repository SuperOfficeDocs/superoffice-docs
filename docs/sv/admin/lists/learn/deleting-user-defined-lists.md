# Ta bort användardefinierad lista

> [!NOTE]
> Du kan inte ta bort listor som är fördefinierade i SuperOffice CRM. Om du väljer en fördefinierad lista visas inte knappen **Ta bort lista**. Du kan inte heller ta bort listor som används i användardefinierade fält. I dessa fall är knappen **Ta bort lista** på fliken **Listor** inaktiverad.

[!include[Find specific list](includes/tip-find-list.md)]

**Steg:**

1. [!include[Go to Lists](includes/goto-lists.md)]
2. [!include[Click arrow and select](includes/expand-list.md)] önskad lista i listrutan högst upp.
3. Klicka på knappen **Ta bort lista**.
4. Klicka på **Ja** i bekräftelserutan.

    Listan tas bort.
