# Användardefinierade listor

För att komma igång, klicka på knappen **Listor** i navigatorn (<i class="ph ph-list-bullets" aria-hidden="true"></i>).

[!include[Find specific list](includes/tip-find-list.md)]

## Lägga till användardefinierad lista

1. Klicka på knappen **Skapa lista** längst upp till höger i fönstret.

1. Skriv in namnet på listan i fältet **Namn**. *(Obligatoriskt)*

1. Klicka på **OK** för att spara listan.

Listan läggs till listan **Listor**. Om du markerar listan i listan visas beskrivningen högst upp i fönstret **Listor**.

Förutom att ange ett namn och en beskrivning för listan måste du [lägga till poster][1] i den.

## Ändra en listas namn och beskrivning

1. Klicka på pilen och välj önskad lista i listrutan högst upp.

1. Klicka på knappen **Redigera lista**.

1. Skriv in ett nytt namn på listan i fältet **Namn**.

1. Redigera beskrivningen av listan i fältet **Beskrivning** om det behövs.

1. Klicka på **OK** så att ändringarna sparas under Inställningar och underhåll.

## Ta bort användardefinierad lista

> [!NOTE]
> Du kan inte ta bort listor som är fördefinierade i SuperOffice CRM. Om du väljer en fördefinierad lista visas inte knappen **Ta bort lista**. Du kan inte heller ta bort listor som används i användardefinierade fält. I dessa fall är knappen **Ta bort lista** på fliken **Listor** inaktiverad.

1. Klicka på pilen och välj önskad lista i listrutan högst upp.

1. Klicka på knappen **Ta bort lista**.

1. Klicka på **Ja** i bekräftelserutan.

    Listan tas bort.
