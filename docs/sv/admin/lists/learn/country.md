# Allmänt – Land

På flera ställen i SuperOffice CRM kan du välja önskat land från en urvalslista. Listan över länder (och deras flaggor) hämtas från ISO:s standardiserade landlista. Landlistan är fördefinierad i fönstret **Listor** under Inställningar och underhåll. Om det behövs kan du redigera de fördefinierade länderna eller lägga till nya.
