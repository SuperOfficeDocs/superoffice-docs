# GUI – Webbpanel

Du kan ha kortkommandon för webbadresser i SuperOffice CRM så att du kan öppna dem direkt från programmet.

Du definierar vilka webbadresser som ska vara tillgängliga för användarna SuperOffice CRM i fönstret **Listor** under Inställningar och underhåll.

Posterna visas i den ordning du skapar dem. Om du vill ändra ordningen väljer du en post och flyttar den uppåt eller nedåt med pilknapparna till vänster om listan (![ikon][img1] ![ikon][img2]).

När du har lagt till poster kan du ange egenskaper för varje enskild post. Du kan ange

* vilka [användargrupper posten ska vara synlig för][2]
* vilken [rubrik][1] den ska visas under.

## Vad vill du göra nu?

* [Lägga till poster i listan GUI – Webbpanel][3]
