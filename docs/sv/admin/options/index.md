# Alternativ

I fönstret **Systemval** anger du inställningar som gäller för hela systemet, det vill säga inställningar för fritextsökning, tidszoner, nummertilldelning och etiketter. De flesta företag behöver inte ändra standardinställningarna i SuperOffice CRM. Om du behöver ändra dem gör det i fönstret **Systemval**.

## Flikar i fönstret Systemval

| Flik | Beskrivning |
|---|---|
| [Fritextsökning][3] | Här bestämmer du hur funktionen fritextsökning ska fungera i SuperOffice CRM. |
| [Tidszoner][4] | Här avgör du hur tidszonsfunktionen ska fungera. |
| [Nummertilldelning][1] | Här anger du en startsiffra för **Nummer**-fälten i fönstren Företag, Projekt, Försäljning och kontaktkortet samt för dokumentmallarna. |
| [Ledtexter][2] | Här kan du byta namn på enskilda fält i SuperOffice CRM. |
| [Kundspråk][6] | Här kan du konfigurera kundspråken så att det blir enkelt att kommunicera med kunder på olika språk. Du kan också skapa kategorinamn i SuperOffice Kundcenter. |

> [!NOTE]
> Om ditt företag har licens för [SAINT][7] innehåller fönstret **Systemval** även fliken **Statusövervakning**.
