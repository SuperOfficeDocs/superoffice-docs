# Rapporter

På fliken **Rapporter** i fönstret Systemval kan du redigera standardrapporternas namn och beskrivningar. Du kan också välja vilka rapporter som ska publiceras (det vill säga göras tillgängliga för användarna). Du kan även importera nya rapporter.

I rapportlistan visas rapporternas format (utseende), om rapporten är en standardrapport eller inte, rapporttypen (innehållet) och om rapporten är publicerad. Klicka på en rapport för att visa rapportbeskrivningen längst ned på fliken.

## Öppna och redigera en standardrapport

[!include[Go to Options](includes/open-options.md)]

1. Välj fliken **Rapporter**.
1. Dubbelklicka på standardrapporten som du vill redigera.
1. Redigera namnet och beskrivningen i dialogrutan. Du kan inte ändra rapporttyp.
1. Markera kryssrutan **Publicerat** om du vill göra rapporten tillgänglig för användarna.
1. Klicka på **Spara**.

Du kan inte ta bort standardrapporter men du kan ange att de inte ska publiceras.

## Importera rapporter

Du kan importera rapporter som har exporterats från SuperOffice Reporter Studio eller skapats av en konsult eller SuperOffice Support.

1. Gå till fliken **Rapporter** i fönstret **Systemval**.
2. Klicka på **Importera** längst ned på fliken.
3. Klicka på **Bläddra** i dialogrutan som visas.
4. Välj rapporten som du vill importera och klicka på **Öppna** och **Spara**.

Om importen lyckas visas rapporten på fliken **Rapporter** med ditt användarnamn som ägare.
