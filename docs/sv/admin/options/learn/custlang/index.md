# Kundspråk

[!include[Requirement](../../../../learn/includes/req-cep.md)]

För att skapa en bra kundupplevelse måste du interagera med dina kunder på deras egna språk när de besöker ditt kundcenter och i övrig kommunikation.

Om du [konfigurerar extra språk][1] blir det enkelt för dina kunder att skapa, kommentera och kontrollera status på ärenden på valt språk. Dessutom får de tillgång till alla FAQ på valt språk.

Som standard kan du lägga till 6 olika språk. Om språket du vill använda inte är tillgängligt ber vi dig att kontakta oss så kan våra konsulter hjälpa dig att lägga till de språk du behöver.

## Fönstret Kundspråk

På skärmen **Kundspråk** visas en lista över installerade kundspråk. På den här sidan kan du konfigurera kundspråken så att det blir enkelt att kommunicera med kunder på olika språk.

Till exempel:

* Du kan ha [flera språkversioner][2] av SuperOffice Customer Centre, FAQ och svarsmallar.
* Du kan ange ett språk för en kontakt så att personen automatiskt får svarsmallar på det språket.
* [Supportärenden kan delas in][3] i olika grupper för olika avdelningar och språk.

Vid installationen placerades 6 kundspråksmappar (DE, DK, EN, NL, NO och SE) i mappen **\\templates** på servern. Du måste själv konfigurera dessa kundspråk.
