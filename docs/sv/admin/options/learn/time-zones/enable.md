# Aktivera användning av tidszoner

[!include[Go to Options](../includes/open-options.md)]
2. Välj fliken **Tidszoner**.
3. Klicka på **Aktivera tidszoner**.
4. Välj standardtidszon i dialogrutan. Du kan ändra detta senare.
5. Klicka på **Spara**. Listan över tidszoner aktiveras.
6. Om länken **Ladda ned nya tidszonsuppdateringar** visas högst upp i fönstret kan du klicka på den för att uppdatera tidszonerna i SuperOffice.

> [!TIP]
> Gå till **Inställningar** för att visa tidszonsväljaren och ange standardtidszon.
