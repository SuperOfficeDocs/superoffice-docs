# Redigera tidszoner

Du kan välja vilka tidszoner du vill visa från en lista som innehåller alla tillgängliga tidszoner.

[!include[Go to Options](../includes/open-options.md)]
2. Välj fliken **Tidszoner**. Här ser du namnen på alla tillgängliga tidszoner.
3. I den vänstra kolumnen väljer du de tidszoner du vill visa i SuperOffice.

> [!TIP]
> Använd fältet **Filtrera** för att söka efter specifika tidszoner.
