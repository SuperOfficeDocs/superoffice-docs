# Skapa aktör i ERP-systemet

[!include[Requirement](includes/req-integration-server.md)]

Om du till exempel vill koppla ett SuperOffice-företag till en kund i ERP-systemet och denna kund inte finns i ERP-systemet kan du skapa kunden i ERP-systemet baserat på data från SuperOffice.

> [!NOTE]
> Knappen **Skapa** är inaktiverad om SuperOffice inte får skapa aktörer i ERP-systemet.

1. [Öppna dialogrutan Koppla till ERP][1].
2. Markera **Ingen träff i ERP** längst ned i dialogrutan **Koppla till ERP** och klicka på **Skapa**. Dialogrutan **Skapa i ERP** öppnas.
3. Fyll i relevanta värden.
4. Klicka på **Skapa**. Du kommer tillbaka till fliken **ERP**, där synkroniserade data visas.
5. Klicka på **Spara**.
