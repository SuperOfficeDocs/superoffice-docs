# ERP

[!include[Requirement](includes/req-integration-server.md)]

Om din organisation använder ett ERP-system och har kopplat det till SuperOffice CRM kan du synkronisera företag (kunder och leverantörer) och projekt mellan ERP-systemet och SuperOffice CRM.

På huvudkorten för **Företag**, **Kontakt** och **Projekt** visas **ERP**-fliken. Här kan du koppla ett SuperOffice-företag till en kund/leverantör i ERP-systemet. Kontakter som är registrerade på SuperOffice-företaget kan därefter kopplas till kontakter i ERP-systemet. På motsvarande sätt kan SuperOffice-projekt kopplas till projekt i ERP-systemet. När du har upprättat länken synkroniseras data mellan ERP-systemet och SuperOffice CRM.
