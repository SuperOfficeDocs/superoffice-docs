# Skicka en händelse som ett e-postmeddelande

1. Högerklicka på händelsen på detaljkortet **Aktiviteter** i fönstret Företag, Kontakt, Kalender, Försäljning eller Projekt och välj **Skicka som e-post**.

2. Ett meddelande öppnas i ditt e-postprogram:

    * Som standard infogas adressen till en angiven kontakt i **Till**-fältet.

    * Händelsetypen och händelsebeskrivningen från dialogrutan för händelser infogas i ämnesfältet.

    * Redigera e-postmeddelandet efter behov. Du kan till exempel lägga till fler mottagare.

        > [!NOTE]
        > Informationen uppdateras däremot inte i SuperOffice CRM.

3. Klicka på **Skicka** för att skicka meddelandet.
