# Skriva ut

## Skriv ut dag, vecka, månad eller översikt

1. Välj önskad flik i Kalender-fönstret (**Dag**, **Vecka**, **Månad** eller **Översikt**).
2. Högerklicka på en bokning och välj **Skriv ut**.
3. I fönstret som visar innehållet som ska skrivas ut klickar du på **Skriv ut**.

## Skriv ut en händelse (bokning, samtal eller uppgift)

1. Öppna händelsen på detaljkortet **Aktiviteter** i fönstret Företag, Kontakt, Kalender, Försäljning eller Projekt.
2. Klicka på **Uppgift** och välj **Skriv ut**.
3. Klicka på utskriftsknappen i webbläsarfönstret som visar innehållet som ska skrivas ut.
