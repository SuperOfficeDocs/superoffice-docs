# Knappar på toppraden

**Toppraden** innehåller viktiga verktyg för navigering i SuperOffice CRM. Vilka knappar som visas beror på modulen (**CRM**, **Service** eller **Inställningar och underhåll**), men deras funktioner är desamma. Till exempel visas endast **Personliga inställningar**, **Huvudmeny** och **Hjälp** i **Inställningar och underhåll**.

## Översikt över knappar

| Ikon | Namn | Beskrivning |
|:-:|---|---|
| **Nytt** | **Nytt** | Skapa nya poster, såsom företag, kontakter, försäljningar, projekt, urval, händelser, dokument och ärenden. |
| <i class="ph ph-list-magnifying-glass" aria-label="Sök-ikon"></i> | [**Sök**][5] | Sök efter poster som företag, kontakter, försäljningar och dokument (Ej tillgängligt i Service och Inställningar och underhåll). |
| <i class="ph ph-magnifying-glass" aria-label="Förstoringsglas"></i> | [**Sökfält**][10] | Använd fältet **Fritextsök** för att hitta data i hela systemet. |
| <i class="ph ph-bell" aria-label="Klockikon"></i> | [**Aviseringar**][9] | Visa nya aviseringar, såsom inbjudningar, tilldelade ärenden och godkända försäljningar. Klicka på en avisering för att öppna den. |
| <i class="ph ph-user-circle" aria-label="Kontaktikon"></i> | **Personliga inställningar** | Redigera inställningar, e-postsignaturer och lösenord. Visar din profilbild om den är tillagd i ditt kontaktkort. |
| <i class="ph ph-list" aria-label="Huvudmeny"></i> | **Huvudmeny** | Få åtkomst till systeminställningar, växla mellan moduler eller öppna andra applikationer. |
| <i class="ph ph-lightbulb" aria-label="Glödlampa"></i> | **Resurscenter i appen** | Öppna guidade turer, användarguide och inspirationsresurser (endast online). |
| **Hjälp** | **Hjälp** | Få tillgång till användarguider och information anpassad till aktuell uppgift eller modul. |

## Alternativ i menyn Personliga inställningar

| Alternativ | Beskrivning |
|---|---|
| **Inställningar** | Justera [personliga inställningar][6], såsom standardvärden, sortering och visuella effekter. |
| **Redigera personlig e-postsignatur** | Redigera [e-postsignaturen][7] som används i utgående e-post eller ärenden. |
| **E-postalternativ** | Konfigurera inkorgsinställningar. |
| **Din Service-status** | Uppdatera din status (exempelvis Ej närvarande) och lägg till ett valfritt meddelande. |
| **Visa min kontaktinformation** | Öppna ditt kontaktkort för att redigera telefonnummer eller titel. |
| **Lokala inställningar** | Ändra appens språk, nummer-/datumformat och e-postklientinställningar. |
| **Hämta mobilappen** | Konfigurera inställningar för SuperOffice Mobile CRM. |
| **Ändra lösenord** | Uppdatera ditt SuperOffice CRM-lösenord. |
| **Papperskorg** | [Återskapa borttagna poster][3], såsom företag, kontakter eller ärenden. |
| **Logga ut** | Avsluta din session i SuperOffice CRM. |

> [!NOTE]
> I Inställningar och underhåll är endast **Lokala inställningar**, **Ändra lösenord** och **Logga ut** tillgängliga.

## Alternativ i huvudmenyn

### CRM

| Alternativ | Beskrivning |
|---|---|
| **Andra applikationer** | Få åtkomst till [externa verktyg eller länkar][1] integrerade i SuperOffice CRM. Liknar **Verktyg** i navigatorn. |
| **Ladda ned** | Visa och ladda ned verktyg som SuperOffice WebTools. |
| **App Store** | Utforska appar och integrationer i **SuperOffice App Store**. |
| **Kopiera genväg** | [Skapa genvägar till poster][15]. |
| **Dubblettinställningar** | Justera inställningar för [dubblettkontroll][8] för företag, projekt eller urval. |
| **Filter** | Redigera [filterinställningar][2]. |
| **Kunskapsbas** | Få åtkomst till externa dokument, FAQ och svarsmallar i Service. |
| **Anpassade objekt** | Pilotfunktion. |
| **Mål** | Lägg till eller uppdatera [försäljningsmål][17]. |
| **Redigera systemsignatur** | Redigera standardsignaturen för e-post. |
| **Inställningar och underhåll** | Öppna **Inställningar och underhåll** (endast för administratörer). |

### Inställningar och underhåll

| Alternativ | Beskrivning |
|---|---|
| **Ändra serienummer** | Uppdatera serienumret för din SuperOffice-installation. |
| **SuperOffice CRM** | Växla tillbaka till SuperOffice CRM. |

## Alternativ i Hjälpmenyn

| Alternativ | Beskrivning |
|---|---|
| **Resurscenter i appen** | Öppna guidade turer, användarguide och online-resurser. |
| **Community** | Öppna SuperOffice Hjälpcenter och Community-sidan. |
| **Kontakta support** | Skicka supportförfrågningar eller felrapporter. |
| **Hjälpindex (F1)** | Få direkt åtkomst till användarguider. |
| **Om** | Visa programinformation, inklusive version, användarinformation och kund-ID. |
