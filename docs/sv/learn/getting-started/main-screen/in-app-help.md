# Använda hjälp i appen

## Använda hjälpfilerna

Den här hjälpen har skapats för att ge dig lättillgänglig och utförlig information om hur du använder SuperOffice CRM. När du befinner dig i programmet kan du, oavsett vad du gör, alltid få hjälp med exakt den uppgift du arbetar med.

Du kan öppna hjälpen på olika sätt:

* Välj **Hjälpfiler (F1)** på **Hjälp**-menyn.
* Tryck på **F1**. Då öppnas hjälpavsnittet som hör till den aktuella skärmbilden.
* Klicka på **Hjälp**-knappen i de enskilda fönstren.
