# SuperOffice som din lösning (Windows)

Detta är det standardmässiga sättet att ansluta ditt e-postkonto, dokument och kalender till SuperOffice.

> [!NOTE]
> Dessa instruktioner är endast för Windows. Använder du en Mac? Se [instruktionerna för macOS][9].

## <a id="step-2"></a>E-post

[!include[Varför e-post](includes/why-connect-email.md)]

Välj ett av följande alternativ:

* [WebTools][2] (standard).

* [SuperOffice Inbox][1] - om du inte kan använda WebTools.

* [!include[SuperOffice för Outlook](includes/why-superoffice-for-outlook.md)]

## <a id="step-3"></a>Dokument

Standarddokumentintegrationen är **SuperOffice dokumentbibliotek** med [WebTools][2]. Om du valde WebTools för e-post i föregående steg är du redan klar.
