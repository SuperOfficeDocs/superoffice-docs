# Microsoft 365-integration som din lösning

Med Microsoft 365 som din lösning finns det flera sätt att ansluta ditt e-postkonto, dokument och kalender till SuperOffice.

## <a id="step-2"></a>E-post

[!include[Varför e-post](includes/why-connect-email.md)]

Välj ett av följande alternativ:

* [SuperOffice Inbox][1] (standard).

* [!include[SuperOffice för Outlook](includes/why-superoffice-for-outlook.md)]

## <a id="step-3"></a>Dokument

**SharePoint-dokumentbiblioteket** låter dig lagra alla dokument i SharePoint Online.

Du kan spara ett dokument i SuperOffice CRM och öppna det antingen i SuperOffice CRM eller SharePoint. Dokumentet kommer också att lagras på rätt kundkort i SuperOffice CRM så att andra användare kan se det. Du kan också dela det via en SharePoint-länk med dem som inte använder SuperOffice CRM.

> [!NOTE]
> Detta alternativ kräver ett Microsoft 365-abonnemang och är endast tillgängligt om din administratör har [konfigurerat Microsoft SharePoint-dokumentbiblioteket][3]. Alternativt kan du använda SuperOffice dokumentbibliotek med WebTools (eller Google Workspace).
