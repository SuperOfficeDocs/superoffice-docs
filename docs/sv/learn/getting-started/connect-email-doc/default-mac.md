# SuperOffice som din lösning (macOS)

[!include[Legacy version note](includes/legacy-version-note.md)]

Detta är det standardmässiga sättet att ansluta ditt e-postkonto, dokument och kalender till SuperOffice på macOS.

> [!NOTE]
> Dessa instruktioner är endast för macOS. Använder du en Windows-dator? Se [instruktionerna för Windows][9].

## <a id="step-2"></a>E-post

[!include[Varför e-post](includes/why-connect-email.md)]

Välj ett av följande alternativ:

* [SuperOffice Inbox][1] (standard).

* [!include[SuperOffice för Outlook](includes/why-superoffice-for-outlook.md)]

## <a id="step-3"></a>Dokument

Standarddokumentintegrationen på macOS är **SuperOffice dokumentbibliotek** med [WebTools för Mac][2].

## <a id="step-4"></a>Kalender

Välj ett av följande alternativ:

* [!include[SuperOffice Inbox](includes/so-inbox-for-invitations.md)]

* [!include[Synchronizer](includes/why-synchronizer.md)]

* [WebTools för Mac][2] - för att exportera din SuperOffice kalender till din iCalendar och kopiera dina SuperOffice kontakter till din Adressbok.

    > [!NOTE]
    > Detta är **endast en envägsimport**, från SuperOffice till din iCalendar, ingen synkronisering!
