# Ändra aktivt språk

Så här använder du SuperOffice på ett annat språk:

1. Gå till <i class="ph ph-user-circle" aria-hidden="true"></i> **Personliga inställningar** > **Lokala inställningar**.
1. Välj önskat språk i listan **Aktivt språk**.
1. Klicka på **OK**.
