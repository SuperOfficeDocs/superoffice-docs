# Ändra en inställning <i class="ph ph-user-circle" aria-hidden="true"></i>

Du kan ange inställningar för olika delar av SuperOffice CRM, till exempel antal poster som ska visas i historiklistan och standardvaluta för försäljningar. Alla inställningar definieras i Inställningar och underhåll, men användarna kan ändra en del av dessa själva i SuperOffice CRM.

![Gå till din personliga inställning och välj den inställning du vill ändra -screenshot][img13]

## Steg

1. Gå till <i class="ph ph-user-circle" aria-hidden="true"></i> **Personliga inställningar** > **Inställningar**.

2. I dialogrutan **Inställningar** markerar du ett alternativ i listan till vänster.

3. Ange önskad inställning i fönstret till höger.

4. Klicka på **Spara** för att spara ändringarna och stänga dialogrutan **Inställningar**. Eller klicka på **Avbryt** för att ignorera ändringarna.

> [!NOTE]
> För att ändra språk i Mobile CRM-appen klickar du på knappen **Uppgift** (<i class="ph ph-dots-three-circle-vertical" aria-hidden="true"></i>), tryck på **Inställningar**, klicka på **Språk** och välj bland alternativen.

## <a id="options"></a>Dialogrutan Inställningar – alternativ

| Knapp | Alternativ | Beskrivning |
|---|---|---|
| <i class="ph ph-list-checks" aria-label="Check"></i> | Standardvärden | Standardvärdena i SuperOffice CRM. Exempelvis standardbransch för ett nytt företag, standardvaraktighet för händelser, standardmall för utgående dokument. |
| <i class="ph ph-calendar-blank" aria-label="Diary icon"></i> | Kalender | Standardinställningar för kalender och händelser, till exempel standardstarttid för arbetsdagen och varaktighet på larm. |
| <i class="ph ph-toggle-right" aria-label="Functions"></i> | Funktioner | De allmänna inställningarna och funktionerna, bland annat kan du välja om du vill visa Dagens tips när du startar SuperOffice (Visa tips vid start). |
| <i class="ph ph-question" aria-label="Question mark"></i> | Kommunikation i appen | Inställningar för aviseringar i appen. |
| <i class="ph ph-at" aria-label="Email icon"></i> | E-post | Inställningarna för e-post och fax, till exempel den e-postklient du har installerat. |
| <i class="ph ph-bell" aria-label="Bell icon"></i> | Popup-avisering, avisering via e-post, SMS-avisering | Inställningar för ärendemeddelanden. |
| <i class="ph ph-user-circle" aria-label="Contact icon"></i> | Kontaktperson | Inställningar för kontakter. |
| <i class="ph ph-ticket" aria-label="Request icon"></i> | Ärende | Inställningar för ärenden. För aviseringsinställningar, se "Aviseringar via..." ovan. |
| <i class="ph ph-currency-circle-dollar" aria-label="Sale icon"></i> | Försäljning | Inställningar för säljfunktionen. |
| <i class="ph ph-sort-ascending" aria-label="Sort icon"></i> | Sortering | Hur medarbetare, listor och kontakter ska sorteras, till exempel om medarbetare ska sorteras på efternamn. |
| <i class="ph ph-gear" aria-label="Gear icon"></i> | System | Systeminställningar som hemland, egen valuta och standarddokumentarkiv. |
| <i class="ph ph-magic-wand" aria-label="Magic wand"></i> | Visuella effekter | Inställningar för visuella effekter, till exempel hur många poster historiklistan ska innehålla eller om du vill att söndag ska vara den första veckodagen. |
