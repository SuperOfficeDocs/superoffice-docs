# Visa aktiviteter

Vilka [aktiviteter][1] som visas beror på vilka av nedanstående alternativ du har markerat längst ned på detaljkortet:

* Uppföljningar
* Dokument
* Utskick
* Annat (chattposter och formulärinlämningar)

Du kan välja att visa alla aktiviteter eller en av typerna.

> [!TIP]
> Du kan även [filtrera aktiviteter][3] efter datum och användare.
