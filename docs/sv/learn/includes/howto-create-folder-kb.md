Så här skapar du en ny mapp:

1. Gå till den mapp som du vill skapa undermappen under.
2. Klicka på knappen **Ny mapp** överst på skärmen.
3. I fältet **Mapp** visas automatiskt namnet på den överordnade mapp som du har valt. Du kan välja en annan mapp här.
4. Ange ett beskrivande namn på mappen i fältet **Namn**.
5. I fältet **Beskrivning** kan du lägga in en kort beskrivning av vad mappen innehåller.
6. Klicka på **OK**. Den nya mappen skapas.
