# Projektfliken

Detaljkortet **Projekt** i SuperOffice CRM tillåter användare att visa och hantera [projektinformation][1] relaterad till specifika enheter som företag och kontakter. Denna guide ger en enhetlig översikt över hur man får tillgång till och arbetar med projektposter, vilket förbättrar möjligheten att spåra och hantera projektaktiviteter effektivt.

![Detaljkortet Projekt i Företagsfönstret -screenshot][img1]

## Tillgänglighet för Projektfliken

Detaljkortet **Projekt** är tillgänglig för:

| Enhet | Beskrivning av detaljkortet |
|---|---|
| [Företag][9] | Visar alla projekt kopplade till företaget. Minst en anställd måste vara registrerad som projektdeltagare för att ett projekt ska visas här. |
| [Kontakt][10] | Visar projekt där kontakten är projektdeltagare. |

## <a id="columns"></a>Kolumnöversikt för Projektfliken

Detaljkortet **Projekt** innehåller följande kolumner för att visa viktig information om varje projekt:

| Kolumn | Beskrivning |
|---|---|
| Slutfört | Kryssruta som indikerar om projektet är slutfört. En markerad kryssruta (<i class="ph ph-check" aria-hidden="true"></i>) betyder slutförd, medan en tom ruta betyder att det fortfarande pågår. |
| Projektnamn | Namnet på projektet. |
| Projekttyp | Anger vilken typ av projekt det är, som internt eller kundinriktat. |
| Status | Den aktuella statusen för projektet. |
| Projektnummer | Det unika identifieringsnumret som tilldelats projektet. |
| Beskrivning | Visar eventuell beskrivning av projektet. Håll muspekaren över gem-ikonen för att se hela texten. |
| Webbadress | En URL associerad med projektet, om tillämpligt. |
| Användar-ID | Användaren som skapade projektet. |

## Visa projekt i Projektfliken

1. Gå till relevant skärm, som Företag eller Kontakt.

1. Klicka på detaljkortet **Projekt** för att se alla projekt kopplade till enheten.

1. Dubbelklicka på ett projekt i listan för att öppna Projekt-skärmen och se detaljerad information.

### Filtrera projekt

Du kan [filtrera listan över projekt][8] efter datum, användare eller grupp genom att klicka på Filter-knappen. Detta hjälper dig att fokusera på specifika projektaktiviteter som kräver uppmärksamhet.

## Lägg till projektmedlemmar

För att [lägga till projektmedlemmar][4] direkt från detaljkortet **Projekt**:

1. Välj **Lägg till i projekt** för att öppna dialogrutan **Lägg till/redigera projektmedlemmar**.

1. Fyll i de obligatoriska fälten och klicka på **Spara**.

## Ytterligare tips

* **Snabb navigering:** Använd [Navigatören][7] för att snabbt växla mellan olika enheter och deras tillhörande **Projekt**-flikar.
