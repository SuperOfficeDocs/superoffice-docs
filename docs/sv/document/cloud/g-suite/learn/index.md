# Aktivera Google Workspace-integration

Med den kostnadsfria Google Workspace-integrationsappen för dokument kan du enkelt arbeta med dokument från SuperOffice CRM.

Google Workspace-appen (kallades tidigare Google G Suite-integration) integrerar Workspaces dokumenthanteringsfunktion med SuperOffice CRM så att du kan skapa, spara och hitta dokument i SuperOffice CRM.

Appen möjliggör även single sign-on, som gör att du kan logga in i SuperOffice CRM med dina inloggningsuppgifter för Google Workspace.

## Komma igång

Om du vill använda integrationen måste du vidta 2 konfigurationsåtgärder:

* Konfigurera Google Workspace för integrering med SuperOffice CRM Online.

* Konfigurera dokumentmallarna för användning enligt beskrivningen i artikeln "Konfigurera – dokumenthantering i Cloud Office".

> [!NOTE]
> Beroende på ditt företags säkerhetsinställningar kan du behöva administratörsbehörighet i SuperOffice CRM för att koppla samman systemen.
