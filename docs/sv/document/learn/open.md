# Öppna dokument

Du kan öppna och titta på ett dokument från dessa platser:

<!-- markdownlint-disable MD032 -->
[!include[List of Activities section tabs](../../learn/includes/list-activities-section-tabs.md)]
* Detaljkortet **Dokument** i fönstret Urval, om det finns ett aktivt dokumenturval.
* Sökresultaten på fliken **Dokument** i fönstret **Sök**.
<!-- markdownlint-restore -->

Du kan öppna ett dokument i [redigeringsläge eller läsläge][1]. När du öppnar dokumentet i redigeringsläge, är det bara du som kan redigera det. Andra användare kan öppna dokumentet i läsläge.

> [!NOTE]
> Om ditt företag använder ett dokumentbibliotek som SharePoint kan flera personer redigera ett dokument samtidigt.

Dokumentet öppnas i det program det skapades i.

Proceduren beror på hur dokument hanteras i din SuperOffice-installation. Se [Redigera dokument][2].
