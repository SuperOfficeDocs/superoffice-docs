# Fliken Projekt

På detaljkortet **Projekt** visas alla projekt där den aktiva kontakten är projektdeltagare.

Klicka på **Lägg till i projekt** om du vill lägga till projektdeltagare i ett projekt.

Dubbelklicka på ett projekt i listan för att öppna fönstret Projekt, som innehåller information om det aktiva projektet.
