# Detaljkort i fönstret Kontakt

Fönstret Kontakt innehåller följande detaljkort med information kopplad till den aktuella kontakten:

| Detaljkort | Beskrivning |
|---|---|
| [Aktiviteter][1] | Innehåller alla aktiviteter som är kopplade till kontakten. |
| [Försäljning][2] | Här finns information om eventuella försäljningar kopplade till kontakten. |
| [Ärenden][3] | Visar SuperOffice Service-ärenden för kontakten. |
| [Projekt][4] | Visar alla projekt där den aktiva kontakten är projektdeltagare. |
| [Relationer][5] | Visar eventuella kopplingar mellan den aktuella kontakten och andra kontakter eller företag. |
