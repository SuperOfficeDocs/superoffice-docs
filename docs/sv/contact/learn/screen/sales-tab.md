# Fliken Försäljning

Detaljkortet **Försäljning** visar försäljning som är kopplad till kontakten.

[!include[Contents of the Sales section tab](../../../learn/includes/sales-tab.md)]

[!include[Conf. columns](../../../learn/includes/tip-configure-columns.md)]

Om du dubbelklickar på en försäljning i listan öppnas fönstret Försäljning, som innehåller information om den aktuella försäljningen.

Klicka på **Lägg till** om du vill skapa en ny [försäljning][1].

Klicka på knappen **Filtrera** ![ikon][img1] om du vill [filtrera listan][3] på datum, användare eller grupper.

Om du markerar alternativet **Ta med intressenter** längst ned på detaljkortet visas även var [kontakten är involverad som intressent][2].
