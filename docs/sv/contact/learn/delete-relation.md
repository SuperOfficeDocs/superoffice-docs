# Ta bort relation (fönstret Kontakt)

[!include[Go to Contact screen](../../learn/includes/goto-contact.md)]

1. Högerklicka på önskad relation på detaljkortet **Relationer** och välj **Ta bort**.

1. Klicka på **Ja** i dialogrutan **Ta bort relation** för att bekräfta att du verkligen vill ta bort relationen.
