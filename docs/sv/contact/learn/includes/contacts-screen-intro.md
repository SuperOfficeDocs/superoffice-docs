Fönstret Kontakt används för att visa enskilda kontakter. Kontakter som är kopplade till ett företag kan också visas på detaljkortet **Kontakter** i fönstret Företag. Alla aktiviteter som kopplas till en kontakt läggs också till på [kontaktens företag][1].
