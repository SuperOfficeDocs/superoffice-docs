# Kontaktperson

Ett kontaktkort sparar värdefull information om all din kommunikation och interaktion med en person samt dokumentation om personens GDPR-samtycke. Allt du behöver veta om en kontakt finns i samma fönster.

![Fönstret Kontakt -screenshot][img3]

[!include[Intro](includes/contacts-screen-intro.md)]

I fönstret **Kontakt** visas varje post i databasen som ett indexkort med all information om kontakten.

Om du vill flytta mellan olika kontakter klickar du på knapparna föregående/nästa (![ikon][img2] ![ikon][img1]) längst ned till höger på korten.

Du kan [söka][1] efter kontakter med hjälp av kriterier som du anger. Du även skriva dokument, skapa händelser och skapa nya kontakter.

## <a id="section-tabs"></a>Detaljkort under Kontakt

Längst ned i fönstret **Kontakt** finns som standard fem [detaljkort][16] med information kopplad till den aktuella kontakten:

| Detaljkort | Beskrivning |
|---|---|
| [Aktiviteter][11] | Innehåller alla aktiviteter som är kopplade till kontakten. |
| [Försäljning][12] | Här finns information om eventuella försäljningar kopplade till kontakten. |
| [Ärenden][13] | Visar SuperOffice Service-ärenden för kontakten. |
| [Projekt][14] | Visar alla projekt där den aktiva kontakten är projektdeltagare. |
| [Relationer][15] | Visar eventuella kopplingar mellan den aktuella kontakten och andra kontakter eller företag. |
