# Kontaktperson

Fönstret **Kontakt** ger en snabb överblick över all kommunikation och interaktion med en person. Här kan du hantera kontaktuppgifter, spåra aktiviteter och komma åt GDPR-relaterad dokumentation – allt från en och samma skärm.

![Fönstret Kontakt visar nyckelinformation om en kontakt, detaljkort och navigeringsknappar. -screenshot][img3]

Kontakter som är kopplade till ett företag kan också visas på detaljkortet **Kontakter** i [fönstret Företag][8]. Alla aktiviteter som läggs till på en kontakt kopplas automatiskt till företaget.

Du kan navigera mellan kontakter med knapparna **Föregående** och **Nästa** (<i class="ph ph-caret-circle-left" aria-hidden="true"></i><i class="ph ph-caret-circle-right" aria-hidden="true"></i>) längst ned till höger på skärmen.

Använd funktionen **Sök** för att hitta specifika kontakter eller filtrera poster. Från fönstret Kontakt kan du även skapa dokument, skicka e-post och generera uppföljningar.

## Flikar på kontaktkortet

| Flik | Beskrivning |
|---|---|
| [Kontakt][2] | Huvudflik för att ange viktig kontaktinformation, såsom titel, telefon, e-post och adress. |
| Detaljer | Visar utökad kontaktinformation, såsom födelsedatum och föredraget språk. |
| [Mer][19] | Anpassade fält för att ange ytterligare kontaktinformation. |
| [Intressen][2] | Kryssrutor för att ange kontaktens intressen för urval eller kampanjer. En punkt visas när intressen är valda. |
| [Anteckning][20] | Ett fritextfält för att lägga till anteckningar om kontakten. Anteckningar markeras med en punkt när de innehåller text. |
| [www][21] | Snabbåtkomst till en webbadress för att visa externa data eller webbsidor relaterade till kontakten. |
| [ERP][22] | Kopplar kontakten till ERP-systemet för integration eller synkronisering. |

## <a id="section-tabs"></a>Detaljkort under Kontakt

Längst ned i fönstret **Kontakt** finns följande [detaljkort][16] som standard:

| Detaljkort | Beskrivning |
|---|---|
| [Aktiviteter][11] | Innehåller alla aktiviteter som är kopplade till kontakten. |
| [Försäljning][12] | Innehåller information om eventuella försäljningar kopplade till kontakten. |
| [Ärenden][13] | Visar SuperOffice Service-ärenden för kontakten. |
| [Projekt][14] | Visar alla projekt där den aktiva kontakten är projektdeltagare. |
| [Relationer][15] | Visar eventuella kopplingar mellan den aktuella kontakten och andra kontakter eller företag. |
