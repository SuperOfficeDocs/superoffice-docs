# Granska rapporter

Detta fungerar lite olika för de olika rapporttyperna. För att granska rapporttyperna korstabell eller etikett kan du gå vidare till något av avsnitten nedan:

* [Visa korstabeller][7]
* [Granska etiketter][2]

> [!TIP]
> Om du är osäker på vilken rapporttyp du har kan du titta på [miniatyrbilden][1] överst till höger i fönstret **Rapporter**.

1. Välj önskad rapport.

1. Klicka på knappen **Förhandsgranska** under kriterielistan. En PDF-fil genereras i webbläsaren.

1. [!include[Open PDF](includes/step-open-pdf.md)]

Du kan spara PDF-filen inifrån visningsprogrammet för PDF-filer, så att du kan öppna den senare.
