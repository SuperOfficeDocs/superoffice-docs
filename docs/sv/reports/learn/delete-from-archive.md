# Ta bort rapporter från rapportarkivet (detaljkortet Dokument)

Du kan ta bort sparade rapporter ur rapportarkivet.

1. Gå till **Rapporter**.
2. Markera den rapport som du vill ta bort i rapportarkivet (detaljkortet **Dokument**) längst ned i fönstret.
3. Klicka på **Ta bort**-knappen eller högerklicka på rapporten och välj **Ta bort**.
4. Bekräfta borttagningen i dialogrutan som visas.
