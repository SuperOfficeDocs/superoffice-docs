# Standardrapporter

I fönstret **Rapporter** kan du också köra olika typer av standardrapporter. Dessa rapporter är sorterade efter typ, och du hämtar dem genom att klicka på flikarna överst i fönstret Rapporter (se [Visa rapportegenskaper][1]).

SuperOffice CRM levereras med flera olika standardrapporter, men ett företag kan också skapa sina egna rapporter i SuperOffice Reporter Studio, som är ett tillägg. När du använder fönstret Rapporter märker du ingen skillnad på dessa två olika rapportursprung. Därför kallar vi fortsättningsvis alla rapporter för standardrapporter.

> [!TIP]
> Du kan spara standardrapporter som du ofta använder som [favoritrapporter][2], så att du lättare kan få tillgång till dem. Dessutom kan du redigera standardrapporter och spara dem som favoriter.

## <a id="type" />Typer

| Typ | Beskrivning |
|---|---|
| Företag/kontakt | Visar företags-/kontaktrapporter. |
| Projekt | Visar projektrapporter. |
| Försäljning | Visar försäljningsrapporter. |
| Aktivitet | Visar rapporter för händelser och dokument. |
| Urval | Visar urvalsrapporter. |

## Redigera och ta bort standardrapporter

Du kan varken redigera eller ta bort standardrapporterna i sig. Det du kan göra är att [spara en standardrapport som en favoritrapport][3] och sedan [justera sökkriterierna som du vill][4].
