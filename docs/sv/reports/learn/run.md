# Visa en rapport

Med hjälp av dina egna favoritrapporter eller de publicerade rapporterna i fönstret **Rapporter** kan du generera rapporter som innehåller data från SuperOffice CRM.

> [!NOTE]
> Rapportgenerering baseras på samma åtkomstbehörigheter som SuperOffice CRM. Det innebär att om det är något du inte har läsbehörighet för i SuperOffice CRM, så kommer den informationen inte att inkluderas i de rapporter du skapar.

När du vill köra en rapport i fönstret **Rapporter** kan du välja att visa rapporten på skärmen, skriva ut den på en skrivare eller spara den som en fil. I vart och ett av de två sista fallen är det en bra idé att visa rapporten på skärmen först för att se till att den ser ut som du vill.

För att [spara en rapport som ett dokument][1] (typ = Rapport) i SuperOffice CRM, klicka på knappen **Spara** i fönstret **Rapporter**.
