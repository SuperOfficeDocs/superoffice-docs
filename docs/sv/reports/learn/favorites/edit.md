# Redigera favoritrapporter

> [!NOTE]
> Detta förutsätter att du har minst en [favoritrapport][1].

1. [!include[How to-goto-reports](../includes/goto-reports.md)]
2. Välj fliken **favorit**. Favoritrapporterna visas i rapportlistan.
3. Dubbelklicka på rapporten som du vill redigera eller markera den och klicka på **Redigera**.
4. Uppdatera namn och beskrivning efter behov.
5. Klicka på **OK**.

> [!NOTE]
> Rapporten ändras omedelbart och du kan inte ångra dina ändringar.
