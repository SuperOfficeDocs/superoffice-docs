# ANSI-filer

ANSI är den teckenuppsättning som används av Microsoft Windows. Detta innebär att koderna som används för varje tecken är speciellt anpassade till Windows. I en ANSI-fil som sparas som en textfil är fälten i posterna vanligtvis tabbavgränsade. Varje post separeras med en vagnretur och linjematning (CR/LF).

> [!TIP]
> Du kan öppna datafilen som du vill använda för import i valfritt ordbehandlingsprogram innan du importerar den, så att du kan se vilket tecken som skiljer fälten och posterna i filen. Innan du börjar importera, måste du ange vilka tecken som ska användas som avgränsare. Detta kan du göra i dialogrutan **Avancerat** (se [Ange avancerade importinställningar][1]).
