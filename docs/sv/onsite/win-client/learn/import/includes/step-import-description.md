2. Klicka på ![ikonen][img1] bredvid fältet **Importbeskrivningsfil** och leta upp en importbeskrivningsfil i den dialogruta som öppnas.

3. Importbeskrivningsfilen innehåller information om vilken datafil som ska importeras, och den här filen visas sedan automatiskt i fältet **Importfil** . Om ingen fil visas här, kan du klicka på ![ikon][img1]-knappen bredvid fältet och navigera till önskad importfil.

    > [!NOTE]
    > Importbeskrivningen och importfilen måste innehålla samma antal kolumner. Du kan inte infoga vilken fil som helst i fältet **Importfil**.
