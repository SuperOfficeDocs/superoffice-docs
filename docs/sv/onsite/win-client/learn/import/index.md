# Import

[!include[Back up database before import](../../../../admin/import/learn/includes/caution-backup-before-import.md)]

I fönstret **Import** kan du infoga viktiga externa data i SuperOffice-databasen. Du kan till exempel importera ett kundindex så att du kan använda det i SuperOffice CRM. Mer specifikt kan du i standardversionen av inställningar för Windows-klienten importera följande data till SuperOffice-databasen:

* Företag med tillhörande användardefinierade fält och intressen
* Kontakter med tillhörande användardefinierade fält och intressen
* Projekt med associerade användardefinierade fält
* Uppföljningar
* Försäljning
* Produkter
* Valutor
* Postnummer
* Helgdagar

Du kan också [importera företag/kontakter och produkter i Inställningar och underhåll][9].

Fälten i fönstret **Import** och deras användning beskrivs under [Importfiler][2]. Innan du börjar importera, bör du också läsa igenom [Importrutiner][1]
