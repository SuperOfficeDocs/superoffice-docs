# Uppdatera information i databasen

Vid något tillfälle kan du behöva uppdatera databasen. Vissa organisationer väljer att köra en uppdatering av ett kundindex, till exempel en gång i månaden eller på begäran.

Du kan använda [importfunktionen][1] i inställningar för Windows-klienten för att köra nya uppdateringar, till exempel för att ange ett fält med ny information för alla företag som du har i databasen.
