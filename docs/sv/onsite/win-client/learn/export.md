# Exportera

Export av data från SuperOffice-databasen görs med hjälp av fönstren Urval eller Rapporter i SuperOffice CRM.

Om du vill exportera en fil för att importera den via fönstret **Importera** i inställningarna för Windows-klient, exportera den som en [ANSI-fil][1]. Importstegen för ANSI-filer är desamma som när man [importerar filer][2].
