# Fritextsökning och Travel

Fritextsökning är vanligtvis tillgänglig för alla Travel-användare, men om din organisation använder Area Management kan du definiera att specifika områden inte kan använda fritextsökningar. Utan fritextsökningar kommer SuperOffice-databasen som Travel-användare tar med sig att ta upp betydligt mindre plats.

I listan **Aktivera fritextsökning för Travel-områden** kan du se vilka områden som är definierade och för vilka områden som fritextsökning är tillgänglig.

Om du markerar **Aktivera automatiskt för nya områden** längst ned i fönstret kan alla nydefinierade områden använda fritextsökning. Om du vill ta bort den här åtkomsten för ett område, måste du konfigurera detta med hjälp av Area Management.
