Ett dynamiskt urval kan liknas vid en sparad sökning som utförs varje gång du visar urvalet. Varje gång du lägger till en post som uppfyller kriterierna i ett dynamiskt urval uppdateras urvalet med den nya posten, i motsats till ett statiskt urval. Du kan till exempel [skapa ett dynamiskt urval][1] som innehåller alla ärenden som tillhör en viss kategori och som har hög prioritet.
