# Dynamiska urval

Ett dynamiskt urval kan liknas vid en sparad sökning som utförs varje gång du visar urvalet. Varje gång du lägger till poster i SuperOffice CRM som uppfyller sökkriterierna i ett dynamiskt urval uppdateras urvalet med den nya medlemmen, i motsats till ett [statiskt urval][1]. Använd ett dynamiskt urval när du vill ha en uppdaterad översikt över företag, kontakter, försäljningar, projekt, dokument, händelser, ärenden eller produkter som uppfyller en viss uppsättning sökkriterier.

## Exempel på vad du kan använda dynamiska urval till

### Företag

Du kan skapa ett dynamiskt urval som innehåller alla företag i kategorin "Leverantör" med huvudkontor i Tyskland, om du planerar att utöka verksamheten i Tyskland.

### Försäljning

Du kan skapa ett urval som innehåller alla försäljningar som du har genomfört under första kvartalet, eventuellt alla dina försäljningar med statusen **Uppskjuten**. På så sätt kan du bättre se trender och bedöma vad som är viktigt i det fortsatta försäljningsarbetet.

### Projekt

Du kan skapa ett urval som innehåller alla projekt av typen "Internt" och som är pågående. Sedan kan du bedöma vilka som kan avslutas/slutföras och vilka som det är värt att jobba vidare med.

### Dokument

Du kan skapa ett urval som innehåller alla dokument som du har skapat den sista månaden. På så sätt kan du få en översikt över vilka dokument som är gjorda och vilka du eventuellt har glömt.

### Uppföljningar

Du kan skapa ett urval som innehåller alla händelser för en viss kund, så att du kan bilda dig en uppfattning om vilket förhållande ni har till kunden innan du genomför ett säljmöte med dem.

Du kan även skapa ett urval som innehåller alla aktuella händelser för den aktuella månaden, så att du får en översikt över dina möter denna månad och om du behöver upprätta några fler.

### Produkter

Du kan skapa ett urval som innehåller alla produkter som har sålts i år baserat på försäljningar som innehåller offerter och har statusen "Såld". Urvalet kan begränsas till vissa produktkategorier, produktfamiljer och så vidare beroende på vad för slags statistik du har behov av.

Om ni tänker uppgradera en av produkterna ni säljer från version 2 till 3 kan du skapa ett urval av alla kunder som köpt version 2 och använda urvalet som underlag för en kampanj.

### Ärendehantering

Du kan skapa ett urval som innehåller alla öppna ärenden som inte har uppdaterat på över en vecka. Du kan därefter följa upp de ärendehandläggare som ansvarar för dessa ärenden.
