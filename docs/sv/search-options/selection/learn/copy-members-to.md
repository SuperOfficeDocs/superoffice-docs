# Kopiera till urval

Det kan vara praktiskt att kopiera utvalda urvalsmedlemmar till ett annat urval, till exempel om du har skapat ett kundurval och vill inkludera leverantören i samma urval.

1. [!include[Start](includes/steps-start-task.md)]

2. Klicka på **Uppgift**-knappen och välj uppgiften **Kopiera till urval**.

3. I dialogrutan **Kopiera till urval** väljer du ett av följande:

    * **Kopiera till befintligt, statiskt urval**: Markera det här alternativet om du vill kopiera det aktiva urvalet till ett statiskt urval som redan finns. Ange i så fall vilket urval du vill kopiera till.

    * **Kopiera till nytt urval**: Markera det här alternativet om du vill [skapa ett helt nytt urval][2], som ska innehålla medlemmarna i det aktiva urvalet. I det här fallet måste du också ange ett namn på det nya urvalet, vilken typ av urval det är och vem det är synligt för.

4. Klicka på **OK**. Urvalet kopieras till det valda målet.

## Varför kan jag inte se alla mina urval?

Du kan inte kopiera ett urval till ett befintligt dynamiskt urval eller ett kombinationsurval. Därför kan du endast välja bland befintliga statiska urval.

## Varför kan jag inte välja att det nya urvalet ska vara dynamiskt?

Eftersom ett dynamiskt urval inte innehåller några fysiska kontakter, utan endast en uppsättning sökkriterier, visas alternativen **Statiskt** och **Dynamisk** i listrutan **Urvalet är** endast när du kopierar dynamiska urval. När du kopierar ett statiskt urval innehåller listrutan **Urvalet är** endast alternativet **Statiskt**.

För att [skapa en kopia][3] av ett dynamiskt urval kan du använda funktionen **Kopiera urvalet** på fliken **Uppgift**.
