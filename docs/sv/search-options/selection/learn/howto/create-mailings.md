# Skapa utskick

Du kan göra ett utskick till alla företag i ett urval. Utskicket kan vara ett e-postmeddelande, en e-postbilaga (dokument) eller ett SMS.

1. Öppna det urval du vill använda för utskicket.

2. Klicka på **Uppgift**-knappen.

3. Klicka på något av följande alternativ:

    * **Skapa ett e-postutskick**: Välj detta alternativ för att [skicka e-post till hela eller delar av kundgruppen][1].

    * **Skapa ett SMS-utskick**: Välj detta alternativ för att [skicka SMS][2] till hela eller delar av kundgruppen via mobiltelefon.

    * **Skapa ett dokumentutskick**: Välj detta alternativ för att [skapa sammanfogade dokument][3] (PDF) för att skriva ut och skicka via vanlig post till hela eller delar av kundgruppen.

    > [!TIP]
    > De här alternativen är också tillgängliga från knappen **Marknadsföring** i navigatorn.

I SuperOffice CRM läggs utskicket till i detaljkortet **Utskick** på i fönstret Urval för att hjälpa dig att hålla reda på alla utskick som är kopplade till varje urval.
