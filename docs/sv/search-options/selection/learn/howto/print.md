# Skriv ut urval

1. [!include[Start](../includes/steps-start-task.md)]

1. Ställ in de första fem kolumnerna du vill inkludera. Gruppering eller kolumnberäkningar som tillämpas på ditt urval kommer att visas i den utskrivna rapporten.

1. Klicka på **Uppgift**-knappen och välj **Skriv ut urval**.

1. I fönstret som visar innehållet som ska skrivas ut, klickar du på **Skriv ut**.
