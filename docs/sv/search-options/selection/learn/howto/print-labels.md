# Skriva ut etiketter

Etikettutskrift är användbart till exempel när du ska skicka brev till många mottagare och ska skapa adresslappar för att klistra på kuverten.

1. [!include[Start](../includes/steps-start-task.md)]

2. Välj fliken **Företag/kontakt**.

3. Klicka på **Uppgift**-knappen och välj **Skriv ut etiketter**.

4. I dialogrutan **Välj etikett** väljer du det etikettformat du vill ha och klickar på **OK**.

    > [!NOTE]
    > I den här dialogrutan kan du även [lägga till nya etikettformat][2] och redigera eller ta bort befintliga etikettformat.

5. Öppna den genererade PDF-filen och skriv ut den.
