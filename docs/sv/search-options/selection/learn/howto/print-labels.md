# Skriva ut etiketter

Etikettutskrift kan vara användbart när du behöver skicka brev till flera mottagare och snabbt vill skapa adresslappar att fästa på alla kuvert.

> [!NOTE]
> Från version 10.3.10 använder etikettutskriften webbläsarens utskriftsfunktion istället för PDF-utskrift.

## Steg

1. [!include[Start](../includes/steps-start-task.md)]

1. Gå till fliken **Företag/kontakt**.

1. Klicka på **Uppgift**-knappen och välj **Skriv ut etiketter**.

1. Välj önskat etikettformat. Alla format är konfigurerbara och tillgängliga för alla användare i företaget, så att du kan justera dem efter behov.

    ![Urval uppgiftsutskrift - välj etikettformat -screenshot][img1]

1. Granska de nödvändiga inställningarna och klicka på **OK**.

    ![Urval - Utskriftsinställningar och nödvändiga inställningar -screenshot][img2]

1. Justera webbläsarens utskriftsinställningar (såsom skala, marginaler, orientering och papperstorlek) för att säkerställa att adresserna passar korrekt inom etikettområdet utan att överlappa.

    ![Urval - Utskriftsinställningar i webbläsaren -screenshot][img3]

    > [!TIP]
    > Om du använder webbläsarens utskriftsfunktion för andra objekt kan du behöva **justera etikettsinställningarna varje gång** för att säkerställa korrekt justering.

1. Klicka på **Skriv ut**.
