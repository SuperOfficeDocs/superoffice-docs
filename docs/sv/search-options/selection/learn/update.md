# Redigera urval

> [!NOTE]
> För att kunna redigera ett urval måste du äga det eller ha administratörsbehörighet, eller så måste du ha redigeringsåtkomst via inställningen för **Funktioner**.
>
> Du kan inte redigera fältet **Urval av**.
>
> Du kan göra om ett dynamiskt urval och ett kombinationsurval till ett statiskt urval, men du kan inte göra om ett statiskt urval till ett dynamiskt urval eller ett kombinationsurval.

## Ändra ett befintligt urval

1. Öppna urvalet du vill uppdatera. Se [Använd sökskärmen][1].

1. Gå till fliken **Detaljer**.

1. Klicka på <i class="ph ph-pencil-simple" aria-label="Edit"></i>. Fliken **Urval** växlar till redigeringsläge.

1. I listan **Urval** är kan du ändra ett dynamiskt eller kombinerat urval till ett statiskt urval.

1. När du har gjort de nödvändiga ändringarna klickar du på **Spara** för att stänga och spara ändringarna. Klicka på **Avbryt** om du vill avvisa ändringarna.

## Varför kan jag inte göra om urvalet till dynamiskt?

När medlemmar har lagts till i ett urval gäller följande:

* Du kan inte göra om ett statiskt urval till dynamiskt, eftersom det statiska urvalet inte har några sökkriterier.

* Det går däremot bra att göra om ett dynamiskt urval till ett statiskt urval. Då sparas sökresultatet som ett fast, statiskt urval, som inte uppdateras även om du senare lägger till fler företag som uppfyller sökkriterierna från det dynamiska urvalet.

* Det går däremot bra att göra om ett kombinationsurval till ett statiskt urval. Då sparas sökresultatet som ett fast, statiskt urval.

## <a id="delete"></a>Ta bort urval

1. Öppna urvalet du vill radera.

1. Klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Task button"></i> och välj **Ta bort detta urval**.

1. Klicka på **Ja** när du tillfrågas om du vill ta bort urvalet.
