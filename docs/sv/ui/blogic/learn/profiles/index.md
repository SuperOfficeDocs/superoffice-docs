# Profil

[!include[Restricted access](../../../../learn/includes/note-insufficient-rights.md)]

Profilerna i systemet används för att konfigurera skärmobjektens utseende och beteende.

[!include[Legacy](../includes/legacy-profiles.md)]

Till exempel:

* Flytta eller lägg till kolumner i listan med ärenden på skärmen Kontakt
* lägg till eller ta bort fält i huvudet på skärmen Ärende (huvudena på skärmen **Kontakt** och Företag)
* Anpassa skärmen med sökresultat efter dina behov

Varje skärmobjekt kan ha personliga, gemensamma och globala profiler.

* Personliga profiler gäller för enskilda användare.
* Delade profiler är kopplade till en roll och gäller för alla användare som har den aktuella rollen.
* En global profil används som standardprofil och gäller för alla dem som inte har en personlig profil eller inte har tilldelats en delad profil utifrån den roll de tillhör.

Genom att använda profiler kan SuperOffice Service anpassas till individuella användare, grupper av användare eller hela organisationen.
