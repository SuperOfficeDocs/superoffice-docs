# Personliga profiler

[!include[Legacy](../includes/legacy-profiles.md)]

Personliga profiler gäller för enskilda användare. Du konfigurerar en personlig profil på olika skärmar, t.ex. i listan över ärenden på skärmen Kontakt, i området med egenskaper för ärenden på skärmen Ärende eller på skärmen med sökresultat. För att förklara hur detta fungerar ger vi ett konkret exempel. I exemplet nedan antar vi att du vill lägga till ett fält på skärmen Ärende.

1. Öppna fönstret Ärende och välj fliken **Detaljer**.

2. Klicka på <i class="ph ph-wrench" aria-label="Wrench"></i> bredvid egenskaperna för ärendet. Skärmen **Redigera objektsprofil** visas.

3. Där det står **Nytt fält** finns två tomma fält.
    * I rutan längst till vänster anger du fältnamnet.
    * Klicka på <i class="ph ph-caret-down" aria-label="Chevron"></i> och välj vilket fält i databasen som du vill hämta information från.

4. Klicka på **Lägg till** intill de två fälten där du nyss har angett information. Det nya fältet läggs till under förhandsgranskningen överst på skärmen.

5. Klicka på **OK**. Ändringarna sparas och nästa gång du visar skärmen är det nya fältet med.

6. För att återställa standardinställningarna går du tillbaka till skärmen **Redigera objektsprofil** och klickar på knappen **Återställ standard (tar bort den här profilen)** längst ned på skärmen.
