# Extramenyer

[!include[Legacy](../includes/legacy-extra-menus.md)]

På skärmen **Extramenyer** kan du lägga till extramenyer (knappar, menyer, länkar, genvägar, listor o.s.v.) i SuperOffice så att de funktioner och den information som du behöver alltid är enkla att komma åt.

## Var kan jag lägga till extramenyer?

* **Toppmeny**

* Panelen **Status** på instrumentpanelen
  * Knapparna ![ikon][img1] **Uppgift** på huvudskärmarna
  * Menyknappar i extratabellsvyn

* **Navigator**

  * Ytterligare alternativ på navigatorknapparna
  * Ytterligare alternativ under **Systemdesign**
  * Ytterligare alternativ på knappen **Nytt** i det övre fältet.

## Vad kan jag lägga till som extramenyer?

* Webbadresser: Länkar till webbsidor eller specifika poster i SuperOffice.
* Skärmar: Genvägar till specifika skärmar.
* Skripter/makron: Knappar eller menyalternativ som utlöser ett skript eller makro.
* Urval: Genväg till ett urval.

Se även [Egenskaper för meny][1].
