# Lägg till funktion i menyknappen

[!include[Legacy](../includes/legacy-extra-menus.md)]

För att lägga till ett makro i knappen **Uppgift** i fönstret Företag, Kontakt eller Ärende eller fliken **Meddelande**:

1. Välj <i class="ph ph-list" aria-label="Main menu"></i> > **Systemdesign** > **Extramenyer**.
2. Klicka på **Toppmeny** för att visa undermapparna.
3. Håll muspekaren över någon av dessa mappar: **Visa Ärende**, **Visa kontakt**, **Visa företag** eller **Meddelande-meny**. Klicka sedan på **+**. Skärmen **Egenskaper för meny** visas.
4. Skriv knappens namn i fältet **Namn**. Namnet ska vara ganska kort för att passa på knappen.
5. Välj **Använd skript** och välj ett makro.
6. Välj önskad placering i listan **Placering** (0 = överst).
7. Klicka på **OK**.
8. Öppna ett ärende, företag, ärende, meddelande eller en kontakt och klicka på knappen **Uppgift** och kontrollera att knappen visas och fungerar som det är tänkt.

> [!NOTE]
> Du kan också lägga till ett makro till knappen **Uppgift** när du [skapar eller redigerar ett makro][1].
