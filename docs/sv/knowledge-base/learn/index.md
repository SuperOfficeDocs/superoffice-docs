# Kunskapsbas

**Kunskapsbasen** är tillgänglig från **Huvudmenyn** i det övre fältet och ger ärendehandläggare åtkomst till sparad information på en enda plats, vilket gör det enklare att hitta information och göra informationen tillgänglig för kontakter. Du kan använda kunskapsbasen både för sökningar och för att hämta information som kan återanvändas i kommunikationer med kontakter. Kunskapsbasen består av följande huvuddelar:

* **FAQ**: Här finns en [lista över vanliga frågor][2] med tillhörande svar. Dessa poster kan göras tillgängliga i ett webbaserat kundcenter där ,kontakterna själva kan söka efter svar och ärendehandläggare kan lägga in länkar till FAQ-poster när de svarar på förfrågningar.

* **Svarsmallar**: Här hittar du en lista över alla [svarsmallar][3] i systemet. En svarsmall består av färdig text som kan klistras in i ett meddelande som du skriver till en kontakt.

* **Externa dokument**: Här kan du [lägga till dokument][1] som ska vara tillgängliga för alla som använder SuperOffice Service, även oregistrerade kontakter.

* **E-postadresser**: Här kan du [skapa en lista över e-postadresser][4] som du ofta använder, t.ex. till personer som varken är användare eller är registrerade som kontakter i SuperOffice CRM. Dessa adresser blir då tillgängliga på de flesta ställen där du kan vilja skicka ett e-post.
