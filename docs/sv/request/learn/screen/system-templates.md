# Inställningar för systemmallar

På fliken **Systemmallar** kan du ange vilka svarsmallar systemet ska använda.

> [!TIP]
> Använd sökfältet längst upp för att hitta specifika mallar.

## Ärenden

I avsnittet **Ärenden** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för avisering om aktiva ärenden | Den svarsmall som ska användas för att meddela användare om aktiva ärenden. Detta är ett e-postmeddelande som skickas ut varje natt till användare som har flera dagar gamla aktiva ärenden. Antalet dagar ställs in på fliken **Inställningar** på skärmen **System** i SuperOffice-ärenden. |
| Svarsmall för avisering om nytt meddelande | Den svarsmall som ska användas för att meddela användare när det kommer in ett nytt meddelande i ett ärende. |
| Svarsmall för avisering om aktiverat ärende | Den svarsmall som ska användas för att meddela användare om att ett uppskjutet ärende aktiverades. |
| Svarsmall för avisering om eskalerat ärende | Den svarsmall som ska användas för att meddela användare om att ett ärende eskalerades. |
| Svarsmall för avisering om nytt ärende | Den svarsmall som ska användas för att meddela användare om att de har tilldelats ett ärende. |
| Svarsmall för avisering om övertagande av ärende | Detta fält visar vilken svarsmall som ska användas för att meddela användare om att en annan användare har tagit över ett ärende. |
| Begär mall för att skriva ut ett ärende | I det här fältet visas vilken svarsmall som ska användas när ett ärende skrivs ut. |

## Favoriter

I avsnittet **Favoriter** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för att meddela om ett nytt meddelande i favoritlistan | den svarsmall som ska användas för att meddela användare när ett nytt meddelande kommer in i ett ärende på favoritlistan. |
| Svarsmall för avisering om aktiverat ärende i favoritlistan | Detta fält visar vilken svarsmall som ska användas för att meddela användare om att ett uppskjutet ärende på favoritlistan aktiverades. |
| Svarsmall för avisering om eskalerat ärende på favoritlistan | Detta fält visar vilken svarsmall som ska användas för att meddela användare om att ett ärende på favoritlistan eskalerades. |
| Svarsmall för avisering om övertagande av ärende på favoritlistan | Detta fält visar vilken svarsmall som ska användas för att meddela användare om att en annan användare har tagit över ett ärende på favoritlistan. |

## Kunder

I avsnittet **Kunder** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för meddelande till nya kontakter | Den svarsmall som ska skickas till nya kontakter hos kunder när de registreras för första gången. |
| Svarsmall för meddelande till kontakt vid mottagning av ärende | Den svarsmall som ska användas för att skicka automatiska standardsvar till kontakter när e-post från dem kommer in. Detta värde åsidosätts för varje [e-postlåda][1]. |
| Svarsmall för utskick av lösenord till kontakt | Den svarsmall som ska skickas till kontakter när de ber att få sitt lösenord skickat till sig. |
| Svarsmall för utskick av bekräftande e-post till kontakt | Den svarsmall som används för bekräftande e-post som skickats till kontakter. |
| Svarsmall för kundchattlogg | Den svarsmall som används för att skicka chattloggar till kunder som har kontaktat supporten i chatten. |
| Svarsmall för ny länk | Den svarsmall som används för att skicka en ny länk till kunden när den gamla länken går ut. |

## FAQ

På fliken **FAQ** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för formatering av FAQ-frågor | Den svarsmall som ska användas vid publicering av FAQ-frågor. |
| Svarsmall för formatering av FAQ-svar | Den svarsmall som ska användas vid publicering av FAQ-svar. |

Dessa låter dig definiera din egen formatering för föreslagna frågor och svar från FAQ.

## Veckovis statistik

På fliken **Veckovis statistik** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för veckovis statistik | Den svarsmall som ska användas för veckovis statistik.
| E-postadresser för veckovis statistik | Här kan du lägga till en eller flera (separerade med kommatecken) e-postadresser som det ska skickas veckovis statistik för SuperOffice till. |

## Så här byter du mall

För att välja en annan mall kan du klicka på listknappen ![ikon][img1] intill det aktuella fältet och välja en mall i den lista som visas.
