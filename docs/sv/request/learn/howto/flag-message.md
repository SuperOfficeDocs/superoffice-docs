# Hålla koll på meddelanden

Komplexa ärenden kan ta mycket tid att lösa. Detta kan innebära en hel del kommunikation mellan kunden och flera ärendehanterare.

Om du vill göra det enklare att hitta och sortera flera meddelanden i ett ärende kan du flagga viktiga meddelanden och filtrera meddelandelistan.

## Flagga ett meddelande

1. [Öppna ärendet][1].
1. Klicka på fliken **Meddelanden**.
1. Markera ett meddelande genom att klicka på den grå flaggan (![ikon][img1]). Flaggan blir röd (![ikon][img2]). Meddelandet är nu "flaggat".

## Filtrera meddelanden

1. [Öppna ärendet][1].
1. Klicka på fliken **Meddelanden**.
1. I det övre högra hörnet klickar du på filterlistan och väljer ett filter. Meddelandena filtreras.
