# Ta över ansvaret för ärenden

Detta är särskilt relevant för otilldelade ärenden (ärenden utan en angiven ägare). För att ta över ansvaret för ett ärende är det viktigt att du följer proceduren nedan för att identifiera att du har tagit över ansvaret för ärendet. Annars finns det risk för att andra besvarar samma ärende samtidigt.

1. [Öppna ärendet][1].
1. Acceptera:

    * Klicka på ![ikon][img1] knappen **Uppgift** och välj **Acceptera**.
    * Eller välj dig själv i fältet **Ägare**.
