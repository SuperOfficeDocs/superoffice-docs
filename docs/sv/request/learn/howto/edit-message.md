# Redigera meddelande

Endast användare med den funktionella rättigheten *general administrator* kan ändra egenskaperna för ett meddelande i ett ärende. Det går dock inte att redigera själva meddelandetexten.

1. [Öppna ärendet][1].
1. Klicka på fliken **Meddelanden**.
1. Klicka på knappen ![ikon][img1] **Uppgift** och välj **Byt**.
1. Här har du följande alternativ:
    * **Författare**: Visar namnet på den person som skapade ärendet.
    * **Förbrukad tid**: Visar hur lång tid som har lagts på meddelandet.
    * **Åtkomstnivå**: Åtkomstnivån för meddelandet. Du kan välja mellan **Externt** och **Internt**.
    * **Bilaga**: Om meddelandet innehåller bilagor visas de i den här tabellen. Du kan ta bort befintliga bilagor men inte lägga till nya.

1. Klicka på **Spara** när du är klar, eller **Avbryt** om du vill ignorera ändringarna. Du kan också ta bort hela meddelandet genom att klicka på **Ta bort** och sedan **OK**.
