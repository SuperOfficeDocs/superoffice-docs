# Ta bort ärenden

> [!NOTE]
> För att du ska få tillgång till den här funktionen måste din användare tillhöra en roll som har den funktionella behörigheten **Service-administration** eller **Allmän administratör**.

Ärenden bör i allmänhet inte tas bort. Dock kan du behöva ta bort ett ärende för att följa ditt företags [integritetspolicy][4]. Alla meddelanden och bilagor/filer som är länkade till ärendet tas också bort.

1. [Öppna ärendet du vill ta bort][1].
1. Klicka på ![ikon][img1] knappen **Uppgift** och välj **Ta bort**.
1. Bekräfta om du verkligen vill ta bort ärendet genom att klicka på **Ja** eller **Nej**.

> [!NOTE]
> Ärendet tas inte bort från databasen omedelbart. Status för ärendet ändras till Borttaget, så det är fortfarande möjligt att öppna och redigera ärendet. Ärendet tas bort permanent från databasen efter 14 dagar (systeminställning). Du kan komma åt det borttagna ärendet i [papperskorgen][3].
