# Slå ihop ärenden

Du kan slå ihop två eller fler ärenden. Om till exempel flera kontakter i samma företag har rapporterat samma tekniska problem (via telefon, webbformulär och e-post) kan du slå ihop dessa ärenden till ett ärende som ska behandlas. Kontakter från alla hopslagna ärenden behålls.

> [!NOTE]
> Du kan också söka efter och välja ärenden att slå ihop på sökresultatskärmen eller listor. Klicka sedan på knappen ![ikon][img1] **Uppgift** i det nedre vänstra hörnet och välj **Slå ihop ärenden**.

## Steg

1. [Öppna ett av de ärenden][1] som du vill slå ihop.

1. Klicka på ![ikon][img1] knappen **Uppgift** och välj **Slå ihop ärenden**. Ett fönster öppnas med de fält som måste slås samman. Nu måste du hitta ärendet som ska slås ihop med detta ärende.

1. Börja skriva ärendenamnet i sökfältet eller ange ett ärende-ID. Skriv **%** för att visa alla ärenden.

1. Välj ett ärende i listan med sökresultat.

1. Om du vill lägga till fler ärenden upprepar du stegen.

1. Klicka på **Slå ihop**. Ärendena slås ihop.

De ärende-ID som skrevs över är inte längre tillgängliga i favoriter, historiklistor eller vanliga sökningar. Om du anger ett överskrivet ärende-ID i sökfältet, omdirigeras du till det hopslagna ärendet.
