# Slå ihop ärenden

Du kan slå ihop två eller fler ärenden. Om till exempel flera kontakter i samma företag har rapporterat samma tekniska problem via telefon, webbformulär eller e-post, kan du slå ihop dessa ärenden till ett enda ärende att hantera. Kontakter och meddelanden från alla hopslagna ärenden behålls.

> [!NOTE]
> Du kan även söka efter och välja ärenden att slå ihop från sökresultatsskärmen eller listor. Klicka sedan på knappen <i class="ph ph-list" aria-hidden="true"></i> **Uppgift** i det nedre vänstra hörnet och välj **Slå ihop ärenden**.

## Steg

1. [Öppna ett av de ärenden][1] du vill slå ihop.

1. Klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Uppgift-meny"></i> och välj **Slå ihop**. Ett fönster öppnas där du ser de fält som kommer att slås samman. Nu måste du hitta det ärende som ska slås ihop med detta.

1. Börja skriva ett ärendenamn eller ange ett ärende-ID i sökfältet. Ange **%** för att visa alla ärenden.

1. Välj ett ärende från listan med sökresultat.

1. Om du vill lägga till fler ärenden, upprepa föregående steg.

1. Klicka på **Slå ihop**. Ärendena slås nu samman.

De ärende-ID som har skrivits över är inte längre tillgängliga i favoriter, historiklistor eller vanliga sökningar. Om du anger ett överskrivet ärende-ID i sökfältet, omdirigeras du automatiskt till det hopslagna ärendet.
