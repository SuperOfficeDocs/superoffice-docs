# Ta bort ärenden

> [!NOTE]
> För att få tillgång till den här funktionen måste din användare tillhöra en roll med den funktionella behörigheten **Service-administration** eller **Allmän administratör**.

I allmänhet rekommenderas det inte att ta bort ett ärende. Dock kan du behöva ta bort ett ärende för att följa ditt företags [integritetspolicy][4]. Alla meddelanden och bilagor/filer som är länkade till ärendet tas också bort.

## Steg

1. [Öppna ärendet du vill ta bort][1].
1. Klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Uppgiftsmeny"></i> och välj **Ta bort**.
1. Bekräfta att du verkligen vill ta bort ärendet genom att klicka på **Ja** eller **Nej**.

> [!NOTE]
> Ärendet tas inte bort från databasen omedelbart. Statusen för ärendet ändras till **Borttaget**, vilket innebär att det fortfarande är möjligt att öppna och redigera ärendet. Ärendet raderas permanent från databasen efter 14 dagar (systeminställning). Du kan komma åt borttagna ärenden i [papperskorgen][3].
