# Inställningar för systemmallar

På fliken **Systemmallar** kan du ange vilka svarsmallar systemet ska använda.

> [!TIP]
> Använd sökfältet längst upp för att hitta specifika mallar.

## Ärenden

I avsnittet **Ärenden** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för avisering om aktiva ärenden | Mallen som används för att meddela användare om aktiva ärenden. Ett e-postmeddelande skickas varje kväll till användare med olösta ärenden äldre än ett visst antal dagar. Antalet dagar anges i Inställningar och underhåll (System). |
| Svarsmall för avisering om nytt meddelande | Mallen som används för att meddela användare när ett nytt meddelande har kommit in i ett ärende. |
| Svarsmall för avisering om aktiverat ärende | Mallen som används för att meddela användare om att ett uppskjutet ärende har aktiverats. |
| Svarsmall för avisering om eskalerat ärende | Mallen som används för att meddela användare om att ett ärende har eskalerats. |
| Svarsmall för avisering om nytt ärende | Mallen som används för att meddela användare om att de har blivit tilldelade ett ärende. |
| Svarsmall för avisering om övertagande av ärende | Mallen som används för att meddela användare om att en annan användare har tagit över ett ärende. |
| Svarsmall för att skriva ut ett ärende | Mallen som används vid utskrift av ett ärende. |

## Favoriter

I avsnittet **Favoriter** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för avisering om nytt meddelande i favoritlistan | Mallen som används för att meddela användare när ett nytt meddelande har kommit in i ett ärende på favoritlistan. |
| Svarsmall för avisering om aktiverat ärende i favoritlistan | Mallen som används för att meddela användare om att ett uppskjutet ärende på favoritlistan har aktiverats. |
| Svarsmall för avisering om eskalerat ärende i favoritlistan | Mallen som används för att meddela användare om att ett ärende på favoritlistan har eskalerats. |
| Svarsmall för avisering om övertagande av ärende i favoritlistan | Mallen som används för att meddela användare om att en annan användare har tagit över ett ärende på favoritlistan. |

## Kunder

I avsnittet **Kunder** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för nya kontakter | Mallen som skickas till nya kontakter när de registreras för första gången. |
| Svarsmall för autosvar | Mallen som används för att skicka automatiska standardsvar till kontakter vid mottagning av e-post. Detta värde kan åsidosättas per [e-postlåda][1]. |
| Svarsmall för utskick av lösenord till kontakt | Mallen som används för att skicka lösenord till kontakter som begär det. |
| Svarsmall för utskick av bekräftande e-post till kontakt | Mallen som används för att skicka bekräftelsemejl till kontakter. |
| Svarsmall för kundchattlogg | Mallen som används för att skicka chattloggar till kunder som kontaktat support via chatten. |
| Svarsmall för ny länk | Mallen som används för att skicka en ny länk till kunden när den gamla länken har gått ut. |

## FAQ

På fliken **FAQ** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för formatering av FAQ-frågor | Mallen som används vid publicering av FAQ-frågor. |
| Svarsmall för formatering av FAQ-svar | Mallen som används vid publicering av FAQ-svar. |

Dessa inställningar låter dig definiera din egen formatering för förslag på frågor och svar i FAQ.

## Veckovis statistik

På fliken **Veckovis statistik** finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
| Svarsmall för veckovis statistik | Mallen som används för att skicka veckostatistik. |
| E-postadresser för veckovis statistik | Här kan du ange en eller flera (kommaseparerade) e-postadresser som veckostatistik för SuperOffice ska skickas till. |

## Så här byter du mall

För att välja en annan mall, klicka på <i class="ph ph-caret-down" aria-label="Chevron"></i> intill det aktuella fältet och välj en mall från listan som visas.
