# Ta bort en figur

Du kan ta bort figurer från en dashboard så länge du har redigeringsåtkomst till den.

> [!NOTE]
> Borttagna figurer kan fortfarande [läggas till på dashboards][1].

**Steg:**

1. Gå till önskat dashboard.

2. Klicka på **Uppgift**-knappen välj **Lägg till figur/Redigera layout**.

3. På dashboarden klickar du på ikonen **Ta bort figur** ![ikon][img1] på den figur du vill ta bort.

4. Bekräfta genom att klicka på **OK**.
