# Dashboard för utskick

Skapa dashboards för dina marknadsföringsuppgifter, till exempel formulärinlämningar och utskick, för att spåra och analysera resultat, trender och produktivitet i alla marknadsföringsaktiviteter.

När dashboarden har skapats är det dags att lägga till och redigera figurer.

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/lz9rEZI_2SY]

I exemplen nedan hittar du förslag på figurer som du kan lägga till i dashboarden för utskick eller formulär.

## Exempel på olika figurer för utskick

* Figuren **Toppresterande utskick i år** är ett bra sätt att se vilka utskick som ger bäst resultat. Granska de här utskicken för att ta reda på vilken typ av utskick, design och innehåll som interagerar bäst med din kundbas.

* Använd figuren **Genomsnittlig öppningsfrekvens YTD** för att se om mottagarna klickar på länkar eller bilder i dina utskick. Om öppningsfrekvensen är låg kan det vara en bra idé att testa om andra ämnesrader ökar frekvensen.

![Exemplet visar olika utskicksfigurer som du kan använda för att få en översikt över dina data -screenshot][img1]

## Exempel på olika figurer för formulärinlämningar

* Använd figuren **Aktiva formulär detta kvartal** för att få en översikt över de formulär har skickats innevarande kvartal.

* Figuren **Mina kunders formulärinlämningar den här månaden** kan hjälpa dig att hålla reda på kundernas inlämningar varje månad.

* I figuren **Obehandlade inlämningar** ser du vilka inlämningar som inte har behandlats och behöver din uppmärksamhet.

![Exemplet visar olika formulärinlämningsfigurer som du kan använda för att få en översikt över dina data -screenshot][img2]
