# Dashboard för leads

Dashboards hjälper dig att övervaka leadvolym, köpmognad, källor och konverteringstrender. Genom att kombinera paneler som visar var leads kommer från, hur de utvecklas och hur de konverterar kan du utvärdera effekten av dina marknads- och säljaktiviteter.

Använd denna sida som inspiration när du bygger dashboards för din organisation.

## Exempel på paneler för leadvolym och utveckling

Dessa paneler hjälper dig att förstå hur många leads du får, hur de fördelas över statusar och hur de rör sig genom kvalificeringsprocessen.

* **Leadkvalificeringstratt:** Visar hur leads rör sig från *Öppen* till *Kvalificerad*. Använd denna panel för att identifiera var leads fastnar.

* **Nya leads per kampanj YTD:** Visar hur många leads varje kampanj har genererat i år.

* **Nya leads per leadskapandekälla YTD:** Visar vilka kanaler som genererar flest nya leads, såsom webbformulär, sociala medier eller nyhetsbrev.

* **Nya leads per första kontaktkälla YTD:** Hjälper dig att förstå den första interaktionen som utlöste ett lead.

* **Nya leads från chattar YTD:** Mäter hur många leads som kommer från chattkonversationer.

* **Nya leads från formulär YTD:** Visar antalet formulärinlämningar som resulterade i nya leads.

## Exempel på paneler för leadkvalitet och konvertering

Dessa paneler belyser vilka källor som genererar kvalificerade leads och vilka som resulterar i försäljning.

* **Antal affärer per leadskapandekälla:** Visar vilka källor som bidrar mest till faktiska försäljningar.

* **Pipeline per leadskapandekälla:** Visar värdet av aktiva möjligheter kopplade till leads från olika källor.

* **Topp hänvisande webbplatser för nya leads YTD:** Listar de webbplatser som skickar mest trafik som blir till nya leads.

* **Topp 10 formulär för leadinfångning YTD:** Visar vilka formulär som skapar flest leads under året.

![Dashboard som visar leadkonverteringspaneler -screenshot][img2]

## Exempel på paneler för marknadsöverlämning

Om du använder UTM-spårning (kräver Marketing Premium) hjälper ytterligare paneler dig att utvärdera kanal- och kampanjprestanda för leadanskaffning.

* **Nya leads per leadskapandekälla YTD:** Visar vilka kanaler som skapar nya leads, såsom Google, LinkedIn, Facebook, nyhetsbrev eller andra spårade källor.

* **Nya leads per första kontaktkälla YTD:** Belyser den allra första interaktionen som förvandlade en anonym besökare till ett lead. Användbart för attribuering och utvärdering av kanaler i tidigt skede.

* **Nya leads per kampanj YTD:** Visar vilka kampanjer som genererade flest nya leads i år, vilket hjälper dig att jämföra kampanjprestanda och identifiera aktiviteter med hög konvertering.

![Tre dashboardpaneler med donutdiagram för leadanskaffning -screenshot][img3]

## Tips för att bygga effektiva leaddashboards

* Kombinera **statuspaneler** och **källpaneler** för att förstå både köpmognad och ursprung.
* Lägg till **filter** (till exempel efter medarbetare eller land) för att fokusera på de leads som är viktigast.
* Använd **dynamiska urval som paneler** när du behöver listor över leads istället för diagram.
* Skapa olika dashboards för **marknadsföring**, **leadnurturing** och **försäljning**, beroende på deras behov.
