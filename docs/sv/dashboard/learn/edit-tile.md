# Redigera en figur

1. Gå till önskat dashboard.

2. Klicka på **Uppgift**-knappen välj **Lägg till figur/Redigera layout**.

3. Klicka på ikonen **Redigera figur** ![ikon][img2] på dashboarden.

4. På fliken **Dataset** kan du [redigera kriterierna][1] för de data som visas i figuren. Kom ihåg att uppdatera figurens namn för att spegla eventuella ändringar som har gjorts här.

5. Under fliken **Layout** kan du redigera olika inställningar, värden och formatering av figuren. Förhandsgranskningen till höger kommer att uppdateras med de valda inställningarna.

6. Klicka på ![ikon][img1] för att [lägga till en översättning][2] till figurnamnet, rubrikerna och ledtexterna.

7. Klicka på **Spara**.
