# Anpassa din dashboard

Du kan anpassa varje dashboard för att visa den information som är mest relevant för dig. Välj bland fördefinierade paneler eller skapa egna för att visualisera aktiviteter, kundsegment, produkttrender, projektstatus eller försäljningsresultat.

## Redigera dashboardinställningar

Du kan uppdatera namn, beskrivning, layout eller synlighet för en befintlig dashboard.

1. Öppna skärmen **Dashboard**.

1. Välj den dashboard du vill redigera.

1. Klicka på knappen **Uppgift** (<i class="ph ph-dots-three-circle-vertical" aria-hidden="true"></i>) och välj **Redigera dashboard-information**.

1. Gör dina ändringar:
    * Ändra **namn** eller **beskrivning**
    * Välj ett annat **tema**
    * Justera antal **kolumner**
    * Om du har rätt behörighet kan du uppdatera fälten **Synlig för** och **Fäst på**

    > [!NOTE]
    > Den person som skapar dashboarden anges som **ägare**. Ägarskap kan inte ändras senare.

1. Klicka på **Spara**.

> [!NOTE]
> Om du ändrar antalet kolumner justeras panelerna automatiskt för att passa det nya upplägget. Inga paneler tas bort, men layouten kan få tomma utrymmen. Du kan sedan flytta eller ändra storlek på panelerna.

## Redigera paneler och layout

Så här lägger du till, flyttar, redigerar eller tar bort paneler:

1. Öppna dashboarden.
1. Klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Task button"></i> och välj **Redigera dashboard-paneler**.
1. [Följ dessa steg för att hantera paneler][2].

## <a id="delete"></a>Ta bort en dashboard

Endast **ägaren** till en dashboard kan ta bort den. Inte ens administratörer kan ta bort dashboards de inte äger. Ägarskap kan inte överföras.

1. Öppna skärmen **Dashboard**.
1. Välj dashboarden du vill ta bort.
1. Klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Task button"></i> och välj **Ta bort den här dashboarden**.
1. Klicka på **OK** för att bekräfta.

> [!NOTE]
> Om dashboarden var fastnålad för andra tas den bort från deras vy också.
