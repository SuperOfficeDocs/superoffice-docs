# Offert (detaljkort på Försäljning)

På detaljkortet **Offert** längst ner i fönstret Försäljning, kan du visa befintliga offerter eller skapa nya. De tillgängliga alternativen beror på om försäljningen har en offert.

## Försäljning utan befintlig offert

**Skapa en ny offert**: Används för att [skapa en offert][1] för denna försäljning.

**Kopiera offert från en annan försäljning**: Används för att [använda en befintlig offert][2] som grund för en ny offert.

## Försäljning med befintlig offert

**Klicka för att skapa ett offertdokument**: Se [Skapa offertdokument][3].

Alternativ: Om offerten innehåller ett eller flera [alternativ][4], kan du välja ett i listan högst upp på detaljkortet **Offert**. Ikonerna ![ikon][img1]/![ikon][img2] längst ner i listan anger om det valda alternativet är [favoritalternativet][5].

Till höger om detaljkortet **Offert** finns version, offertnummer och [status][6] (till exempel **Utkast** eller **Sålt**). Klicka på **Öppna** för att redigera offerten.

[!include[Table showing quote states](includes/table-quote-states.md)]

Produktlistan: Om produkter har lagts till i offerten, visas de i listan. Längst ner i listan visas det totala priset för produkterna i den.

**Exportera**: Används för att exportera listan till ett kalkylark.
