# Dialogrutan Redigera offert för

## Den övre delen av dialogrutan

* [Klicka för att skapa ett offertdokument][1].
* [Meddelande/varning (med ikoner)][9].
* **Version**: Den aktuella versionen av offerten visas här. Du kan välja tidigare versioner från listan.
* **Nummer**: Offertnumret.
* **Status**/statusikon: Visar [offertens status][8] (till exempel **Utkast**, **Måste godkännas**, **Skickad** eller **Såld**)
* **Skicka**: Klicka på denna knapp för att [skicka offerten][3].
* **Registrera order:** Klicka på denna knapp för att [registrera ordern][4].

## Fliken Produkter (och eventuella alternativ)

Produkterna i offerten visas på den här fliken.

Om offerten innehåller flera alternativ, anges favoritalternativet med <i class="ph ph-star" aria-label="Star icon"></i> bredvid fliknamnet. Om alternativet innehåller [varningar eller meddelanden][9] för en eller flera produkter, visas en ikon bredvid fliknamnet. Ett meddelande visas också för att förklara orsaken till felet.

* **Rabatt**: Här kan du ange en total rabatt (rabattprocent eller rabattbelopp) för hela alternativet. Den totala rabatten tillämpas utöver eventuella rabatter som har tillämpats på de olika produkterna.

    > [!NOTE]
    > Total rabatt måste vara aktiverat som ett alternativ i Inställningar och underhåll.

* **Bruttovinst/Totalt**: Detta visar bruttovinsten och summan för hela alternativet.
* **Lägg till**: Används för att söka efter och [lägga till produkter][6].
* Dubbelklicka på en produkt för att redigera den.
* **Ta bort**: Tar bort den eller de markerade produkterna.
* **Exportera**: Används för att [exportera listan][7] till ett kalkylark.
* Sortera: Använd <i class="ph ph-arrow-circle-up" aria-label="Arrow up"></i> <i class="ph ph-arrow-circle-down" aria-label="Arrow down"></i> för att ändra ordning på produkterna.

> [!TIP]
> Du kan dubbelklicka på flikarna för att ändra namnen på alternativen i offerten.

## Fliken Detaljer

Här väljer du önskade alternativ för betalning och leverans. Du kan också välja fakturaadress och leveransadress här.

## Fliken bilagor

Här kan du välja en eller flera bilagor att bifoga till offerten. De bilagor som visas här är fördefinierade i Inställningar och underhåll.

Den nedre delen av dialogrutan

**Uppdatera priser**: Används för att uppdatera priserna i offerten utifrån prislistan eller prislistorna.
