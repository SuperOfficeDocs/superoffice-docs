# Skapa offertdokument

När du har skapat offerten och lagt till de produkter du vill ha, kan du skapa offertdokumentet som ska skickas till kunden. Detta är "brevet" som skickas till kunden. Listan över produkter som erbjuds läggs till när offerten skickas till kunden.

1. Öppna en befintlig offert eller [skapa en ny offert][1].

2. Klicka på **Klicka för att skapa ett offertdokument** högst upp på offerten eller på detaljkortet **Offert**.

3. I dialogrutan **Dokument** [anger du den information som behövs i fälten][2].

4. Skapa det nya dokumentet med den valda dokumentmallen som grund genom att klicka på **Skapa**.

5. Ange dokumentinnehållet och spara och stäng det på vanligt sätt i programmet. Dokumentet visas sedan på detaljkortet **Aktiviteter** i fönstret Försäljning (och fönsterna Företag och Kalender).

> [!TIP]
> Det finns en [lista över de variabler][3] som du kan infoga i dokument som du skapar.

## Hur ändrar jag offertdokumentet?

Om du har skapat ett offertdokument för en offert, men vill använda ett annat dokument:

1. Öppna offerten.

2. Klicka på knappen ![ikon][img1] **Uppgift** och välj **Byt offertdokument**.

3. Välj offertdokumentet i listan.

    > [!NOTE]
    > Listan visar alla offertdokument som är registrerade för den aktiva försäljningen.
