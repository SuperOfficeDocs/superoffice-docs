# Skicka offert

När du har lagt till produkter i offerten och skapat ett offertdokument, kan du skicka offerten till kunden.

1. Öppna offerten i fråga.
2. Klicka på **Skicka**. Dialogrutan **Publicera/skicka en offert** visas.

    > [!NOTE]
    > Visas det ett felmeddelande? Du kan inte skicka en offert om den [behöver godkännas][1] eller [innehåller fel eller varningar][2].

3. Om det behövs, redigera offertens giltighet i fältet **Giltig till**. Antalet dagar tills offerten upphör att gälla visas till höger om fältet.

4. Välj önskad mall för offerten i fältet **Offertformat**.

5. Klicka på fältet **Språk** för att välja vilket språk offerten ska skickas på.

6. Om offerten innehåller alternativ kan du välja **Använd endast favoritalternativ** om du bara vill inkludera det här alternativet. Annars inkluderas alla alternativ i offerten.

7. Välj ett av följande alternativ i fältet **Publicera**:
    * **Skicka som e-post och arkivera som PDF**. Du måste välja en e-postmall i listan.
    * **Arkivera endast som PDF**. Du måste själv skicka offerten till kunden.

8. Markera **Uppföljning** om du till exempel vill skapa en påminnelse till dig själv om att följa upp offerten. Välj datum för uppföljningen och ange en uppföljningstext.

9. Klicka på **Förhandsgranskning** om du vill visa en PDF-version av offerten innan du skickar den.

10. Klicka på **OK**.

    Om du valde att skicka offerten som e-post visas dialogrutan **Nytt e-postmeddelande**. Här kan du redigera texten och skicka offerten till kunden. Om arkiveringsalternativet är aktiverat, öppnas dialogrutan **Dokument**.

Offerten markeras som **Skickad** och arkiveras som PDF-dokument i detaljkortet **Aktiviteter**.
