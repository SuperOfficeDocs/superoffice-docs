# Konfigurera produkter

Du kan anpassa och konfigurera offerthanteringsfunktionerna i Inställningar och underhåll så att de matchar de namn och etiketter som användarna känner till.

Fönstret Konfigurera produkt finns på fliken Inställningar. Här kan du ändra och anpassa alla etiketter som används i offerthanteringen. Dessutom kan du kontrollera om ett fält är redigerbart och/eller obligatoriskt.

![Du kan ändra alla etiketter för fälten i produktmenyn i fönstret Konfigurera produkt -screenshot][img3]

## Steg

Du öppnar dialogrutan **Konfigurera produkt** genom att gå till fönstret **Of/Synk** &gt; fliken **Inställningar** &gt; knappen **Konfigurera produkt**.

Här kan du anpassa innehållet i fönstret Produkt, där användare lägger till och redigerar produkter i offerten.

## Välj språk

I listan högst upp kan du välja det språk du vill redigera produktfältet för.

## Ange fältegenskaper

Översikten visar alla produktfält med tillhörande egenskaper. Längst ned finns information om alternativen i de olika kolumnerna.

### Visa fält

* Markera varje fält som du vill visa i dialogrutan Produkt.
* Avmarkera de fält som du inte vill visa.
* Gråa/icke-redigerbara checkrutor är fält som inte kan tas bort.

[!include[Why set all fields when creating product](../includes/quote-config-product.md)]

### Etikett

Standardetiketten för fältet (fältnamn) visas här. Du kan redigera den här texten. Om du tar bort all text i det här fältet, återställs standardetiketten.

### Hjälptext

Standardhjälptexten för fältet (fältnamn) visas här. Den här texten visas när du håller muspekaren över fältnamnet i dialogrutan **Produkt**. Du kan redigera den här texten. Om hjälptexten är lång, kan du använda pilknapparna till höger om fältet för att gå uppåt och nedåt i fältet.

### Redigerbar/Obligatorisk

Markera här om användarna ska kunna redigera fältet och/eller vara tvungna att fylla i fältet. Fältet ställs automatiskt in på **Redigerbart** om du väljer **Obligatoriskt**.

> [!NOTE]
> I vissa fall är dessa alternativ låsta av funktionella skäl.

### Ordning

Klicka på <i class="ph ph-arrow-circle-up" aria-label="Arrow up"></i> <i class="ph ph-arrow-circle-down" aria-label="Arrow down"></i> för att välja fältets position. Denna order används också i dialogrutan **Lägg till/redigera produkt**.
