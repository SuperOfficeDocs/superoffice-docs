# Ta bort/inaktivera prislistor eller produkter

På fliken **SuperOffice-produkter** i fönstret **Offert/Synk** kan du ta bort eller inaktivera både prislistor och produkter i prislistorna. Dessa prislistor och produkter kommer då inte längre att vara tillgängliga när användare skapar offerter.

> [!NOTE]
> Vi rekommenderar att du inaktiverar prislistor och produkter istället för att tar bort dem permanent. Om du tar bort dem, kan det uppstå problem när man uppdaterar priser och skickar offerter.

## Inaktivera prislistor

1. Välj den prislista du vill inaktivera.
2. Klicka på **Redigera**.
3. Gör något av följande:
    * Avmarkera checkrutan **Aktiv**.
    * Markera **Begränsad giltighet** och ange inaktiveringsdatumet i fältet **Giltig till**.
4. Klicka på **OK**.

> [!NOTE]
> Inaktiverade prislistor är inte tillgängliga när användarna skapar offerter.

## Inaktivera produkter

1. Välj den prislista som innehåller produkten du vill inaktivera. Produkterna i prislistan visas i listan längst ned.

2. Avmarkera kryssrutan för varje produkt som du vill inaktivera.

Inaktiverade produkter är inte tillgängliga när användarna skapar offerter.
