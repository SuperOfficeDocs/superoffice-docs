# SuperOffice-produkter

På fliken **SuperOffice-produkter** i fönstret Offert/Synk kan du lägga till prislistor och produkter. De sparas i SuperOffice-databasen, INTE i något ERP-system som är anslutet till SuperOffice.

I listan **Prislista**, ska du välja önskad prislista. Produkterna i prislistan visas längst ned i vyn. Du kan [justera den här visningen][6].

## Alternativ

* Dubbelklicka på en produkt för att redigera den. Se [Lägga till produkt i prislista][1].

* Klicka på checkrutan i kolumnen längst till vänster för att [aktivera/inaktivera en produkt][3].

* Klicka på **Lägg till** för att [lägga till en produkt i prislistan][1].

* Klicka på **Ta bort** för att ta bort en produkt. Vi rekommenderar dock att du [inaktiverar produkter istället för att ta bort dem][3].

* Klicka på **Exportera** om du vill exportera listan till ett kalkylblad. Se nedan.

## Exportera en prislista

Du kan exportera en öppen prislista genom att klicka på <i class="ph ph-download-simple" aria-hidden="true"></i> **Exportera** längst ned på fliken **SuperOffice-produkter**. Om du bara vill exportera vissa produkter, kan du välja dem i prislistan. Om du inte väljer några produkter, exporteras alla produkter i prislistan.

### Syns inte exporten?

Antingen har du inte [åtkomstbehörighet][4] till exportfunktionen eller så är den här [inställningen][5] inte aktiverad.

> [!TIP]
> Innan du exporterar prislistan kan det vara bra att välja vilka kolumner du vill ha och deras ordning.
