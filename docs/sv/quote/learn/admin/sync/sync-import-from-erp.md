# Importera från ERP

Du kan importera företag, kontakter och produkter från en ERP-anslutning.

1. Gå till fliken **Synk** i fönstret Offert/Synk.
2. Välj **Av** för att inaktivera automatisk synkronisering.
3. Markera ERP-kopplingen och klicka på knappen **Importera från ERP**.
4. Fönstret **Import** öppnas. [!include[Select source](../../../../admin/import/learn/includes/select-source.md)]
5. Se [importera från ERP][1] för mer information.
