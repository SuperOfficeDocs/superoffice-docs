# Omsynkronisera från CRM

Den här funktionen används för att åtgärda problem, särskilt om du vet att data i SuperOffice CRM har högre kvalitet än ERP-data. Det fungerar som en normal synkronisering, men tidsstämplar ignoreras och synkroniseringen sker endast i en riktning (CRM till ERP).

[!include[Back up database before import](../../../../admin/import/learn/includes/caution-backup-before-import.md)]

1. Gå till fliken **Synk** i fönstret Offert/Synk.
2. Klicka på **Synka om från CRM** nederst i listan **ERP-kopplingar**.
3. Bekräfta genom att klicka på **Ja**.
