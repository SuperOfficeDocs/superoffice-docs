# Redigera ERP-koppling (Sync Connector)

1. Gå till fliken **Synk** i fönstret Offert/Synk.
2. Avmarkera rutan i kolumnen **Aktiv** för ERP-kopplingen du vill redigera.
3. Dubbelklicka på ERP-kopplingen.
4. I dialogrutan **Redigera ERP-anslutning** [redigerar du ERP-anslutningen][1].
