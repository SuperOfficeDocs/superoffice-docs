# Sync

På fliken **Synk** i fönstret Offert/Synk visas Sync Connectors och tillhörande ERP-kopplingar. Sync Connectors hanterar synkroniseringen av data mellan SuperOffice och ERP-systemet, medan ERP-kopplingarna utgör förbindelsen mellan SuperOffice och Sync Connectors.
