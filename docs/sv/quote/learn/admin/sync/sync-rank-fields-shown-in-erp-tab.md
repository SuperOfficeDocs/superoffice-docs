# Ordna fält som visas på ERP-fliken

Du kan ställa in ordning för ERP-fälten som visas på ERP-fliken i SuperOffice CRM. (Du bestämmer vilka fält som ska visas på fliken ERP i dialogrutan [Tilldelningar][1].)

1. [!include[Go to sync](../includes/goto-sync.md)]

1. Markera ERP-kopplingen och klicka på knappen **Ordna fält** nederst. Dialogrutan **Ordna fält som visas på ERP-fliken** visar flikarna för de tillgängliga aktörerna i ERP-systemet.

1. Välj fliken för den aktör som du vill konfigurera standardvärden för. Den här fliken visar alla tillgängliga fält för aktören (inklusive både fält som visas på fliken **ERP** i SuperOffice CRM och de som inte gör det).

1. Välj ett fält i listan och använd ![ikon][img1] ![ikon][img2] pilarna för att ange fältets ordning.

1. Upprepa steg 4 och 5 för att ändra fältrangordningen för andra aktörer.

1. Klicka på **Klar**.

> [!TIP]
> Dubbelklicka på ett fält om du vill redigera standardvärdet för fältet. Se [Konfigurera standardvärden för synkronisering][2].
