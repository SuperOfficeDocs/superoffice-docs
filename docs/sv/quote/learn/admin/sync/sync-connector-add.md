# Lägga till Sync Connector

1. Gå till fliken **Synk** i fönstret Offert/Synk.
2. Klicka på **Lägg till** nederst i listan **Sync Connectors**.
3. I dialogrutan anger du namn på och URL till Sync Connector.
4. Klicka på **OK**.

    Sync Connector visas i listan.
