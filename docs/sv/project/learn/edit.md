# Redigera projekt

Du kan redigera information om ett projekt direkt från flikarna på projektkortet.

## Steg

1. Öppna **Projekt**-fönstret för det projekt du vill uppdatera. Se [Använda fönstret Sök][5].

1. Klicka på <i class="ph ph-pencil-simple" aria-label="Edit"></i> i det övre högra hörnet av projektkortet.

1. Gör de önskade ändringarna. Du kan byta mellan flikarna medan du är i redigeringsläge.

    > [!CAUTION]
    > Om du byter projekttyp som är kopplad till en [projektguide][4] tas guiden bort. Aktiviteter och dokument finns kvar på fliken **Aktiviteter** och återansluts om du byter tillbaka till den ursprungliga typen.

1. Klicka på **Spara** för att spara ändringarna och avsluta redigeringsläget, eller klicka på **Avbryt** för att avvisa ändringarna.

    > [!NOTE]
    > Om **Spara**-knappen är nedtonad, kontrollera att alla obligatoriska fält är ifyllda. Obligatoriska fält måste fyllas i innan du kan spara.
