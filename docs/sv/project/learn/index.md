# Projekt

Du kan länka vilken aktivitet som helst till ett externt företag och till ett projekt. Du kan även arbeta med en projektguide som leder dig genom varje steg i projektet, om projekttypen du har valt har en guide. Ett projekt kan vara riktat mot många kontakter, och en kontakt kan delta i flera projekt.

![Projekt-fönstret -screenshot][img1]

Projekt hjälper dig att organisera och dokumentera arbetet på ett ställe, så att du och ditt team kan få bättre kontroll och förbättra samarbetet.

Du kan ansluta kontakter, företag, händelser, projektdeltagare och till och med affärsmöjligheter till ett projekt i SuperOffice CRM. Du kan också skapa eller överföra dokument och filer till projektet.

All information du sparar i projektet taggas med ett datum och en ägare, vilket gör det enklare att sortera och filtrera information när det behövs.

**Projekt**-fönstret innehåller information om de projekt som lagts in i SuperOffice CRM. Varje projekt visas som ett indexkort som visar all information om projektet.

För att navigera mellan projekt klickar du på knapparna **Föregående** och **Nästa** (<i class="ph ph-caret-circle-left" aria-hidden="true"></i><i class="ph ph-caret-circle-right" aria-hidden="true"></i>) längst ned till höger på korten.

Om du har valt en projekttyp för vilken en [projektguide][1] har definierats, visas fliken **Projektguide** på detaljkortet.

## Flikar på projektkortet

| Flik | Beskrivning |
|---|---|
| [Projekt][2] | Innehåller information om det aktuella projektet. |
| [Mer][19] | Användardefinierade fält för att lägga till ytterligare projektspecifik information. |
| [Anteckning][14] | Fri textyta för att lägga till anteckningar om projektet. En punkt markerar fliken när den innehåller information. |

| [Länkar][10] | Koppla relevanta aktiviteter, försäljningar, dokument eller webbplatser till projektet. |
