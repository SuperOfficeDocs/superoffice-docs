# Ta bort ett projekt och tillhörande data

1. Öppna fönstret **Projekt** för det projekt du vill ta bort.

1. Klicka på knappen **Uppgift** (<i class="ph ph-dots-three-circle-vertical" aria-hidden="true"></i>) i det övre högra hörnet av projektkortet.
    Välj **Ta bort** i menyn.

    > [!NOTE]
    > Om alternativet **Ta bort** är nedtonat betyder det att du inte har tillräckliga användarrättigheter för att ta bort projektet. Du kan inte ta bort projekt som du inte äger, om du inte har nödvändiga behörigheter.

1. I dialogrutan **Ta bort projekt**, granska projektuppgifterna för att säkerställa att du tar bort rätt projekt.

1. Klicka på **Ja** för att bekräfta borttagningen.
