# Ta bort ett helt projekt med tillhörande data

1. [!include[Go to project](includes/goto-project.md)]
2. Klicka på knappen ![ikon][img1] **Uppgift**.
3. Välj **Ta bort "projektnamn"**.
4. Klicka på **Ja** när du tillfrågas om du verkligen vill ta bort urvalet.
