# Projektstatus

Fliken **Projekt** i projektet innehåller fältet **Status**. För det här fältet finns ett antal fördefinierade alternativ och projektstatus som konfigureras i fönstret Listor i Inställningar och underhåll.

Vilka statusar som visas i listan beror på [projekttypen][1]. Du kan välja vilka statusar som ska vara tillgängliga för de olika projekttyperna i listan Projekt - Typ, status.

## Vad vill du göra nu?

* [Lägg till en projektstatus][2]
