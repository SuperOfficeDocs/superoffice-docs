# Projekt – Typ, status

Fliken **Projekt** i projektet innehåller fältet **Typ**. För det här fältet finns ett antal fördefinierade alternativ som konfigureras i fönstret Listor i Inställningar och underhåll.

Varje projekt i SuperOffice CRM måste ha en projekttyp och varje projekttyp kan innehålla flera [statusar][1]. Du kan också ange att specifika projekttyper ska ha en [projektguide][3] som guidar dig genom projektets olika statusar, med tillhörande händelser och dokument.

## Vad vill du göra nu?

* [Lägg till projekttyp][2]
