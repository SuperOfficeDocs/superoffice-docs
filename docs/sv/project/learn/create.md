# Skapa ett projekt

Titta på den här filmen och följ stegen nedan för att lära dig hur man skapar ett projekt (filmens längd – 2:49):

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/zLJwNll223w]

## Steg

1. Klicka på **Nytt** på toppraden och välj **Projekt**.

2. Fyll i önskade uppgifter i det tomma projektkortet i fälten på fliken **Projekt**. Läs mer nedan.

3. Du kan också gå till en av flikarna för att ange ytterligare information.

4. Klicka på **Spara**. Nu kan du [lägga till projektdeltagare][5].
