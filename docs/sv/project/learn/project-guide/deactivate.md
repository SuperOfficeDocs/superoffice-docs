# Inaktivera projektguide

[!include[Must be admin](../../../learn/includes/req-admin.md)]

Om du vill ta bort en projektguide från en projekttyp, kan du inaktivera projektguiden. Det gör du genom att avmarkera **Denna projekttyp har en anknuten guide** för den aktuella projekttypen (se [Lägga till poster i listan Projekt – Typ, status][1]).

Om du sedan vill återanvända projektguiden markerar du helt enkelt det här alternativet igen. Alla händelser och dokument som tidigare lagts till i projektguidens status hämtas automatiskt. Du kan också lägga till nya händelser och/eller dokument, eller ta bort de som du inte vill använda i den här projektguiden.
