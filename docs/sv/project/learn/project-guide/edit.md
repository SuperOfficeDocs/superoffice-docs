# Redigera en projektguide

[!include[Must be admin](../../../learn/includes/req-admin.md)]

1. Klicka på knappen ![ikon][img1] **Arbetsflöde** i navigatorn.
2. Välj fliken **Projektguide**.
3. Välj önskad projekttyp i listrutan högst upp.
4. [Redigera information om projekttypen och dess status][3].
5. [Redigera de enskilda statusarna][3].
6. Redigera händelser och dokument genom att dubbelklicka på dem i listorna längst ned.

    * [Lägg till en händelse för en status][2]
    * [Lägg till ett dokument för en status][1]

    Ändringarna sparas automatiskt.
