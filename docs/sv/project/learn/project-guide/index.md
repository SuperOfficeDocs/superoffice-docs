# Projektguide

Med hjälp av projektguiden kan ett företag göra projektprocessen enklare och mer strukturerad för sina projektledare. Projektguiden visar vägen genom de olika statusarna av en fördefinierad projektprocess och innehåller förslag på vilka händelser som bör utföras och vilka dokument som bör skapas i varje status. Dessutom kan den föreslå nästa steg i projektet och visas projektets milstolpar.

![Detaljkortet Projektguide med översikt över Planerad status -screenshot][img1]

Genom att skapa en projektguide kan du planera alla stadier i ditt projekt på förhand för att se till att ditt projekt blir lyckat. Som standard finns det flera tillgängliga projektguider som du kan använda som inspiration. Eller så kan du redigera dem så att de passar dina behov.

Varje **status** för projektguiden kommer med specifika aktiviteter som händelser och dokument. På så sätt vet du alltid vilken aktivitet eller milstolpe som kommer härnäst. Du kan bygga din egen process beroende på vilka olika typer av projekt ditt företag är verksamt inom. Det bästa sättet att definiera dina egna projektguider är att först diskutera och avtala bästa praxis för alla dina projekt.

Projektguiden definieras i Inställningar och underhåll.

> [!TIP]
> Du kan förhandsgranska dokument och aktiviteter i **Förhandsgranska** på sidopanelen.
