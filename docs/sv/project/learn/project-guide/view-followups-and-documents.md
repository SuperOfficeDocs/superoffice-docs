# Visa händelser och dokument

Som standard visas uppföljningar och dokument för aktuell projektstatus i den nedre halvan av detaljkortet, men du kan välja vilka händelser och dokument du vill visa med hjälp av checkrutorna längst ned på detaljkortet.

## Visa händelser och dokument med en status

Du kan visa aktiviteter och dokument för en av statusarna genom att klicka på knappen med statusens namn.

## Visa alla händelser och dokument

Markera **Visa alla statusar** för att visa alla föreslagna och skapade händelser och dokument för alla statusar.

## Visa endast skapade händelser och dokument

Markera **Endast skapade** om du bara vill visa händelser och dokument som har skapats och dölja föreslagna händelser och dokument.

## Visa endast milstolpar

Markera **Endast milstolpar** om du bara vill visa de händelser som definierats som milstolpar. En händelse kan definieras som en milstolpe i Inställningar och underhåll.

Du kan flytta projektet till nästa status med hjälp av listan **Status** på huvudkortet eller genom att högerklicka på statusknappen på detaljkortet och välja **Flytta till denna status**.
