# Detaljkortet Projektguide

När du skapar ett projekt och väljer en projekttyp som har en guide, visas detaljkortet **Projektguide** bredvid de andra detaljkorten.

Knapparna för de olika statusarna visas högst upp på detaljkortet. Knapparnas utseende ändras när du slutför statusarna i projektguiden:

| Knapp | Beskrivning |
|:-:|---|
| ![ikon][img1] | Slutförd-status |
| ![ikon][img2] | Aktuell status |
| ![ikon][img3] | Framtida status |

Varje status i projektguiden innehåller olika händelser och/eller dokument, vilka visas i den nedre halvan av detaljkortet. De listas i två kolumner, **Händelser** och **Dokument**. Om du klickar på knappen för en status visar den nedre halvan av detaljkortet vilka aktiviteter som tillhör statusen.

Ikonen ![ikon][img7] visas framför händelser som är definierade som milstolpar. Definitionen görs i Inställningar och underhåll. Huvudkortet visar datumet för nästa milstolpshändelse i fältet **Nästa milstolpe** .

Klicka på **Skapa** bredvid händelserna och dokumenten för att skapa dem. Du kan också klicka på händelsens eller dokumentets titel för att skapa det.

![Skapa alternativ i projektguiden -screenshot][img5]

> [!TIP]
> Alla händelser och dokument som du skapar för en projektguide visas också på detaljkortet **Aktiviteter** på projektskärmen. Detta ger dig en översikt över alla aktiviteter som är kopplade till projektet – både slutförda och ej slutförda aktiviteter.

När en händelse eller ett dokument har skapats, kan du ta bort eller redigera det genom att högerklicka på händelsens/dokumentets titel och välja från menyn som visas. På samma sätt kan du också skapa en ny händelse eller ett nytt dokument av samma typ.

![Alternativ vid högerklick i projektguiden -screenshot][img6]
