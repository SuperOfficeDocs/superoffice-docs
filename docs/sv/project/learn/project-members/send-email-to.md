# Skicka e-post till intressenter eller projektdeltagare

1. Gå till önskad försäljning på fönstret Försäljning eller till önskat projekt på projektskärmen.

2. Välj detaljkortet **Intressenter** eller **Projektdeltagare**.

3. Välj en eller flera intressenter eller projektdeltagare.

4. Högerklicka och välj **Skicka e-postmeddelande**. Detta öppnar ett meddelande till mottagaren eller mottagarna i standardprogrammet för e-post.

    > [!NOTE]
    > Endast kontakter med angivna e-postadresser visas i **Till**-fältet. Kontakter som saknar e-postadress utelämnas.

5. Ange eventuell extra information.

6. Skicka e-postmeddelandet genom att klicka på någon av följande knappar:
    * Skicka: E-postmeddelandet skickas och visas på detaljkortet **Aktiviteter**.
    * Skicka och arkivera: E-postmeddelandet skickas och sparas som ett dokument.

7. Om du har valt **Skicka och arkivera** öppnas dialogrutan **Dokument** när du skickar e-postmeddelandet. [Ange önskad information i dialogrutan][1].

8. Klicka på **Skapa** i dialogrutan **Dokument**.

> [!NOTE]
> Du kan också skicka ett e-postmeddelande till en intressent eller projektdeltagare genom att klicka direkt på deras e-postadress på detaljkortet **Intressenter** eller **Projektdeltagare**.
