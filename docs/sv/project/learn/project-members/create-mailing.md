# Skapa ett dokumentutskick till projektdeltagare

Om du vill skicka samma dokument till alla deltagare i ett projekt kan du använda funktionen **Marknadsföring** i SuperOffice. Gör något av följande:

* Välj <i class="ph ph-subtract-square" aria-hidden="true"></i> **Urval** i navigatorn, klicka på <i class="ph ph-dots-three-circle-vertical" aria-label="Task menu"></i> och välj **Skapa ett dokumentutskick**.

* Välj <i class="ph ph-target" aria-hidden="true"></i> **Marknadsföring** i navigatorn, välj fliken **Utskick** och [skapa ett nytt dokumentutskick][1].

> [!NOTE]
> I **Steg 4: Mottagare** väljer du från vilket projekt du vill lägga till projektdeltagare som mottagare.
