# Skapa ett dokumentutskick till projektdeltagare

Om du vill skicka samma dokument till alla deltagare i ett projekt kan du använda funktionen ![ikon][img1] **Marknadsföring** i SuperOffice. Gör något av följande:

* Välj ![ikon][img2] **Urval** i navigatorn, gå till fliken **Uppgift** och klicka på **Skapa ett dokumentutskick**.

* Välj ![ikon][img1] **Marknadsföring** i navigatorn, välj fliken **Utskick** och [skapa ett nytt dokumentutskick][1].

> [!NOTE]
> I **Steg 4: Mottagare** väljer du från vilket projekt du vill lägga till projektdeltagare som mottagare.
