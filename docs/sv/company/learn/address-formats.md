# Adressformat för andra länder

Du kan skapa företag med adresser i många olika länder. För att det ska bli lättare att skicka brev till företag i andra länder finns åtta olika adressformat i systemet: europeiskt, australiensiskt/kanadensiskt, amerikanskt, tyskt, brittiskt, spanskt, japanskt och ett generellt adressformat.

När du [skapar ett företag][1] och väljer vilket land företaget tillhör ändras adressfälten på fliken **Företag** enligt ditt val. Adressfälten som ändras beskrivs nedan.

> [!TIP]
> Du kan ändra standardadressformatet för ett land under Inställningar och underhåll.

## Generellt adressformat

Det här adressformatet är standardformatet för alla länder som inte omfattas av områdena nedan, till exempel länderna i Mellanöstern, Latinamerika och Afrika.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Adress 3 | Tredje adressraden |
| Postnummer/ort | Postnummer och postort |
| Land | Land |

## Tyskland

Det här adressformatet används för Tyskland. Tyskland har alltid olika postnummer/orter för kontorsadresser och postadresser.

| Fält | Beskrivning |
|---|---|
| Besöksadress | Kontorets adress |
| Postnummer/ort | Postnummer och postort för kontorsadress. |
| Postbox | Postadress |
| Postnummer/ort | Postnummer och postort för postadress. |
| Land | Land |

## Spanien

Det här adressformatet används för Spanien.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Adress 3 | Tredje adressraden |
| Postnummer/ort | Postnummer och postort för postadress. |
| Län | Län |
| Land | Land |

## Övriga Europa

Det här adressformatet används när du skapar ett företag i Norge och de flesta andra europeiska länder, förutom Storbritannien, Tyskland och Spanien, som har egna adressformat.

| Fält | Beskrivning |
|---|---|
| Besöksadress | Kontorets adress |
| Postadress | Postadress |
| Postnummer/ort | Postnummer och ort |
| Land | Land |

## Kanada

Det här adressformatet används för Kanada.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Ort | Postortens namn |
| Provins/postnummer | Provins och postnummer |
| Land | Land |

## Australien

Det här adressformatet används för Australien.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Ort | Andra adressraden |
| Provins/postnummer | Provins och postnummer |
| Land | Land |

## USA

Det här adressformatet används för USA. Det liknar formatet för Australien/Kanada men fälten har andra namn.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Ort | Postortens namn |
| Delstat/postnummer | Delstatskod (till exempel CA) och postnummer (zip code) |
| Land | Land |

## Brittisk

Det här adressformatet används för Storbritannien.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Adress 3 | Tredje adressraden |
| Ort | Postort |
| County/postnummer | Län (county) och postnummer. Till exempel "Somerset TA1 2WB" |
| Land | Land |

## Japan

Det här adressformatet används för Japan.

| Fält | Beskrivning |
|---|---|
| Adress 1 | Första adressraden |
| Adress 2 | Andra adressraden |
| Ort/postnummer | Postort och postnummer |
| Land | Land |
