# Redigera företagsinformation

1. [!include[Go to company](../../learn/includes/goto-company.md)]

1. Klicka på <i class="ph ph-pencil-simple" aria-label="Redigera"></i> i det övre högra hörnet av företagskortet.

1. Gör de nödvändiga ändringarna.

1. Klicka på **Spara** för att spara ändringarna, eller klicka på **Avbryt** för att avvisa dem. Båda alternativen avslutar redigeringsläget.

    Om knappen **Spara** är nedtonad måste alla obligatoriska fält fyllas i innan du kan spara.
