# Fliken Försäljning

![Fliken Försäljning -screenshot][img1]

På detaljkortet **Försäljning** visas [försäljning kopplad till företaget][2].

[!include[Contents of the Sales section tab](../../../learn/includes/sales-tab.md)]

[!include[Conf. columns](../../../learn/includes/tip-configure-columns.md)]

Om du dubbelklickar på en försäljning i listan öppnas fönstret Försäljning, som innehåller information om den aktuella försäljningen.

Klicka på **Lägg till** om du vill skapa en ny försäljning.

Klicka på knappen **Filtrera** ![ikon][img2] om du vill [filtrera listan][4] på datum, användare eller grupper.

Om du markerar alternativet **Ta med intressenter** längst ned på detaljkortet visas även försäljningar där företaget är involverat som [intressent][3].
