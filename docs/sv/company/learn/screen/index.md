# Detaljkorten i fönstret Företag

Fönstret Företag innehåller följande detaljkort:

| Detaljkort | Beskrivning |
|---|---|
| [Kontaktpersoner][1] | Här hittar du all information om medarbetarna på det aktiva företaget. |
| [Relationer][2] | Visar eventuella kopplingar mellan företag och/eller kontakter. |
| [Aktiviteter][3] | Innehåller alla aktiviteter som är kopplade till det aktiva företaget. |
| [Projekt][4] | Visar alla projekt för det aktiva företaget. |
| [Försäljning][5] | Här finns information om eventuella försäljningar kopplade till företaget. |
| [Ärenden][6] | Visar SuperOffice Service-ärenden för det aktiva företaget. |
| [ERP][7] | Här kan du koppla ett SuperOffice-företag till en kund/leverantör i ERP-systemet. |
