# Fliken Projekt

![Sektionen Projekt under företag -screenshot][img1]

På detaljkortet **Projekt** visas alla projekt för det aktiva företaget. För att ett projekt ska visas här måste minst en medarbetare på det aktuella företaget vara registrerad som projektdeltagare.

Klicka på **Lägg till** om du vill [skapa ett nytt projekt][2]

Dubbelklicka på ett projekt i listan för att öppna fönstret Projekt som innehåller information om det aktiva projektet.
