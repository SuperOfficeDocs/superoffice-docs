# Fliken Makron och skript

Så här går du till fliken **Makron och skript**:

1. [!include[Go to](../../../../learn/includes/goto-sm.md)]

1. Klicka på **CRMScript** och gå till fliken **Makron och skript**.

## Alternativ

På skärmen finns följande alternativ:

| Alternativ | Beskrivning |
|---|---|
 Nytt skript | Används för att [skapa ett nytt skript][2]. |
 Nytt makro | Används för att [skapa ett nytt makro][3]. |
 Ny mapp | Används för att [skapa en ny mapp][4]. |
 Sök | Används för att söka efter makron och skript i de valda mapparna och undermapparna. |

## Fönstret Skript

När du klickar på ett skript eller makro på fliken **Makron och skript** öppnas skärmen **Skript**. Utöver den information som visas längst upp på skärmen finns det mer skriptinformation på flikarna **Beskrivning** och **Skript**.

**Alternativ:**

| Ikon | Alternativ | Beskrivning |
|---|---|---|
| ![ikon][img1] | Makron och skript | Används för att återgå till listan över skript och makron. |
| ![ikon][img3] | Redigera skript | Används för att redigera [skriptet eller makrot][1]. |
| ![ikon][img5] | Kör skript | Används för att köra skriptet eller makrot i standardläge. |
| ![ikon][img6] | Felsök skript | Används för att öppna **felsökningsfönstret** och köra det valda skriptet eller makrot i felsökningsläge. Skriptet avbryts vid första raden. Du kan öppna en [felsökningssession][2] på fliken **Felsökningssessioner**. |
| ![ikon][img4] | Spåra skript | Används för att starta en [spårning av skriptet eller makrot][3]. |
