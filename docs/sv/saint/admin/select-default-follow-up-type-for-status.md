# Välja typ av standardhändelse för status

[!include[Requirement](../includes/note-saint-req.md)]

## Varför ska man välja standardhändelsetyp?

När du stöter på ett företag (kontakt eller projekt) med en status som kräver din uppmärksamhet, skulle du kunna klicka i mitten av kortet och för att öppna en ny händelse. Den här standardhändelsen kan innehålla text som tydligt anger vad du behöver göra med ett företag (kontakt eller projekt) som har denna specifika status.

## Steg

Du kan ange en standardhändelsetyp och beskrivande text som ska visas i SuperOffice CRM när användaren klickar på **Skapa händelse** i statusdialogrutan.

1. [!include[Open SAINT](includes/open-saint-select-tab.md)]

1. Välj önskad status i statuslistan till vänster på skärmen.

1. Klicka på fliken **Skapa händelse**.

1. Klicka på <i class="ph ph-caret-down" aria-label="Chevron"></i> till höger om fältet **Typ** och välj en uppföljning.

1. Ange önskad beskrivning.
