# Textanalys

Öka globaliseringen genom att detektera och översätta främmande språk i ärenden. I fönstret **Ärenden** kan dina kundtjänstmedarbetare översätta innehåll i meddelanden på andra språk.

* **Språkidentifiering**: Välj det här alternativet för att aktivera automatisk språkidentifiering för inkommande förfrågningar. Det identifierade språket visas under ärendemeddelanden i SuperOffice.

* **Översätt meddelande**: Välj det här alternativet för att aktivera översättning av inkommande förfrågningar. Ärendehandläggare kan då klicka på en länk i meddelandet för att översätta det. Språkidentifiering måste vara aktiverat för att den här funktionen ska kunna användas.

Öka kundnöjdheten genom att använda AI för att identifiera om meddelandet är viktigt/brådskande och kundens känslomässiga språk. Kör automatisk analys av inkommande meddelande i ärenden för att avgöra om innehållet är positivt, negativt eller neutralt.

* **Sentimentanalys**: Välj det här alternativet för att aktivera automatisk sentimentanalys för inkommande förfrågningar. Ärendehandläggare kan snabbt se sinnesstämningen i ett meddelande utifrån uttryckssymbolen (glad, ledsen eller neutral) under meddelandet i en förfrågning.
