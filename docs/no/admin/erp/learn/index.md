# ERP

[!include[Requirement](includes/req-integration-server.md)]

Hvis organisasjonen din bruker et ERP-system og har koblet det til SuperOffice CRM, har du muligheten til å synkronisere bedrifter (kunder og leverandører) og prosjekter mellom ERP-systemet og SuperOffice CRM.

På hovedkortet **Firma**, **Person** og **Prosjekt** vises **ERP-fanen**. Her kan du knytte et SuperOffice-firma til en kunde/leverandør i ERP-systemet. Personer som er registrert på SuperOffice-firmaet kan deretter knyttes til personer i ERP-systemet. Tilsvarende kan SuperOffice-prosjekter knyttes til prosjekter i ERP-systemet. Når du har konfigurert koblingen, synkroniseres data mellom ERP-systemet og SuperOffice CRM.
