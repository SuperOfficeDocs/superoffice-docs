# Opprette aktør i ERP-systemet

[!include[Requirement](includes/req-integration-server.md)]

Hvis du for eksempel vil knytte et SuperOffice-firma til en kunde i ERP-systemet, og denne kunden ikke finnes i ERP-systemet, kan du opprette kunden i ERP-systemet basert på dataene fra SuperOffice.

> [!NOTE]
> **Opprett**-knappen er deaktivert hvis SuperOffice ikke har lov til å opprette aktører i ERP-systemet.

1. [Åpne dialogboksen Koble til ERP][1].
2. Merk av for **Ingen treff i ERP** nederst i dialogboksen **Tilknytt ERP**, og klikk på **Opprett**. Dialogboksen **Opprett i ERP** vises.
3. Fyll inn relevante verdier.
4. Klikk på **Opprett**. Dette tar deg tilbake til **ERP-fanen** der de synkroniserte dataene vises.
5. Klikk på **Lagre**.
