# Globale preferanser

**Globale preferanser** under ![ikon][img1] **Preferanser** brukes til å konfigurere noen av de viktigste preferansene i SuperOffice CRM.

Områder du kan konfigurere preferanser for:

* Statistikk
* Funksjoner
* E-post
* Markedsføring
* Grupperte lister
* Mobile CRM
* Salg
* [Systeminnstillinger for SuperOffice Service][1]
* System
* Mål
* [Dataspeiling][5]
* [SMS-konfigurasjon][4]
* [SMTP][2] (bare ONSITE)
* [Passordregler for kundesenter][3]
