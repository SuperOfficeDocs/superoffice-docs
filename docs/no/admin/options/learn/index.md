# Systemvalg

I **Systemvalg**-bildet definerer du innstillinger som gjelder for hele systemet, nærmere bestemt for fritekstsøk, tidssoner, nummertildeling, etiketter og rapporter. De fleste bedrifter trenger ikke å endre standardinnstillingene som SuperOffice CRM leveres med. Men hvis du trenger å endre dem, bruker du **Systemvalg**-bildet.

## Faner i Systemvalg-bildet

| Fane | Beskrivelse | Les mer |
|---|---|---|
| Fritekstsøk | Her kan du styre hvordan funksjonen for fritekstsøk fungerer i SuperOffice CRM. | [Fritekstsøk][1] |
| Tidssoner | Her kan du styre tidssonefunksjonalitet. | [Tidssoner][2] |
| Nummertildeling | Her kan du angi et startsiffer for **Nummer**-feltene i Firma-bildet, Prosjekt-bildet, Salg-bildet og personkortet, samt for dokumentmaler. | [Nummertildeling][3] |
| Ledetekster | Her kan du endre navn på bestemte felt i SuperOffice CRM. | [Ledetekster][4] |
| Rapporter | Her kan du endre informasjon om standardrapporter og importere nye rapportmaler. | [Rapporter][5] |
| Kundespråk | Her kan du konfigurere kundespråkene og gjøre det enkelt å kommunisere med kunder på forskjellige språk. Du kan også angi kategorinavn i SuperOffice Customer Centre. | [Kundespråk][6] |

> [!NOTE]
> Hvis din organisasjon har lisens for [SAINT][7], inneholder **Systemvalg**-bildet også fanen **Statusovervåking**.
