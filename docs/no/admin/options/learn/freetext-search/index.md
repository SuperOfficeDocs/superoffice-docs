# Fritekstsøk

Ved hjelp av funksjonen for fritekstsøk i SuperOffice CRM kan brukerne søke etter tekst i hele applikasjonen. Dette gjelder tekst de selv har lagt inn i databasen, for eksempel firmanavn, avdelinger og tekst fra bildene Person og Dokument. Du kan definere nøyaktig hvordan dette skal fungere i SuperOffice CRM fra denne fanen i Innstillinger og vedlikehold.
