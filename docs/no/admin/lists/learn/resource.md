# Generelt - Ressurs

Ressurser vises i dialogboksen for oppfølginger i SuperOffice CRM, og her kan du legge til ressurser fra fanen **Deltakere**. Du kan for eksempel legge til ressurser som møterom og projektorer.

## Hva vil du gjøre nå?
