# GUI - Webpanel

Du kan ha snarveier til URL-adresser i SuperOffice CRM, slik at du kan åpne dem direkte fra programmet.

Hvilke applikasjoner som skal være tilgjengelige for brukerne i SuperOffice CRM, defineres i **Lister**-bildet i Innstillinger og vedlikehold.

Forekomstene vises i den rekkefølgen de blir lagt inn. Hvis du vil endre rekkefølgen, velger du et element og flytter det opp eller ned ved hjelp av pilknappene til venstre for listen ( ![ikon][img1] ![ikon][img2] ).

Når du har lagt til forekomster, kan du angi detaljer for hver forekomst. Du kan blant annet angi følgende:

* Hvilke [brukergrupper elementet er synlig for][2].
* Hvilken  [overskrift][1] den skal vises under.
