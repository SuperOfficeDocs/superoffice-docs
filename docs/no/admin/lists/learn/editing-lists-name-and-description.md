# Endre navn og beskrivelse

[!include[Find specific list](includes/tip-find-list.md)]

1. Gå til **Lister**-bildet.
2. [!include[Click arrow and select](includes/expand-list.md)] ønsket liste.
3. Klikk på **Rediger liste**-knappen.
4. Endre eventuelt navnet på listen i feltet **Navn**.
5. Endre eventuelt beskrivelsen av listen i feltet **Beskrivelse**.
6. Klikk på **OK** for å lagre endringene i Innstillinger og vedlikehold.
