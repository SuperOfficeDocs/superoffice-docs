# Legge til salgsbeløpsklasse

[!include[Must be admin](../../../learn/includes/req-admin.md)]

> [!NOTE]
> Listen **Salg - Beløpsklasse** er en del av funksjonen statusovervåking (SAINT) og krever en **Sales Premium** eller **Service Premium** brukerplan.

Beløpsklasse for salg kan brukes til å kategorisere salg i ulike klasser, for eksempel små, mellomstore, store og svært store salg. Du angir beløpsintervallet for de ulike beløpsklassene i valutaen firmaet ditt bruker (og flere valutaer, hvis du ønsker). Beløpsklasse kan for eksempel brukes til å søke etter store tapte salg innenfor en periode.

[!include[Note difference](includes/different-edit-list-item-dialog.md)]

[!include[Edit list item](includes/edit-list-item.md)] når forekomster er beløpsklasser:

1. Skriv inn ønsket navn på forekomsten i feltet **Navn**.

    > [!NOTE]
    > [!include[Mandatory](includes/note-mandatory-field.md)]

2. I feltene **Beløp fra** og **Beløp til** kan du angi beløpsintervallet for salg som skal tilhøre denne kategorien. Beløpet må angis i den valutaen firmaet bruker.

    > [!NOTE]
    > Beløpsintervallet kan ikke overlappe beløpsintervall i eksisterende beløpsklasser.

3. Velg valutaen som organisasjonen din bruker, i listen **Valuta**. Du kan legge til flere beløpsklasser med andre valutaer hvis det trengs i organisasjonen din.

4. Skriv eventuelt inn en beskrivelse av beløpsklassen i feltet **Beskrivelse**.

5. [!include[Save or cancel](includes/save-or-cancel.md)]
