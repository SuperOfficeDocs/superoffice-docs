# Logge av brukere (lokalt)

Hvis en bruker nektes tilgang til SuperOffice CRM, for eksempel etter et programkrasj, kan administratoren logge av brukeren fra Innstillinger og vedlikehold, slik at brukeren kan logge seg på igjen som vanlig.

1. [!include[Click Users](includes/goto-users.md)]

2. Velg fanen **Medarbeidere**.

3. Velg ønsket firma i listeboksen **Firma**.

4. Velg **Logget på Windows-klient** i listeboksen under brukerlisten.

5. [!include[Select user](includes/select-user.md)]

6. Klikk på knappen **Oppgave**, og velg **Logg av bruker**. Brukeren logges av.

> [!NOTE]
> Hvis brukeren ikke er logget på eller det er valgt noe annet enn **Logget på Windows-klient** i listeboksen **Vis**, er denne knappen deaktivert.
