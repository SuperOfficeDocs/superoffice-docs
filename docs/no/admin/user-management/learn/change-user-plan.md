# Endre brukerplaner for aktive brukere

Når personer tar på seg nye oppgaver eller får en ny stilling i bedriften din, kan det hende du må endre brukerplanen deres.

Du kan endre brukerplanen i delen Innstillinger og vedlikehold for å gi brukere tilgang til nye sett med funksjonalitet.

> [!TIP]
> Gi brukeren nødvendig opplæring og oppfølging av forløpet, slik at de tar i bruk de nye funksjonene raskere.

## Trinn

Brukerinnstillinger i SuperOffice CRM, der du endrer brukerplaner

1. Klikk på **Brukere** i Innstillinger og vedlikehold.

2. I vinduet for medarbeidere velger du den aktive brukeren som du skal endre brukerplan for.

    Sørg for at vedkommende har riktig rolle og tilhører riktig primærgruppe.

3. Velg ønsket brukerplan i feltet Brukerplan.

4. Klikk på Lagre. Neste gang brukeren logger på SuperOffice CRM, blir brukeren oppdatert med de nye funksjonene som er tilgjengelige.
