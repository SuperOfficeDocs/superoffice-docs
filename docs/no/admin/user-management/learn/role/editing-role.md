# Redigere en rolle

[!include[Requirement](../includes/note-anon-req.md)]

1. [!include[Open Roles](includes/open-roles.md)]

2. Velg fanen **Medarbeidere** eller **Eksterne**.

    [Slik redigerer du rollen for anonyme brukere.][1]

3. Dobbeltklikk på [rollen][3] du ønsker å redigere, i listen.

4. I dialogboksen **Opprett ny rolle** gjør du ønskede endringer i navnet og beskrivelsen til rollen, og deretter klikker du på **Lagre**.

5. [Rediger eventuelt rettighetene til rollen.][4]

6. Hvis du redigerer roller for medarbeidere: Gå eventuelt til fanen **Funksjonelle rettigheter**, og rediger de funksjonelle rettighetene til rollen.

    [!include[How to set functional rights](includes/add-remove-right.md)]

    Endringene lagres automatisk.
