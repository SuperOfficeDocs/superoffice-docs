# Importere fra Gmail

[!include[Back up database before import](includes/caution-backup-before-import.md)]

## Importer firmaer/personer fra Gmail

1. [!include[Open Import](includes/open-import.md)]

2. Klikk på ![ikon][img2] Gmail-ikonet under **Velg en importkilde**.

3. Gi tilgang. Dialogboksen som vises, avhenger av om du allerede er logget på Gmail

    * Hvis dialogboksen **Forespørsel om tillatelse** vises, er du allerede logget på Gmail. Klikk på **Tillat**.

    * Hvis dialogboksen **Google-kontoer** vises, er du ikke logget på Gmail. Skriv inn brukernavn og passord, og logg på Gmail. Klikk på **Tillat**.
        Gmail-firmaene dine vises i Innstillinger og vedlikehold.

    > [!NOTE]
    > Firmaene fra Gmail blir ikke importert til SuperOffice-databasen ennå. SuperOffice laster bare inn firmalisten fra Gmail, slik at du kan angi importinnstillingene.
