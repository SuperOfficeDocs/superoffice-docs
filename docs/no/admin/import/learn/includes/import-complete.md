Når importen er fullført, vises to lister:

* Nye firmaer og personer som ble importert.
* Eksisterende firmaer og personer som ble endret som følge av importen.

Over hver liste er det en kobling til utvalget med de nye/endrede firmaene/personene.

> [!TIP]
> Hvis du har importert nye firmaer/personer og det er problemer med de importerte dataene, kan du åpne dette utvalget i SuperOffice CRM og slette de nye firmaene/personene.
