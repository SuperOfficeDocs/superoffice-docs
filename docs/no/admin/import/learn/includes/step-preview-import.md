1. Klikk på **Neste**. **Forhåndsvisning**-vinduet vises.

1. I **Forhåndsvisning** kan du se gjennom importdataene før importen starter.

    | Ikon | Beskrivelse
    |:-:|---|
    | ![Ikon – firma ny][img1] | Firmaet finnes ikke og vil bli lagt til. |
    | ![Ikon – firma endret][img2] | Firmaet finnes allerede og vil bli oppdatert. |
    | ![Ikon – person ny][img3] | Personen finnes ikke og vil bli lagt til. |
    | ![Ikon – person endret][img4] | Personen finnes allerede og vil bli oppdatert. |
    | tom | Ingen oppdatering. |
