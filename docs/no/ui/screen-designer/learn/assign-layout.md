# Tilordne layout til gruppe, type eller mal

[!include[Krav](../../../../../common/includes/req-dev-tools.md)]

## Trinn

1. Åpne Skjermdesigneren.

1. Øverst velger du en skjerm eller dialog du vil tilpasse.

1. Velg tilordningsmetode (hvis ikke satt): Klikk på teksten ved siden av **Tilordne layout til**, og velg gruppe, type eller mal.

    * Firma og person: kun brukergruppe
    * Salg: brukergruppe eller salgstype
    * Prosjekt: brukergruppe eller prosjekttype
    * Sak: brukergruppe eller sakstype
    * Oppfølging: brukergruppe eller oppfølgingstype
    * Dokument: brukergruppe eller dokumentmal

1. Velg en layout i listen til venstre.

Klikk på feltet **Tilordne til**, og velg en gruppe, type, eller mal. Dine alternativer avhenger av den valgte skjermen.

## Begrensninger

* Standardlayouten kan ikke tilordnes.
* En brukergruppe kan kun tilordnes en layout per skjerm.
* Hvis du sletter en layout, vil de som er tilordnet den, gå tilbake til standardlayouten.
