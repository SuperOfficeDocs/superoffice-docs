# Globale profiler

[!include[Legacy](../includes/legacy-profiles.md)]

En global profil brukes som standardprofil og gjelder for alle som ikke har en personlig profil eller er tilordnet en felles profil fra rollen de tilhører.

## Opprett

Hvis du vil opprette en global profil, følger du instruksjonene for å [opprette en felles profil][1], men du merker også av for **Angi som standardprofil** øverst i **Rediger elementprofil**.
