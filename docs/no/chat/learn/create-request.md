# Opprette sak i en aktiv chat

Hvis du ikke kan løse spørsmålet via chatsamtalen, kan du opprette en sak som vil bli håndtert av en annen spesialist.

1. Klikk på <i class="ph ph-list" aria-label="Task menu"></i>, og velg deretter **Opprett sak**.
2. Fyll ut de relevante feltene i dialogboksen.
3. Chatloggen legges til saken, men du kan også legge til mer informasjon her hvis det er nødvendig.
4. Klikk på **Lagre**.
5. En kort melding som inneholder saks-IDen er lagt til i tekstfeltet i chatsamtalen. Klikk på **Send** for å sende denne til kunden.
