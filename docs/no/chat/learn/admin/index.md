# Chatadministrasjon

Med **Chat**-funksjonen i SuperOffice kan brukerstøtteagentene og selgerne dine chatte med kunder, potensielle kunder og andre personer via chatvinduet på websiden din. Denne funksjonen kan for eksempel brukes til service/støtte og forhåndssalg, og den lar seg enkelt integrere med FAQ, dokumenter/vedlegg og maler. Chatfunksjonen kan settes opp med flere chatkanaler og være rettet mot ulike områder, som teknisk støtte, produktspørsmål og salgsstøtte.

Agentene dine kan få tilgang til chat via Navigator i SuperOffice.

## Chatrobot-kobling

Chatroboter er laget for å simulere samtaler, og de brukes ofte til å hjelpe kunder som besøker websiden din med supportspørsmål, samle inn mer informasjon, foreslå produkter eller til og med hjelpe kunder med oppgaver som å betale en regning eller foreta en reservasjon. Bruksområdene er mange og stadig voksende, og selv om fordelene kan være mange, er det av natur ett av de mer komplekse bruksområdene for AI-teknologi. AI prøver å få maskiner til å lære kompleksiteten i menneskers språk, men det er fortsatt noen utfordringer i å snakke med uforutsigbare mennesker.

Vi har opprettet en chatrobot-kobling som gjør at SuperOffice kan snakke med robottjenesten du velger. Ved å bruke data fra robottjenesten vil chatroboten kunne forutsi neste spørsmål og gi ytterligere informasjon som kunden kan trenge. Hvis det er nødvendig, kan du alltid henvise kunden til en live chatagent eller tilby hjelp via sakshånteringssystemet i SuperOffice.

Det finnes forskjellige alternativer i chatkanalen for å aktivere chatroboten.

## Komme i gang med Chat

Live Chat er en populær måte for kunder og potensielle kunder å komme i kontakt med deg når de besøker websiden din. Denne veiledningen viser hvordan du konfigurerer chatwidgeten og deretter legger den til på websiden din.

Med SuperOffice Chat kan organisasjonen hjelpe og sende informasjon til kundene umiddelbart.

SuperOffice Chat kan brukes på alle sider på websiden din. Hvordan og hvor du vil bruke SuperOffice Chat bestemmer hvor mange chattekanaler du trenger, hvordan widgeten for chat skal se ut, og på hvilke sider på websiden du bør legge til widgeten for chat.

Dette er trinnene for å konfigurere chatwidgeten og deretter legge den til på websiden din:
