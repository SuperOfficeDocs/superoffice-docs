# Aktivere SuperOffice Gmail Link

Se denne videoen for å lære hvordan du konfigurerer Gmail Link og kobler Gmail-kontoen til SuperOffice (videolengde – 2:53), eller følg trinnene nedenfor:

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/kiQyO9KwiEs]

## Trinn

1. Gå til appoppføringen for Gmail Link i [SuperOffice App Store][1].

1. Klikk på **Installer** og velg hvilken nettleser du vil installere utvidelsen for.

    **Chrome:** Hvis du velger Chrome, blir du ført til Chrome Nettmarked. Web Store. Der klikker du på knappen **Legg til i Chrome** og bekrefter at du vil legge til denne utvidelsen.
    * **Firefox**: Hvis du velger Firefox, åpnes installasjonsprogrammet for Firefox og spør deg om å tillate plugin-en for Firefox nettleser. Bekreft ved å klikke på **Fortsett til installasjonen**.

1. Det siste trinnet for begge nettleserne er å logge på SuperOffice CRM-kontoen din. SuperOffice og Gmail vil automatisk oppdage at utvidelsen ble lagt til.
