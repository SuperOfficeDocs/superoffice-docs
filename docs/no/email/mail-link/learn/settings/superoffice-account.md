# SuperOffice-konto

Disse alternativene vises i ruten **SuperOffice-konto** i dialogboksen **SuperOffice CRM-alternativer**.

| Alternativ | Beskrivelse |
|---|---|
| Foretrukket SuperOffice-klient | Hvilken SuperOffice-klient du skal bruke når du [søker etter avsendere][2]: **SuperOffice CRM** (Windows-klient), **SuperOffice CRM** (webklient) eller **service**. |
| Brukernavn | Skriv inn brukernavnet ditt for den valgte klienten og klikk på **Test tilkobling** for å bekrefte at du kan logge på den aktuelle klienten, og dermed at forbindelsen mellom e-postklienten og SuperOffice-klienten fungerer. |

## Avanserte innstillinger

Hvis du klikker på pil ned til høyre for **Avanserte innstillinger**, vises følgende alternativer:

| Alternativ | Beskrivelse |
|---|---|
| Aktivere feilsøkingslogging | Dette alternativet bør aktiveres ved feilsøking hvis SuperOffice ber om det. |
| Vis loggfil | Åpne loggfilen for SuperOffice Mail Link. Også relevant ved feilsøking. |
| URL | Dette feltet inneholder adressen til SuperOffice-klienten der e-postene arkiveres. Bare relevant for SuperOffice CRM for Web og SuperOffice Service. |
