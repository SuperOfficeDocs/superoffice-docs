# Dialogboksen SuperOffice CRM-alternativer

Åpne dialogboksen **SuperOffice CRM-alternativer** ved å klikke på den lille pilen til høyre for **SuperOffice**-knappen i e-postvinduet eller innboksen.

Alternativt kan du velge **SuperOffice CRM-alternativer** på **Verktøy**-menyen i Outlook eller **Handlinger**-menyen i Notes.

Denne dialogboksen lar deg angi arkiv-, generelle- og kontoalternativer og vise versjonsinformasjon om SuperOffice Mail Link.

## Tilgjengelige systemvalg

* [Alternativer for arkivering][1]
* [Generelle innstillinger][2]
* [Invitasjoner][3]
* [SuperOffice-konto][4]

## Om-ruten

**Om**-ruten i dialogboksen inneholder informasjon om gjeldende versjon av SuperOffice Mail Link og om installerte komponenter.
