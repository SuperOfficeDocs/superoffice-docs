# Slå standardarkivering på/av

For å [arkivere][1] alle e-poster du sender fra e-postklienten din i SuperOffice CRM, aktiver funksjonen for arkivering som standard ved å velge fra alternativene under **Arkiver e-post som standard når:** i dialogboksen for **alternativer** i **SuperOffice CRM**. Alle e-poster du sender blir arkivert i SuperOffice CRM som standard.

Klikk på **Arkivering er PÅ / AV**-knappen i det nye e-postvinduet for å aktivere / deaktivere arkivet som standardfunksjon for e-postmeldinger opprettet i e-postklienten din.
