# Prosjektdeltakere

Prosjektdeltakere er deltakerne i et prosjekt. For hver deltaker kan du få tilgang til informasjon om firma, personnavn, funksjon i prosjektet, eventuell kommentar, e-postadresse og telefonnummer.
