# Vise oppfølginger og dokumenter

Som standard vises oppfølgingene og dokumentene for gjeldende prosjektstatus i nedre halvdel av detaljkortet, men du kan velge hvilke oppfølginger og dokumenter du vil vise, ved hjelp av avmerkingsboksene nederst på detaljkortet.

## Vise oppfølginger og dokumenter for én status

Du kan vise oppfølgingene og dokumentene til en av statusene ved å klikke på knappen med navnet på statusen.

## Vise alle oppfølginger og dokumenter

Merk av for **Vis alle statuser** for å vise alle foreslåtte og opprettede oppfølginger og dokumenter for alle statuser.

## Vise bare opprettede oppfølginger og dokumenter

Merk av for **Kun opprettede** for å vise bare oppfølginger og dokumenter som er opprettet, og skjule foreslåtte oppfølginger og dokumenter.

## Vise bare milepæler

Merk av for **Kun milepæler** for å vise bare de oppfølgingene som er angitt som milepæler. En oppfølging kan angis som en milepæl i Innstillinger og vedlikehold.

Du kan flytte prosjektet til neste status ved å bruke listen **Status** på hovedkortet eller ved å høyreklikke på statusknappen i detaljkortet og velge **Gå til denne statusen**.
