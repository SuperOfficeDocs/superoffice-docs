# Deaktivere prosjektguide

[!include[Must be admin](../../../learn/includes/req-admin.md)]

Hvis du vil fjerne en prosjektguide fra en prosjekttype, kan du deaktivere prosjektguiden. Du gjør dette ved å fjerne merkingen for **Denne prosjekttypen er tilknyttet en guide** for den aktuelle prosjekttypen (se [Legge til forekomster på listen Prosjekt - Type, status][1]).

Hvis du senere vil bruke prosjektguiden igjen, kan du bare merke av for dette alternativet på nytt. Eventuelle oppfølginger og dokumenter som var lagt til i statusene i prosjektguiden, blir hentet inn igjen automatisk. Du kan også legge til nye oppfølginger og/eller dokumenter eller slette de du ikke vil bruke med denne prosjektguiden.
