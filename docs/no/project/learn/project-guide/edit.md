# Redigere prosjektguide

[!include[Must be admin](../../../learn/includes/req-admin.md)]

1. Klikk på ![ikon][img1]**Arbeidsflyt**-knappen i navigatoren.
2. Velg fanen **Prosjektguide**.
3. Velg ønsket prosjekttype fra listeboksen øverst.
4. [Rediger informasjon om prosjekttype og tilhørende statuser][3].
5. [Rediger de enkelte statusene][3].
6. Rediger oppfølginger og dokumenter ved å dobbeltklikke på dem i listene nederst.

    * [Legge til oppfølging for en status][2]
    * [Legge til dokument for en status][1]

    Endringene lagres automatisk.
