# Slette et helt prosjekt med tilhørende data

1. [!include[Go to project](includes/goto-project.md)]
2. Klikk på ![ikon][img1] **Oppgave**-knappen.
3. Velg **Slett 'prosjektnavn'**.
4. Når du får spørsmål om du virkelig vil slette prosjektet, klikker du på **Ja**.
