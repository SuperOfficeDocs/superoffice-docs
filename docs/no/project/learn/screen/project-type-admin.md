# Prosjekt – Type, status

Fanen **Prosjekt** i Prosjekt inneholder blant annet feltet **Type**. Dette feltet har en forhåndsdefinert valgliste, som defineres i Lister-bildet i Innstillinger og vedlikehold.

Hvert prosjekt i SuperOffice CRM må ha en prosjekttype, og hver prosjekttype kan inneholde flere [statuser][1]. Du kan også angi at bestemte prosjekttyper skal ha en [prosjektguide][3], som kan lede deg gjennom de ulike statusene i et prosjekt, med tilhørende oppfølginger og dokumenter.

## Hva vil du gjøre nå?

* [Legge til prosjekttype][2]
