# Prosjekt – Status

Fanen **Prosjekt** i Prosjekt inneholder blant annet feltet **Status**. Dette feltet har en forhåndsdefinert valgliste, prosjektstatus, som defineres i Lister-bildet i Innstillinger og vedlikehold.

Hvilke statuser som vises i listen, er avhengig av [prosjekttypen][1]. Du kan velge hvilke statuser som skal være tilgjengelig for de ulike prosjekttypene, i listen Prosjekt – Type, status.

## Hva vil du gjøre nå?

* [Legge til en prosjektstatus][2]
