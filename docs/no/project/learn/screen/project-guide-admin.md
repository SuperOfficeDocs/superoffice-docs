# Prosjektguide

I fanen Prosjektguide i **Arbeidsflyt**-bildet i Innstillinger og vedlikehold kan du lage prosjektguider som fungerer som veiledning gjennom prosjektene. En prosjektguide består av statuser som inneholder oppfølginger og dokumenter som skal veilede deg gjennom prosjektet. Hver prosjektguide er tilknyttet en prosjekttype, og prosjektguiden aktiveres i Prosjekt-bildet når brukeren oppretter et nytt prosjekt eller endrer prosjekttypen på et eksisterende prosjekt, og velger en prosjekttype som har en tilknyttet prosjektguide.

## Eksempel

Prosjekttypen "Konferanse" kan bestå av tre statuser: "Planlagt", "Under arbeid" og "Lukking". Statusen "Planlagt" kan for eksempel ha oppfølgingen "Planleggingsmøte" (en avtale) og dokumentet "Konferanseprogram" (et notat).
