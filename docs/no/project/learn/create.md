# Opprette et prosjekt

Se denne videoen, eller følg trinnene nedenfor for å lære hvordan du oppretter et prosjekt (videolengde – 2:49):

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/zLJwNll223w]

## Trinn

1. Klikk på **Ny** i toppraden, og velg **Prosjekt**.

2. I det tomme prosjektkortet fyller du ut ønskede opplysninger i feltene i fanen **Prosjekt**. Se detaljer nedenfor.

3. Alternativt kan du gå til en av fanene for å legge inn ytterligere informasjon.

4. Klikk på **Lagre**. Nå kan du [legge til prosjektdeltakere][5].
