# Aktivere bruk av tidssoner

1. Gå til Innstillinger og vedlikehold og velg <i class="ph ph-sliders-horizontal" aria-hidden="true"></i> **Alternativer**.
2. Velg fanen **Tidssoner**.
3. Klikk på **Aktiver tidssoner**.
4. Velg standard tidssone i dialogboksen. Du kan endre dette senere.
5. Klikk på **Lagre**. Listen over tidssoner er aktivert.
6. Hvis koblingen **Last ned nye tidssoneoppdateringer** vises øverst i bildet, kan du klikke på den for å oppdatere tidssonene i SuperOffice.

> [!TIP]
> Gå til **Preferanser** for å vise tidssonevelgeren og velge standard tidssone.
