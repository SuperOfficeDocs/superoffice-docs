# Bruk statusovervåking i dynamiske utvalg

Hvis du har kjøpt en lisens for SAINT, kan du bruke statuser definert i Innstillinger og vedlikehold som kriterium for å opprette dynamiske utvalg. På denne måten kan du enkelt skaffe deg oversikt over firmaer, personer eller prosjekter som oppfyller statuskriteriene du er interessert i. SAINT-kriteriene finner du under **Tellere (SAINT)** i [listen over kriterier][1] i skjermbildet **Finn**.
