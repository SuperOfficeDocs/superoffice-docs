# Redigere salgsguide

[!include[Must be admin](../../../learn/includes/req-admin.md)]

1. Klikk på **Arbeidsflyt**-knappen i navigatoren (![ikon][img1]).
2. Velg fanen **Salgsguide**.
3. Velg ønsket salgstype i listen **Salgstype**.
4. Klikk på **Rediger**.
5. [Rediger de enkelte fasene][3].
6. Rediger oppfølginger og dokumenter ved å dobbeltklikke på dem i listene nederst.

    * [Legge til oppfølging for en fase][1]
    * [Legge til dokument for en fase][2]

    Endringene lagres automatisk.
