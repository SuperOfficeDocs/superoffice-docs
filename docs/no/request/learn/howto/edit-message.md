# Redigere melding

Kun brukere med funksjonsrettigheten *general administrator* kan redigere egenskapene til en melding i en sak. Det er imidlertid ikke mulig å redigere teksten i meldingen.

1. [Åpne saken][1].
1. Klikk på fanen **Meldinger**.
1. Klikk på ![ikon][img1] **Oppgave**-knappen, og velg **Endre**.
1. Her har du følgende alternativer:
    * **Forfatter**: Viser navnet på personen som opprettet saken.
    * **Medgått tid**: Viser hvor mye tid som er brukt på denne meldingen.
    * **Tilgangsnivå**: Tilgangsnivået for denne meldingen. Du kan velge mellom **Eksternt** og **Internt**.
    * **Vedlegg**: Hvis meldingen inneholder vedlegg, vises de i denne tabellen. Du kan slette eksisterende vedlegg, men du kan ikke legge til nye.

1. Klikk på **Lagre** når du er ferdig, eller **Avbryt** hvis du vil forkaste endringene. Du kan også slette hele meldingen ved å klikke på **Slett** og deretter **OK.**
