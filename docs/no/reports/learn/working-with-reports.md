# Jobbe med rapporter

I **Rapporter**-bildet kan du både definere søkekriterier for rapporter (se [Favorittrapporter][1]) og bruke [standardrapporter][2].

> [!NOTE]
> Du kan ikke lage helt nye søkekriterier i SuperOffice CRM. Søkekriteriene er forhåndsdefinert for de ulike rapporttypene, og du kan bare redigere verdier og variabler for disse og eventuelt lagre dem som favorittrapporter. Hvis du vil legge til nye søkekriterier og opprette nye standardrapporter, må du kjøpe SuperOffice Reporter Studio.
