# Slette rapporter i rapportarkivet (detaljkortet Dokumenter)

Du kan slette lagrede rapporter i rapportarkivet.

1. Gå til **Rapporter**-bildet.
2. Merk rapporten(e) du ønsker å slette, i rapportarkivet (detaljkortet **Dokumenter**) nederst i vinduet.
3. Klikk på **Slett**-knappen eller høyreklikk på den ønskede rapporten og velg **Slett**.
4. Bekreft slettingen i dialogboksen som vises.
