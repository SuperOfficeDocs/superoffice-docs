# Redigere favorittrapporter

> [!NOTE]
> Dette forutsetter at du har minst én [favorittrapport][1].

1. [!include[How to-goto-reports](../includes/goto-reports.md)]
2. Velg **favorittfanen**. Favorittrapportene vises på rapportlisten.
3. Dobbeltklikk på rapporten du ønsker å redigere, eller merk den og velg **Rediger**.
4. Oppdater navnet og beskrivelsen etter behov.
5. Klikk på **OK**.

> [!NOTE]
> Rapporten endres med én gang, og du har ikke mulighet til å angre.
