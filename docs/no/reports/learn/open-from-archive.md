# Åpne rapporter fra rapportarkivet (detaljkortet Dokumenter)

Du åpner lagrede rapporter ved å dobbeltklikke på ønsket rapport på detaljkortet for Rapporter, eller høyreklikke på den og velge **Åpne**.

Last ned dokumentet fra dialogboksen **Dokument**. Rapporten åpnes i standard visningsprogram for [formatet du lagret rapporten i][1].
