# Opprette rapporter

I bildet **Rapporter** kan du bare [kjøre tidligere definerte rapporter][1]. Du kan ikke definere helt nye rapporter.

Hvis du vil [lagre en rapport som et dokument][4] (type = Rapport) i SuperOffice CRM, kan du klikke på **Lagre**-knappen i **Rapporter**-bildet.

Du kan definere nye rapporter i SuperOffice Reporter Studio og importere dem i Innstillinger og vedlikehold (**Systemvalg** > **Rapporter**).

## Opprette en egen rapport

1. [Endre søkekriteriene][2] i en eksisterende rapport.
1. [Lagre den som din egen favorittrapport][3].
