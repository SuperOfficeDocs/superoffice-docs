# Standardrapporter

I **Rapporter**-bildet kan du også kjøre forskjellige typer standardrapporter. Disse rapportene er sortert etter type, og du henter dem frem ved å klikke på fanene øverst i Rapporter-bildet (se [Vise rapportegenskaper][1]).

En rekke rapporter følger med SuperOffice CRM som standard, men en organisasjon kan også opprette sine egne rapporter i SuperOffice Reporter Studio, som er et valgfritt tillegg. Når du bruker Rapporter-bildet, vil du imidlertid ikke merke noen forskjell på hvor disse rapportene kommer fra, og derfor omtaler vi alle som standardrapporter her.

> [!TIP]
> Du kan lagre standardrapporter du bruker ofte, som [favorittrapporter][2] for å gjøre dem lettere tilgjengelig. I tillegg kan du redigere standardrapporter og lagre dem som favoritter.

## <a id="type" />Typer

| Type | Beskrivelse |
|---|---|
| Firma/person | Viser firma-/personrapporter. |
| Prosjekt | Viser prosjektrapporter. |
| Salg | Viser salgsrapporter. |
| Aktivitet | Viser rapporter for oppfølginger og dokumenter. |
| Utvalg | Viser utvalgsrapporter. |

## Redigere og slette standardrapporter

Du kan verken redigere eller slette selve standardrapportene. Det du imidlertid kan gjøre, er å [lagre en standardrapport som favorittrapport][3] og deretter [justere søkekriteriene slik du ønsker][4].
