# Adresseformater for ulike land

Du kan opprette firmaer med adresser i ulike land. For å lette internasjonal korrespondanse er flere adresseformater tilgjengelige, inkludert europeisk, australsk/kanadisk, amerikansk, tysk, britisk, spansk, japansk og et generelt format.

Når du [oppretter et firma][1] og velger land, tilpasses adressefeltene i fanen **Firma** deretter. De spesifikke adressefeltene som endres, er beskrevet nedenfor.

> [!TIP]
> Du kan endre standard adresseformat for et land i Innstillinger og vedlikehold.

## Generelt adresseformat

Dette adresseformatet er standardformat for alle land som ikke omfattes av områdene som nevnes nedenfor, for eksempel land i Midtøsten, Latin-Amerika og Afrika.

| Felt | Beskrivelse |
|---|---|
| Adresse 1 | Første adresselinje |
| Adresse 2 | Andre adresselinje |
| Adresse 3 | Tredje adresselinje |
| Postnr/sted | Postnummer og poststed |
| Land | Land |

## Tyskland

Dette adresseformatet brukes for Tyskland. Tyskland har alltid forskjellige postnummer/-sted for besøksadresse og postadresse.

| Felt | Beskrivelse |
|---|---|
| Besøk | Besøksadresse |
| Postnr/sted | Postnummer og poststed for besøksadresse. |
| Postboks | Postadresse |
| Postnr/sted | Postnummer og poststed for postadresse. |
| Land | Land |

## Spania

Dette adresseformatet brukes for Spania.

| Felt | Beskrivelse |
|---|---|
| Adresse 1 | Første adresselinje |
| Adresse 2 | Andre adresselinje |
| Adresse 3 | Tredje adresselinje |
| Postnr/sted | Postnummer og poststed for postadresse. |
| Fylke | Fylke |
| Land | Land |

## Storbritannia

Dette adresseformatet brukes for Storbritannia.

| Felt | Beskrivelse |
|---|---|
| Adresse 1 | Første adresselinje |
| Adresse 2 | Andre adresselinje |
| Adresse 3 | Tredje adresselinje |
| Sted | Poststed |
| Fylke/postnr | Fylke og postnummer. For eksempel Somerset TA1 2WB |
| Land | Land |

## Europa forøvrig

Dette adresseformatet brukes når du oppretter et firma i Norge og de fleste europeiske land, unntatt Storbritannia, Tyskland og Spania, siden de har egne adresseformater.

| Felt | Beskrivelse |
|---|---|
| Besøk | Besøksadresse |
| Post | Postadresse |
| Postnr/sted | Postnummer og sted |
| Land | Land |

## Canada

| Adresse 1 | Første adresselinje |
| Adresse 2 | Andre adresselinje |
| Sted | Navn på poststed |
| Provins/postnr | Provins og postnummer |
| Land | Land |

## Australia

| Adresse 2 | Andre adresselinje |

## USA

Dette adresseformatet brukes for USA. Det er likt oppsettet for Australia/Canada, bare at feltene har forskjellige navn.

| Felt | Beskrivelse |
|---|---|
| Adresse 1 | Første adresselinje |
| Adresse 2 | Andre adresselinje |
| Sted | Navn på poststed |
| Stat/postnummer | Delstatskode (for eksempel CA) og postnummer |
| Land | Land |

|---|---|
