# Legge til Sync Connector

1. Gå til fanen **Sync** i Tilbud/Sync-bildet.
2. Klikk på **Legg til** nederst i listen **Sync Connectors**.
3. I dialogboksen angir du navnet og URL-adressen til Sync Connector.
4. Klikk på **OK**.

    Sync Connector vises i listen.
