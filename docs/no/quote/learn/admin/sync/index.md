# Sync

I fanen **Sync** i Tilbud/Sync-bildet vises Sync Connectors og tilknyttede ERP-koblinger. Sync Connectors håndterer synkroniseringen av data mellom SuperOffice og ERP-systemet, mens ERP-koblingene er bindeleddet mellom SuperOffice og Sync Connectors.
