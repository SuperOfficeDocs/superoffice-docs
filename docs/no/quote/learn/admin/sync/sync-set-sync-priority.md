# Angi synkroniseringsprioritet

Hvis SuperOffice er tilknyttet flere ERP-systemer, og det til enhver tid er flere aktive Sync Connectors og ERP-koblinger, bør du vurdere hvilke ERP-koblinger som skal ha prioritet hvis det er mye datatrafikk. I dialogboksen **Angi synkroniseringsprioritet** kan du angi prioriteringsrekkefølgen for ERP-koblingene.

1. Gå til fanen **Sync** i Tilbud/Sync-bildet.
2. Klikk på **Angi synkroniseringsprioritet** nederst i listen **ERP-koblinger**.
3. I dialogboksen merker du en ERP-tilkobling du vil angi synkroniseringsprioritet for.
4. Klikk på pilene ![ikon][img1] ![ikon][img2] nederst for å angi prioritet.
5. Klikk på **Ferdig** når du har angitt prioriteten for ERP-tilkoblingene.
