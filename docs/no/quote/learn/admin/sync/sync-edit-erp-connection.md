# Redigere ERP-kobling (Sync Connector)

1. Gå til fanen **Sync** i Tilbud/Sync-bildet.
2. Fjern merkingen i avmerkingsboksen i kolonnen **Aktiv** for ERP-koblingen du vil redigere.
3. Dobbeltklikk på ERP-koblingen.
4. I dialogboksen **Rediger ERP-tilkobling**, [redigerer du ERP-tilkoblingen][1].
