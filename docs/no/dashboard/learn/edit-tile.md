# Redigere en figur

1. Gå til ønsket dashbord.

2. Klikk på **Oppgave** -knappen, og velg **Legg til figurer / rediger layout**.

3. Klikk på **Rediger figur**-ikonet (![ikon][img2]) på dashbordet.

4. I **Datasett** -fanen kan du [redigere kriteriene][1] for dataene som vises i figuren. Husk å oppdatere figurnavnet for å gjenspeile eventuelle endringer.

5. I **Layout** -fanen kan du redigere ulike innstillinger, verdier og formatering av figuren. Forhåndsvisningen til høyre oppdateres med de valgte innstillingene.

6. Klikk på ![ikon][img1] for å [legge til en oversettelse][2] til figurnavnet, titlene og etikettene.

7. Klikk på **Lagre**.
