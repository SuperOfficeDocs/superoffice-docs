# Dashbord for utsendelser

Opprett dashbord for markedsføringsoppgavene dine, for eksempel skjemainnsendinger og utsendelser, for å spore og analysere ytelse, trender og produktivitet på tvers av alle markedsføringsaktiviteter.

Når dashbordet er opprettet, må du legge til og redigere figurer.

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/lz9rEZI_2SY]

Studer eksemplene nedenfor for å få ideer til figurer som du kan legge til i dashbordet for utsendelse eller skjemainnsending.

## Eksempel på forskjellige figurer for utsendelser

* Figuren **Topputsendelser i år** er en god måte å se hvilke utsendelser som gir best resultater på. Se gjennom disse utsendelsene for å få en forståelse av hvilke typer utsendelser, utforming og innhold som passer best for kundebasen.

* Bruk figuren **Gjennomsnittlig åpningsrate YTD** til å se om mottakerne klikker på lenker eller bilder i utsendelsene dine. Hvis åpningsraten din er lav, kan det være lurt å prøve om andre overskrifter kan øke frekvensen.

![Eksempelet viser forskjellige figurer for utsendelser, som du kan bruke til å få oversikt over dataene dine -screenshot][img1]

## Eksempel på forskjellige figurer for skjemainnsendinger

* Bruk figuren **Aktive skjemaer, dette kvartalet** til å få en oversikt over skjemaene som ble sendt inn dette kvartalet.

* Figuren **Mine kunders skjemainnsendinger denne måneden** kan hjelpe deg med å holde oversikt over kundenes innsendinger hver måned.

* Figuren **Ikke-behandlede innsendinger** viser hvilke innsendinger som ikke er behandlet, og som trenger din oppmerksomhet.

![Eksemplet viser ulike skjemainnsendingsfigurer du kan bruke til å få en oversikt over dataene dine -screenshot][img2]
