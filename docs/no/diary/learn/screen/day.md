# Fanen Dag

**Dag**-fanen viser én dag om gangen, med timeintervaller langs venstre kant.

![Dagbok-bilde, Dag-fanen -screenshot][img1]

Hvis [tidssoner][1] er aktivert, kan du bytte tidssone nederst til venstre.
