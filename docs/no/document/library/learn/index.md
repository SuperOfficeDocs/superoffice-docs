# Dokumentbibliotek

Et **dokumentbibliotek** er der alle filer, maler og e-poster lagres.

Som standard er SuperOffice-området ditt koblet til SuperOffice-dokumentbiblioteket. Hvis organisasjonen din bruker SharePoint, kan en administrator konfigurere SuperOffice til å bruke SharePoint som foretrukket dokumentbibliotek.

![På Preferanser i Innstillinger og vedlikehold finner du Dokumentbiblioteket hvor du kan konfigurere SharePoint-integrasjonen -screenshot][img1]

## SuperOffice dokumentbibliotek

Standardoppsettet for SuperOffice er **SO_Arc**-dokumentbiblioteket. Alle filer, maler og e-poster lagres i dette biblioteket.

## SharePoint dokumentbibliotek integrasjon

SuperOffice's SharePoint-dokumentbibliotek integrasjon lar deg åpne, redigere og lagre dokumenter i Microsoft 365 SharePoint direkte fra SuperOffice.

Denne integrasjonen lar deg lagre alle dokumenter og e-poster i SharePoint Online, hvor de kan nås av alle i firmaet, også de som ikke bruker SuperOffice CRM. Tilgangen til SuperOffice-dokumentområdet i SharePoint kan styres gjennom Microsoft Entra domenegrupper (tidligere Azure Active Directory).
