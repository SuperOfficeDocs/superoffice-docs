# Skriv ut utvalg

1. [!include[Start](../includes/steps-start-task.md)]

1. Sett opp de første fem kolonnene du vil inkludere. Gruppering eller kolonneberegninger som brukes på utvalget ditt, vil vises i den utskrevne rapporten.

1. Klikk på **Oppgave**-knappen og velg **Skriv ut utvalg**.

1. I vinduet som viser innholdet som skal skrives ut, klikker du på **Skriv ut**.
