# Import

[!include[Back up database before import](../../../../admin/import/learn/includes/caution-backup-before-import.md)]

Fra **Import**-bildet kan du hente inn viktige data til SuperOffice-databasen. Du kan for eksempel importere et kunderegister du har fra før, slik at du kan bruke det i SuperOffice CRM. Helt konkret kan du i standardversjonen av Innstillinger for Windows-klient importere følgende data til SuperOffice-databasen:

* Firmaer med tilhørende brukerdefinerte felt og interesser
* Personer med tilhørende brukerdefinerte felt og interesser
* Prosjekter med tilhørende brukerdefinerte felt
* Oppfølginger
* Salg
* Produkter
* Valutaer
* Postnummer
* Røde dager

Du kan også [importere firmaer/personer og produkter i Innstillinger og vedlikehold][9].

Hvilke felt du finner i **Import**-bildet, og hvordan du bruker dem, er beskrevet under [Importere filer][2]. Før du begynner å importere, bør du også lese [Importrutiner][1]
