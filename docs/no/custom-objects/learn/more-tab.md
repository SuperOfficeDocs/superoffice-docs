# Mer-fanen

**Mer-fanen** i SuperOffice CRM lar brukere se og administrere brukerdefinerte felt for ulike enheter, inkludert firma, personer, salg, prosjekter, oppfølginger og dokumenter. Denne tilpasningsfunksjonen hjelper deg med å fange spesifikk informasjon som er unik for din organisasjons behov, og forbedrer fleksibiliteten til din CRM.

Brukerdefinerte felt i **Mer-fanen** gjør det enklere for organisasjonen din å spore ekstra detaljer som maskinvare eid av kontakter, ansatturnover eller annen tilpasset data som ikke er tilgjengelig som standard i SuperOffice CRM.

Hvis for eksempel organisasjonen din selger IT-produkter og trenger felt for hver kontakts maskinvare og programvare, kan du spesifisere i Innstillinger og vedlikehold at disse feltene skal legges til her. Andre eksempler: År ansatt, ERP-ID-er, brukertype, CXM-info.

## Innhold i Mer-fanen

**Mer-fanen** er tilgjengelig for flere enhetstyper, og lar brukerne lagre tilpasset informasjon skreddersydd til deres forretningsbehov:

| Enhet | Beskrivelse av Mer-fanen|
|---|---|
| Firma | Viser tilpassede felt som omsetning, antall ansatte og notater.|
| Kontakt | Lagre bruker-spesifikk data, for eksempel maskinvare, programvare eller kontaktroller. |
| Salg | Inkluderer ekstra informasjon som er spesifikk for salget, for eksempel godkjenningsstatus. |
| Prosjekt | Sporer ekstra detaljer som klientkrav eller prosjekttagger.   |
| Oppfølging | Holder ekstra felt for videre kategorisering og dokumentasjon av aktiviteter. |
| Dokument | Lagre tilpassede dokument-spesifikke felt, for eksempel versjonsnotater eller metadata. |

Hver av disse **Mer-fanene** lar brukerne tilpasse CRM-oppsettet til deres spesifikke krav uten omfattende tilpasningsinnsats.

## Se informasjon i Mer-fanen

For å vise data i Mer-fanen:

1. Gå til den relevante skjermen, som Firma eller Person.
1. Velg **Mer-fanen** for å se de tilpassede feltene.
