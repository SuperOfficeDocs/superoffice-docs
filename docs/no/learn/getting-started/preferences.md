# Endre en preferanse <i class="ph ph-user-circle" aria-hidden="true"></i>

Du kan angi innstillinger for ulike deler av SuperOffice CRM, for eksempel antall varer som skal vises i historikklisten, og standardvalutaen for salg. Alle preferanseinnstillinger er definert i Innstillinger og vedlikehold, men brukerne kan endre noen individuelt i SuperOffice CRM.

![Gå til din personlige innstilling, og velg innstillingen du vil endre -screenshot][img13]

## Trinn

1. Velg <i class="ph ph-user-circle" aria-hidden="true"></i> **Personlige innstillinger** > **Preferanser**.

2. I dialogboksen **Preferanser** velger du ønsket preferanse fra listen til venstre.

3. Angi ønsket innstilling i vinduet til høyre.

4. Klikk på **Lagre** for å lagre endringene og lukke dialogboksen **Preferanser**. Du kan også klikke **Avbryt** for å forkaste endringene.

> [!NOTE]
> Hvis du vil endre språket i Mobile CRM-appen, klikker du på **Oppgave**-knappen (<i class="ph ph-dots-three-circle-vertical" aria-hidden="true"></i>), deretter på **Innstillinger**, og på **Språk** og velger blant alternativene.

## <a id="options"></a>Dialogboks for Preferanser – alternativer

| Knapp | Alternativ | Beskrivelse |
|---|---|---|
| <i class="ph ph-list-checks" aria-label="Check"></i> | Standardverdier | Standardverdiene i SuperOffice CRM. For eksempel standard virksomhet for nye firmaer, standard varighet for oppfølginger, standardmalen for utgående dokumenter. |
| <i class="ph ph-calendar-blank" aria-label="Diary icon"></i> | Dagbok | Standardinnstillingene for dagboken og oppfølgingene, for eksempel standard starttid for arbeidsdagen og varigheten av alarmer. |
| <i class="ph ph-toggle-right" aria-label="Functions"></i> | Funksjoner | De generelle innstillingene og funksjonene, inkludert om du vil vise et Dagens tips når du starter SuperOffice (Vis tips ved oppstart). |
| <i class="ph ph-question" aria-label="Question mark"></i> | Kommunikasjon i appen | Innstillingene for meldinger i appen. |
| <i class="ph ph-at" aria-label="Email icon"></i> | E-post | Innstillingene for e-post og faks, for eksempel e-postklienten du har installert. |
| <i class="ph ph-bell" aria-label="Bell icon"></i> | Varsler via popup; Varsler via e-post; Varsler via SMS | Innstillingene for varslinger om saker. |
| <i class="ph ph-user-circle" aria-label="Contact icon"></i> | Person | Innstillingene for personer. |
| <i class="ph ph-ticket" aria-label="Request icon"></i> | Sak | Innstillingene for saker. For varslingsinnstillinger, se "Varsling via ..." ovenfor. |
| <i class="ph ph-currency-circle-dollar" aria-label="Sale icon"></i> | Salg | Innstillingene for salgsfunksjonen. |
| <i class="ph ph-sort-ascending" aria-label="Sort icon"></i> | Sortering | Hvordan medarbeidere, lister og personer bør sorteres, for eksempel hvis medarbeiderne skal sorteres etter etternavn. |
| <i class="ph ph-gear" aria-label="Gear icon"></i> | System | Systeminnstillingene, for eksempel hjemland, valuta og standard dokumentarkiv. |
| <i class="ph ph-magic-wand" aria-label="Magic wand"></i> | Visuelle effekter | Innstillingene for visuelle effekter, for eksempel antall elementer i historikklisten, eller om du vil at søndag skal være den første dagen i uken. |
