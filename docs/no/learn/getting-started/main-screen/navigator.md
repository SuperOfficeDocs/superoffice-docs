# Navigatoren

Du beveger deg mellom hovedseksjonene i SuperOffice ved hjelp av navigatoren på venstre side.

> [!TIP]
> Du kan endre bredden på navigatoren ved å klikke på ![icon][img21] og ![icon][img22] over navigatoren.

| Knapp | Beskrivelse | Les mer |

| ![icon][img5] | Åpner skjermbildet Person, som inneholder alle personene dine. | [Person][3] |

| ![icon][img8] | Åpner skjermbildet Prosjekt, som inneholder informasjon om ulike prosjekter. | [Prosjekt][6] |

## <a id="navigator"></a>Alternativer i navigatoren

Når du klikker på en navigatorknapp, vises en eller flere av knappene nedenfor.

| Knapp | Beskrivelse |
|:-:|---|
| ![icon][img26] | **Handlinger/lister**: Klikk på denne knappen for å vise tilgjengelige handlinger/funksjoner, lister og lignende for den valgte navigatorknappen. |
| ![icon][img23] | **Historikk**: Klikk på denne knappen for å vise nylig åpnede oppføringer, f.eks. saker, personer eller utvalg. |
| ![icon][img24] | **Favoritter**: Klikk denne knappen for å vise postene du har [angitt som favoritter][15]. |
| ![icon][img25] | **Søk**: Klikk på denne knappen for å vise resultatene av det siste søket. Søkefeltet over knappene er dynamisk - begynn å skrive inn søkefrasen for å starte søket. |

> [!NOTE]
> Ved å klikke på **Dashboard**-knappen åpnes bare dashbordet.

Hvis du holder musepekeren over en oppføring i noen av listene, vises en infoboks som inneholder tilleggsdata om oppføringen.
