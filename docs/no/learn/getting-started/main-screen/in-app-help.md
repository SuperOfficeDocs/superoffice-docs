# Bruke hjelp i appen

## Bruke hjelpefilene

Denne hjelpen er utviklet for å gi deg omfattende og lett tilgjengelig informasjon om hvordan du bruker SuperOffice CRM. Uansett hvor du er i programmet og hva du gjør, kan du når som helst få hjelp til akkurat den oppgaven du holder på med.

Du kan åpne hjelpen på forskjellige måter:

* Velg **Hjelpefiler (F1)** på **Hjelp**-menyen.
* Trykk på **F1**. Dette åpner hjelpeemnet for det aktuelle skjermbildet.
* Klikk på **Hjelp**-knappen som vises i de enkelte vinduene.
