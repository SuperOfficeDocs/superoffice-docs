# SuperOffice som din løsning (Windows)

Dette er standardmåten for å koble e-postkontoen, dokumenter og kalenderen til SuperOffice.

> [!NOTE]
> Disse instruksjonene gjelder kun for Windows. Bruker du en Mac? Se [instruksjonene for macOS][9].

## <a id="step-2"></a>E-post

[!include[Hvorfor epost](includes/why-connect-email.md)]

Velg et av følgende alternativer:

* [WebTools][2] (standard).

* [SuperOffice Inbox][1] - hvis du ikke kan bruke WebTools. [Se denne videoen for å komme i gang][10].

* [!include[SuperOffice for Outlook](includes/why-superoffice-for-outlook.md)]

## <a id="step-3"></a>Dokument

Standard dokumentintegrasjon er **SuperOffice dokumentbibliotek** med [WebTools][2]. Hvis du valgte WebTools for e-post i forrige trinn, er du allerede ferdig.
