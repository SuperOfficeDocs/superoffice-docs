* [!include[SuperOffice for Outlook](includes/why-superoffice-for-outlook.md)]

## <a id="step-3"></a>Dokument

**SharePoint-dokumentbiblioteket** lar deg lagre alle dokumenter i SharePoint Online.

Du kan lagre et dokument i SuperOffice CRM og åpne det enten i SuperOffice CRM eller SharePoint. Dokumentet vil også bli lagret på riktig kundekort i SuperOffice CRM, slik at andre brukere kan se det. Du kan også dele det via en SharePoint-lenke med de som ikke bruker SuperOffice CRM.

> [!NOTE]
> Dette alternativet krever et Microsoft 365-abonnement og er bare tilgjengelig hvis administratoren din har [satt opp Microsoft SharePoint-dokumentbiblioteket][3]. Alternativt kan du bruke SuperOffice dokumentbibliotek med WebTools (eller Google Workspace).
