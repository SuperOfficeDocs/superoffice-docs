# Vise aktiviteter

[Aktivitetene][1] som vises, avhenger av hvilken av følgende avmerkingsbokser du valgte nederst på detaljkortet:

* Oppfølginger
* Dokumenter
* Utsendelser
* Andre (chatoppføringer og skjemainnsendinger)

Du kan vise alle aktiviteter eller én av typene.

> [!TIP]
> Du kan også [filtrere aktiviteter][3] etter dato og bruker.
