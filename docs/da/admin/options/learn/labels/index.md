# Etiketter

Ved hjælp af fanen **Etiketter** på skærmbilledet Indstillinger kan du ændre navnet på ethvert felt i SuperOffice. Du kan f.eks. beslutte, at alle **Firma**-felter skal kaldes *Organisation* i stedet.
