# Generelt – Ressource

Ressourcer vises i dialogboksen for opfølgning i SuperOffice CRM, og det er her, du kan tilføje ressourcer fra fanen **Deltagere**. Du kan f.eks. tilføje ressourcer som mødelokaler og projektorer.
