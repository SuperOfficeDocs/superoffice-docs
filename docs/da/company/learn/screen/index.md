# Detaljekortene i skærmbilledet Firma

Skærmbilledet Firma indeholder følgende detaljekort:

| Detaljekort | Beskrivelse |
|---|---|
| [Kontaktpersoner][1] | Det indeholder oplysninger om medarbejdere i det aktive firma. |
| [Relationer][2] | Viser eventuelle forbindelser mellem virksomheder og/eller personer. |
| [Aktiviteter][3] | Indeholder alle de aktiviteter, der er knyttet til det aktive firma. |
| [Projekter][4] | Viser alle projekter for den aktive virksomhed. |
| [Salg][5] | Indeholder oplysninger om eventuelle salg, som er knyttet til firmaet. |
| [Sager][6] | Viser SuperOffice Service-anmodninger for det aktive firma. |
| [ERP][7] | Du kan knytte et SuperOffice-firma til en kunde/leverandør i ERP-systemet. |
