# Fanen Dag

Fanen **Dag** viser én dag ad gangen med timetidsintervaller langs venstre kant.

![Skærmbilledet Dagbog, fanen Dag -screenshot][img1]

Hvis [tidszoner][1] er aktiveret, kan du skifte tidszone i nederste venstre hjørne.
