# Standardrapporter

I skærmbilledet **Rapporter** kan du også køre forskellige standardrapporttyper. Disse rapporter er sorteret efter type/kategori, og du henter dem frem ved at klikke på fanerne øverst i skærmbilledet Rapporter (se [Visning af rapportegenskaber][1]).

SuperOffice CRM leveres med flere forskellige rapport som standard, men en organisation kan også oprette sine egne rapporter i SuperOffice Reporter Studio, som er et valgfrit tillægsmodul. Når du bruger skærmbilledet Rapporter, vil du imidlertid ikke mærke nogen forskel på, hvor disse rapporter kommer fra, og derfor omtaler vi dem alle som standardrapporter her.

> [!TIP]
> Du kan gemme de standardrapporter, du bruger ofte, som [foretrukne rapporter][2] for at gøre dem lettere tilgængelige. Desuden kan du redigere standardrapporter og gemme dem som foretrukne.

## <a id="type" />Typer

| Type | Beskrivelse |
|---|---|
| Firma/person | Viser firma-/personrapporter. |
| Projekt | Viser projektrapporter. |
| Salg | Viser salgsrapporter. |
| Aktivitet | Viser rapporter til opfølgninger og dokumenter. |
| Udvalg | Viser udvalgsrapporter. |

## Redigering og sletning af standardrapporter

Du kan hverken redigere eller slette selve standardrapporterne. Men du kan dog [gemme en standardrapport som en foretrukken rapport][3] og derefter [justere søgekriterierne, som du ønsker][4].
