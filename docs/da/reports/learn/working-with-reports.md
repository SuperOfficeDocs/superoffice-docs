# Arbejde med rapporter

På skærmbilledet **Rapporter** kan du både definere søgekriterier for rapporter (se [Foretrukne rapporter][1]) og bruge [standardrapporter][2].

> [!NOTE]
> Du kan ikke tilføje helt nye søgekriterier til SuperOffice CRM. Søgekriterierne er defineret på forhånd for de forskellige rapporttyper, og du kan kun redigere deres værdier og variabler og gemme dem som foretrukne rapporter. Hvis du vil tilføje nye søgekriterier og oprette nye standardrapporter, skal du købe SuperOffice Reporter Studio.
