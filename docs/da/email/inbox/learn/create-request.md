# Opret sag

Med denne funktion kan du hurtigt tilføje en sag i SuperOffice Service baseret på afsenderen og tekstindholdet af e-mailmeddelelsen.

1. Gå til indbakken.
2. Vælg den ønskede meddelelse i indbakken.
3. Klik på <i class="ph ph-archive" aria-hidden="true"></i> **Arkiver**
4. Klik på **Opret sag**.
5. Udfyld de relevante felter i dialogboksen.
6. Klik på **Gem**.
    * Hvis afsenderens e-mailadresse genkendes, knyttes sagen automatisk til den rigtige person og det rigtige firma.
    * Hvis afsenderens e-mailadresse ikke er registreret, registreres afsenderen automatisk som en ny person i SuperOffice Service.
