# Knapper på toplinjen

**Toplinjen** indeholder vigtige værktøjer til navigation i SuperOffice CRM. Knapperne varierer en smule afhængigt af modulet (CRM, Service eller Indstillinger og vedligeholdelse), men deres funktioner forbliver de samme. For eksempel vises kun **Personlige indstillinger**, **Hovedmenu** og **Hjælp** i **Indstillinger og vedligeholdelse**.

## Oversigt over knapper

| Ikon | Navn | Beskrivelse |
|:-:|---|---|
| **Ny** | **Ny** | Opret nye poster, såsom firmaer, personer, salg, projekter, udvalg, opfølginger, dokumenter og sager. |
| <i class="ph ph-list-magnifying-glass" aria-label="Find ikon"></i> | [**Find**][5] | Søg efter poster, såsom firmaer, personer, salg og dokumenter (Ikke tilgængelig i Service og Indstillinger og vedligeholdelse). |
| <i class="ph ph-magnifying-glass" aria-label="Forstørrelsesglas"></i> | [**Søg**][10] | Brug **Fritekstsøgning** til at finde data i hele systemet. |
| <i class="ph ph-bell" aria-label="Klokke ikon"></i> | [**Notifikationer**][9] | Se nye notifikationer, såsom invitationer, tildelte sager og godkendte salg. Klik på en notifikation for at åbne den tilhørende post. |
| <i class="ph ph-user-circle" aria-label="Kontakt ikon"></i> | **Personlige indstillinger** | Rediger præferencer, e-mailsignatur og adgangskoder. Viser dit profilbillede, hvis det er tilføjet til dit personkort. |
| <i class="ph ph-list" aria-label="Hovedmenu"></i> | **Hovedmenu** | Få adgang til generelle systemindstillinger, skift mellem moduler eller åbn andre applikationer. |
| <i class="ph ph-lightbulb" aria-label="Pære ikon"></i> | **In-app Resource Center** | Åbn vejledninger, brugervejledning og inspirationsressourcer (kun online). |
| **Hjælp** | **Hjælp** | Få adgang til brugervejledninger og information tilpasset den aktuelle opgave eller det aktuelle modul. |

## Menuen Personlige indstillinger

| Mulighed | Beskrivelse |
|---|---|
| **Præferencer** | Juster [personlige præferencer][6], såsom standardværdier, sortering og visuelle effekter. |
| **Rediger personlig signatur** | Rediger den [e-mailsignatur][7], der bruges i udgående e-mails eller sager. |
| **E-mailindstillinger** | Konfigurer indstillinger for indbakke. |
| **Din Service-status** | Opdater din status (f.eks. Ikke til stede) og tilføj en valgfri besked. |
| **Vis mine kontaktoplysninger** | Åbn dit personkort for at redigere detaljer såsom telefonnumre eller titel. |
| **Lokale indstillinger** | Skift sprog, nummer-/datoformat og e-mailklientindstillinger. |
| **Få mobilappen** | Konfigurer indstillinger for SuperOffice Mobile CRM. |
| **Skift adgangskode** | Opdater din adgangskode til SuperOffice CRM. |
| **Papirkurv** | [Gendan slettede poster][3], såsom firmaer, personer eller sager. |
| **Log af** | Afslut din session i SuperOffice CRM. |

> [!NOTE]
> I Indstillinger og vedligeholdelse er kun **Lokale indstillinger**, **Skift adgangskode** og **Log af** tilgængelige.

## Valgmuligheder i hovedmenuen

### CRM

| Mulighed | Beskrivelse |
|---|---|
| **Andre programmer** | Få adgang til [eksterne værktøjer eller links][1] integreret i SuperOffice CRM. Svarer til **Værktøj** i navigatoren. |
| **Download** | Se og download værktøjer som SuperOffice WebTools. |
| **App Store** | Udforsk apps og integrationer i **SuperOffice App Store**. |
| **Kopier genvej** | [Opret genveje til poster][15]. |
| **Duplikatindstillinger** | Juster indstillinger for [håndtering af dubletter][8] i firmaer, projekter eller udvalg. |
| **Filter** | Rediger [filterindstillinger][2]. |
| **Vidensbase** | Få adgang til eksterne dokumenter, FAQ'er og svarskabeloner i Service. |
| **Brugerdefinerede objekter** | I pilot. |
| **Mål** | Indstil eller opdater [salgsmål][17]. |
| **Rediger systemsignatur** | Rediger standardsignaturen til e-mails. |
| **Indstillinger og vedligeholdelse** | Åbn **Indstillinger og vedligeholdelse** (kun for administratorer). |

### Indstillinger og vedligeholdelse

| Mulighed | Beskrivelse |
|---|---|
| **Rediger serienummer** | Opdater serienummeret for din SuperOffice-installation. |
| **SuperOffice CRM** | Skift tilbage til SuperOffice CRM. |

## Valgmuligheder i hjælpemenuen

| Mulighed | Beskrivelse |
|---|---|
| **In-app Resource Center** | Få adgang til vejledninger, brugervejledning og online ressourcer. |
| **Community** | Åbn SuperOffice Hjælpecenter og Community-site. |
| **Kontakt support** | Send supportforespørgsler eller fejlrapporter. |
| **Hjælpefiler (F1)** | Få adgang til brugervejledninger direkte. |
| **Om** | Se softwareoplysninger, herunder version, brugerinfo og kunde-ID. |
