# Sektionskortene i skærmbilledet Person

Skærmbilledet Person indeholder følgende sektionsfaner med oplysninger, der er knyttet til den aktuelle kontaktperson:

| Detaljekort | Beskrivelse |
|---|---|
| [Aktiviteter][1] | Indeholder alle de aktiviteter, der er knyttet til personen. |
| [Salg][2] | Indeholder oplysninger om eventuelle salg, som er knyttet til personen. |
| [Sager][3] | Viser SuperOffice Service-anmodninger for personen. |
| [Projekter][4] | Viser alle de projekter, hvor den aktive kontaktperson er projektmedlem. |
| [Relationer][5] | Viser eventuelle forbindelser, der findes mellem den aktuelle kontaktperson og andre personer eller firmaer. |
