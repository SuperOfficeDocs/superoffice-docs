# Het tabblad ERP

Wanneer uw organisatie een licentie heeft voor SuperOffice Integration Server, wordt het tabblad **ERP** op de schermen Persoon en Project weergegeven.

## Scherm Persoon

Als een bedrijf is verbonden met het ERP-systeem, kunnen alle personen die zijn gekoppeld aan dit bedrijf, ook worden verbonden met een overeenkomend individu in het ERP-systeem.

## Het scherm Project

Hier kunt een verbinding maken tussen projecten in SuperOffice en projecten in het ERP-systeem.

Als er al een verbinding is gemaakt, kunt u klikken op **Nu synchroniseren** om de gegevens tussen SuperOffice CRM en het ERP-systeem te synchroniseren.
