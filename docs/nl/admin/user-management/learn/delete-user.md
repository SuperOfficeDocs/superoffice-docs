# Medewerker verwijderen

1. [!include[Click Users](includes/goto-users.md)]

2. Selecteer het tabblad **Medewerkers**.

3. [!include[Select user](includes/select-user.md)]

4. Klik op de knop **Verwijderen**. U kunt gebruikers die zijn aangemeld, niet verwijderen.

5. Selecteer de gewenste optie in het dialoogvenster **Gebruiker verwijderen**. Klik op een optie om een beschrijving weer te geven van wat er gebeurt bij verwijdering.

    * Gebruikerslicenties verwijderen
    * Markeer medewerker als voormalige werknemer
    * Medewerker verwijderen \[login ID\]
    * Medewerker \[login ID\] en contactpersoon verwijderen\[full name\]

    > [!NOTE]
    > deze opties worden weergegeven in volgorde van belangrijkheid.

6. Klik op **OK**.
