# Algemene voorkeuren

De **Algemene voorkeuren** onder ![pictogram][img1] **Voorkeuren** wordt gebruikt om enkele van de belangrijkste voorkeuren in SuperOffice CRM te configureren.

Gebieden waarvoor u voorkeuren kunt configureren:

* Statistieken
* Functies
* E-mail
* Marketing
* Gegroepeerde lijsten
* Mobile CRM
* Verkoop
* [SuperOffice Service-systeeminstellingen][1]
* Systeem
* Targets
* [Gegevens spiegelen][5]
* [SMS-configuratie][4]
* [SMTP][2] (alleen LOKAAL)
* [Wachtwoordbeleid Customer Centre][3]
