# Items toevoegen aan de lijst Algemeen - Bron

[!include[Note difference](includes/different-edit-list-item-dialog.md)]

[!include[Edit list item](includes/edit-list-item.md)] wanneer de lijstitems bronnen zijn:

1. Typ de gewenste naam voor het item in het veld **Naam**.

    > [!NOTE]
    > [!include[Mandatory](includes/note-mandatory-field.md)]

2. Geef in het veld **Type** het brontype op:

    * **Locatie**: Bronnen van dit type (vergaderruimtes bijvoorbeeld) zijn beschikbaar in de lijst **Locatie** in de vervolgdialoogschermenÂ in SuperOffice CRM, naast de lijst **Bron** in het dialoogvenster **Uitnodiging**.

    * **Anders**: Bronnen van dit type (projectors, speakers, of andere apparatuur) zijn beschikbaar in de lijst **Bron** in het dialoogscherm **Uitnodiging**.

3. Als u **Locatie** selecteert als brontype, kunt u een adres invoeren in het veld **Adres**.

4. Typ zo nodig een beschrijving voor de bron in het veld **Beschrijving**.

5. [!include[Save or cancel](includes/save-or-cancel.md)]
