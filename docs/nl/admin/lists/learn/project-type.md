# Een projecttype toevoegen

[!include[Note difference](includes/different-edit-list-item-dialog.md)]

[!include[Edit list item](includes/edit-list-item.md)] wanneer de lijstitems projecttypen zijn:

1. Typ de naam voor het item in het veld **Naam**.

    > [!NOTE]
    > [!include[Mandatory](includes/note-mandatory-field.md)]

2. Voer in **Geschatte duur van dit projectproces** een duur in voor het project (aantal weken, dagen of maanden). Wanneer u een nieuw project van dit projecttype maakt, wordt de einddatum ingesteld in relatie tot de tijd die u hier invoert (de datum van vandaag + geschatte tijdsduur).

3. Markeer in de lijst **Statussen die worden gebruikt voor dit projecttype** de statussen die u voor het projecttype wilt opnemen. Deze stap is optioneel, maar om het projecttype aan een projectgids te koppelen, moet u ten minste twee statussen inschakelen.

    [Een projectstatus maken][2].

4. Schakel **Dit projecttype is gekoppeld aan een gids** in als u wilt dat alle projecten van dit projecttype een [projectgids][3] volgen.

5. U kunt ook **Voorstellen het project automatisch naar de volgende status te verplaatsen** selecteren. Wanneer alle vervolgactiviteiten en documenten voor een status zijn voltooid, wordt u gevraagd of u naar de volgende status wilt gaan.

6. Voer een beschrijving in van het projecttype.

7. Klik op **Opslaan**. Het projecttype wordt nu als een afzonderlijk item toegevoegd aan de lijst **Items**. Herhaal deze procedure om meer projecttypen toe te voegen.
