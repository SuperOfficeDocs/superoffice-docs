# Naam en beschrijving van een lijst bewerken

[!include[Find specific list](includes/tip-find-list.md)]

1. Open het scherm **Lijsten**.
2. [!include[Click arrow and select](includes/expand-list.md)] de vereiste lijst.
3. Klik op de knop **Lijst bewerken**.
4. Typ een nieuwe naam voor de lijst in het veld **Naam**.
5. Bewerk zo nodig de beschrijving van de lijst in het veld **Beschrijving**.
6. Klik op **OK** om de wijzigingen op te slaan in Instellingen en onderhoud.
