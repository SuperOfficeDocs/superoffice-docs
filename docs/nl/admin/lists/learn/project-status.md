# Project - Status toevoegen

[!include[Edit list item](includes/edit-list-item.md)] wanneer u een item toevoegt aan de lijst **Project - Status**:

1. Typ de naam voor het item in het veld **Naam**. Deze wordt weergegeven in de keuzelijst voor het veld **Status** in het scherm Project.

    > [!include[Mandatory](includes/note-mandatory-field.md)]

2. Typ zo nodig een beschrijving voor de koptekst in het veld **Beschrijving**.

3. [!include[Save or cancel](includes/save-or-cancel.md)]

> [!NOTE]
> wanneer u een status verwijdert waaraan een projectgids is gekoppeld (via een projecttype), kan dit gevolgen hebben voor het gebruik van de projectgids.
