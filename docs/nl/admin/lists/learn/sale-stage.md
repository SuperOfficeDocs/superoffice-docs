# Een verkoopfase toevoegen

[!include[Must be admin](../../../learn/includes/req-admin.md)]

[!include[Note difference](includes/different-edit-list-item-dialog.md)]

[!include[Edit list item](includes/edit-list-item.md)] wanneer de lijstitems verkoopfasen zijn:

1. Typ de gewenste naam voor het item in het veld **Naam**. Deze wordt weergegeven in de keuzelijst voor het veld **Fase** in het scherm Verkoop.

    [!include[Mandatory](includes/note-mandatory-field.md)]

2. Geef de kans op voor het item door een percentage te typen in het veld **Kans**. Dit getal wordt getoond naast de fase in het scherm Verkoop en geeft de kans op een succesvolle verkoop in deze fase aan.

3. Typ zo nodig een beschrijving voor de koptekst in het veld **Beschrijving**.

4. [!include[Save or cancel](includes/save-or-cancel.md)]

> [!NOTE]
> als u een fase verwijdert waaraan een verkoopgids is gekoppeld (via een verkooptype), kan dit gevolgen hebben voor het gebruik van de verkoopgids.
