# Labels

Via het tabblad **Labels** in het scherm Opties kunt u de naam van elk veld in SuperOffice wijzigen. U kunt bijvoorbeeld beslissen dat alle velden **Bedrijf** de naam *Organisatie* moeten krijgen.
