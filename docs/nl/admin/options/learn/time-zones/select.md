# Tijdzones bewerken

U kunt selecteren welke tijdzones u wilt weergeven in een lijst met alle beschikbare tijdzones.

[!include[Go to Options](../includes/open-options.md)]
2. Selecteer het tabblad **Tijdzones**. Dit bevat de namen van alle beschikbare tijdzones.
3. Selecteer de tijdzones die moeten worden weergegeven in SuperOffice CRM.

> [!TIP]
> Gebruik het veld **Filter** om te zoeken naar specifieke tijdzones.
