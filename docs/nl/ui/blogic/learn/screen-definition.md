# Schermdefinitie

[!include[Requirement](../../../includes/req-dev-tools.md)]

Hier kunt u de details van het geselecteerde scherm bekijken.

* **Scherm bewerken**: wordt gebruikt om het scherm te bewerken. Zie [Schermeigenschappen][1].

* **Taal bewerken**: wordt gebruikt om schermteksten voor de beschikbare talen te bewerken. Selecteer een taaltabblad en bewerk de beschikbare teksten.

* **Nieuw element**: wordt gebruikt om een nieuw element aan het scherm toe te voegen. Zie elementeigenschappen hieronder.

* **Nieuw script**: wordt gebruikt om een nieuw script toe te voegen. Hier kunt u een script invoeren dat wordt uitgevoerd wanneer de gebruiker op de knop klikt.

* **Nieuwe macro**: wordt gebruikt om [een nieuwe macro toe te voegen][5].

* **Scherm weergeven**: geeft het scherm weer.

* **Ontwerpmodus**: toont het scherm in de ontwerpmodus (zie hieronder).

* **Lijstmodus**: toont alle schermelementen in een lijst in plaats van in de ontwerpmodus.

## Script eigenschappen knop

* **Naam**: de naam van de knop.
* **Eerst veldwaarden controleren**: wanneer deze optie is ingeschakeld, wordt de inhoud van de formuliervelden gecontroleerd voordat het script wordt uitgevoerd. Wanneer een of meer waarden ongeldig zijn, wordt het betreffende formulier opnieuw weergegeven en worden de fouten gemarkeerd.
* **Script**: hier kunt u het script invoeren dat met deze knop wordt uitgevoerd.

## Elementeigenschappen

Hier kunt u de eigenschappen van een schermelement bewerken, inclusief een constructiescript voor het element en eenvoudige waarden die specifiek zijn voor de diverse elementtypen (zoals titel, naam). De hoofdtekst is een dergelijke eigenschap. Veel elementen hebben deze en aangezien ze vrij lang kunnen zijn, worden deze weergegeven op een eigen tabblad.

## Ontwerpmodus

In de ontwerpmodus kunt u een element maken door op de werkbalk aan de rechterkant te dubbelklikken op de gewenste optie. Elk element heeft een rechthoekige handgreep waarop u kunt klikken om een menu weer te geven. Via dit menu kunt u het element bijvoorbeeld verplaatsen of het type element toevoegen dat is geselecteerd in de werkbalk.

Veel van de elementen bevatten voorbeeldgegevens wanneer u ze invoegt, zodat u kunt zien hoe de elementen uiteindelijk in het scherm worden weergegeven.

 De volgende tabbladen worden weergegeven:

* **Elementen**: de elementen in het scherm.
* **Knoppen**: overzicht van de knoppen gekoppeld aan acties in het scherm.
