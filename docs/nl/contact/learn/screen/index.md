# Het scherm Persoon sectiekaarten

Het scherm Persoon bevat de volgende sectietabbladen met informatie die zijn gekoppeld aan de huidige persoon:

| Sectietabblad | Beschrijving |
|---|---|
| [Activiteiten][1] | Bevat alle activiteiten die aan de persoon zijn gekoppeld. |
| [Verkoop][2] | Bevat informatie over verkopen met betrekking tot de persoon. |
| [Verzoeken][3] | Toont SuperOffice Service-aanvragen voor de persoon. |
| [Projecten][4] | Toont alle projecten waarbij de actieve persoon een projectlid is. |
| [Relaties][5] | Hier worden eventuele verbindingen weergegeven tussen de huidige persoon en andere personen of bedrijven. |
