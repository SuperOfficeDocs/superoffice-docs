# Tabblad Projecten

Het sectietabblad **Projecten** geeft alle projecten weer waarvan de actieve persoon projectlid is.

Klik op **Toevoegen aan project** om projectleden toe te voegen aan een project.

Als u dubbelklikt op een project in de lijst, wordt het scherm Project geopend. Hierin wordt informatie over het actieve project weergegeven.
