# Een alarm instellen voor vervolgactiviteiten <i class="ph ph-bell" aria-hidden="true"></i>

SuperOffice CRM kan je herinneren voordat je vergadering begint. Op het ingestelde moment verschijnt een systeemmelding op je apparaat.

![SuperOffice-browsermelding -screenshot][img4]

Voor videogesprekken verschijnt een melding van de vergadersysteemprovider (zoals Microsoft Teams) bij aanvang van de vergadering. Klik op de knop **Deelnemen aan vergadering** om deel te nemen, of als je de host bent, klik op **Vergadering starten**.

## Stappen

[!include[Recurrence](includes/note-repetition.md)]

1. [Open of maak een vervolg][1].
1. Klik op het bel-icoon in de kop en selecteer een tijd.
1. Klik op **Opslaan**.

![Alarm vervolgkeuzelijst in de kop van het opvolgingsdialoogvenster -screenshot][img3]

> [!NOTE]
> Alarms zijn niet hetzelfde als [activiteitsmeldingen][2], die betrekking hebben op nieuwe of bijgewerkte vergaderuitnodigingen die worden weergegeven in het meldingenpaneel binnen SuperOffice.
