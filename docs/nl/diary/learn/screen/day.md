# Het tabblad Dag

Het tabblad **Dag** wordt één dag tegelijk weergegeven, met tijdsintervallen per uur langs de linkerrand.

![Scherm Agenda, het tabblad Dag -screenshot][img1]

Als [tijdzones][1] zijn ingeschakeld, kunt u van tijdzone wisselen in de linkerbenedenhoek.
