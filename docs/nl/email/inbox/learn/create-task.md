# Taak maken

Met deze functie kunt u snel een taak toevoegen in SuperOffice CRM op basis van de afzender en de inhoud van de tekst in het e-mailbericht.

1. Ga naar Postvak IN.
2. Selecteer het betreffende bericht in het Postvak-IN.
3. Klik op **Archiveren** ![pictogram][img1].
4. Klik op **Taak maken**. Het dialoogvenster **Taak** wordt geopend.

    Zie [het dialoogvenster voor vervolgactiviteiten][2].
