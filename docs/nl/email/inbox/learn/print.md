# Een e-mail afdrukken

1. Ga naar Postvak IN.
2. Selecteer het bericht dat u wilt afdrukken.
3. Klik op de knop **Acties** ![pictogram][img1] en selecteer **Geselecteerde e-mail afdrukken**. Het bericht wordt geopend in een voorbeeldvenster in de browser.
4. Klik op de afdrukoptie in het menu in het venster.
