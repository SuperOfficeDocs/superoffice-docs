Als SuperOffice CRM for Web is geïnstalleerd op een HTTPS-site, werken webpanelen met HTTP URL's niet. U dient HTTPS URL's te gebruiken.

U kunt de URL in veilige modus openen met uw browser, maar de site zal niet langer veilig zijn.

Als SuperOffice CRM for Web is geïnstalleerd op een HTTP-site, gebruik dan "normale" HTTP URL's.
