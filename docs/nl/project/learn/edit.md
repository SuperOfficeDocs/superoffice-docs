# Projecten bewerken

U kunt informatie over projecten bewerken via de tabbladen in de projectkaart.

1. [!include[Go to project](includes/goto-project.md)]

2. Klik op **Bewerken**.

    De tabbladen in het scherm Project schakelen over naar de bewerkingsmodus. U kunt tussen de tabbladen schakelen zonder de bewerkingsmodus af te sluiten.

3. Wanneer u klaar bent, klikt u op **Opslaan** om te sluiten en op te slaan. Klik op **Annuleren** als u de wijzigingen niet wilt opslaan.
