# Een project maken

Bekijk deze video of volg de onderstaande stappen om te leren hoe u een project maakt (duur van de video - 2:49):

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/zLJwNll223w]

## Stappen

1. Klik op **Nieuw** op de bovenste balk en selecteer **Project**.

2. Typ op de lege projectkaart de gewenste informatie in de velden op het tabblad **Project**. Zie details hieronder.

3. Ga eventueel naar een van de tabbladen om meer informatie in te voeren.

4. Klik op **Opslaan**. U kunt nu [projectleden toevoegen][5].
