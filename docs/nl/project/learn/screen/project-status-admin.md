# Project - Status

Het tabblad **Project** in het scherm Project bevat het veld **Status**. Dit veld heeft een vooraf gedefinieerde optielijst, project-status, die is gedefinieerd in het scherm Lijsten in Instellingen en onderhoud.

De weergegeven statussen in de lijst worden bepaald door het [projecttype][1]. U kunt selecteren welke statussen beschikbaar moeten zijn voor de verschillende projecttypen in de statuslijst Project - Type.
