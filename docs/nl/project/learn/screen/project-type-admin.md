# Project - Type, status

Het tabblad **Project** in het scherm Project bevat het veld **Type**. Dit veld heeft een vooraf gedefinieerde optielijst die is gedefinieerd in het scherm Lijsten in Instellingen en onderhoud.

Elk project in SuperOffice CRM moet een projecttype hebben, en elk projecttype kan verschillende [statussen][1] bevatten. U kunt ook opgeven dat specifieke projecttypen een [projectgids][3] moeten hebben om u door de verschillende statussen van een project, inclusief bijbehorende vervolgactiviteiten en documenten, te begeleiden.

## Wat wilt u nu doen?

* [Projecttype toevoegen][2]
