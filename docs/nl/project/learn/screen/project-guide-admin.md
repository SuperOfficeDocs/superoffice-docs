# Projectgids

In het tabblad **Projectgids** in het scherm Workflow in Instellingen en onderhoud kunt u projectgidsen opstellen die gelden als leidraad voor het project. Een projectgids bestaat uit statussen met vervolgactiviteiten en documenten om u door het project te begeleiden. Elke projectgids is gekoppeld aan een projecttype en de projectgids wordt ingeschakeld in het scherm Project in wanneer de gebruiker een nieuw project maakt of het projecttype van een bestaand project bewerkt en een projecttype selecteert waaraan een projectgids is gekoppeld.

## Voorbeeld

Het projecttype 'Vergadering' kan uit drie statussen bestaan: Gepland, Wordt uitgevoerd en Sluiten. De status Gepland kan bijvoorbeeld de vervolgactiviteit "Planningsvergadering" en het document Vergaderprogramma (een opmerking) bevatten.
