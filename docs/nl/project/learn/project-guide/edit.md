# Een projectgids bewerken

[!include[Must be admin](../../../learn/includes/req-admin.md)]

1. Klik op de ![pictogram][img1] knop **Workflow** in de navigator.
2. Selecteer het tabblad **Projectgids**.
3. Selecteer het gewenste projecttype in de keuzelijst bovenin.
4. [Informatie over het projecttype en de statussen ervan bewerken][3].
5. [Bewerk de afzonderlijke statussen][3].
6. Bewerk vervolgactiviteiten en documenten door er in de lijsten onderin op te dubbelklikken.

    * [Een vervolgactiviteit toevoegen voor een status][2]
    * [Een document toevoegen voor een status][1]

    De wijzigingen worden automatisch opgeslagen.
