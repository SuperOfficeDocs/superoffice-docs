# Projectgids deactiveren

[!include[Must be admin](../../../learn/includes/req-admin.md)]

U kunt een projectgids deactiveren om deze uit een projecttype te verwijderen. U doet dit door het selectievakje **Dit projecttype is gekoppeld aan een gids** voor het betreffende projecttype uit te schakelen (zie [Items toevoegen aan de lijst Project - Type, status][1]).

Als u vervolgens de projectgids opnieuw wilt gebruiken, kunt u deze optie opnieuw inschakelen. Vervolgactiviteiten en documenten die eerder zijn toegevoegd aan de statussen in de projectgids worden automatisch opgehaald. U kunt ook nieuwe vervolgactiviteiten en/of documenten toevoegen of verwijderen wanneer u deze niet langer in de projectgids wilt gebruiken.
