# Documentmailing aan projectleden maken

Als u een document wilt verzenden naar alle leden van een project, kunt u de met de functie **Marketing** uitvoeren in SuperOffice. Voer een van de volgende handelingen uit:

* Selecteer <i class="ph ph-subtract-square" aria-hidden="true"></i> **Selecties** in de navigator, klik op  <i class="ph ph-dots-three-circle-vertical" aria-label="Task menu"></i> en selecteer **Documentmailing aanmaken**.

* Selecteer <i class="ph ph-target" aria-hidden="true"></i> **Marketing** in de navigator, selecteer het tabblad **Mailings** en [maak een nieuwe documentmailing][1].

> [!NOTE]
> In **Stap 4: Ontvangers** moet u het project selecteren om de leden aan het project toe te voegen als ontvangers.
