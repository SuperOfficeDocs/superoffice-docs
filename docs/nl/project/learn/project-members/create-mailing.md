# Documentmailing aan projectleden maken

Als u een document wilt verzenden naar alle leden van een project, kunt u de met het ![pictogram][img1] de functie **Marketing** uitvoeren in SuperOffice. Voer een van de volgende handelingen uit:

* Selecteer het ![pictogram][img2] **Selecties** in de navigator, ga naar het tabblad **Taak** en klik op **Documentmailing aanmaken**.

* Selecteer het ![pictogram][img1] **Marketing** in de navigator, selecteer het tabblad **Mailings** en [maak een nieuwe documentmailing][1].

> [!NOTE]
> In **Stap 4: Ontvangers** moet u het project selecteren om de leden aan het project toe te voegen als ontvangers.
