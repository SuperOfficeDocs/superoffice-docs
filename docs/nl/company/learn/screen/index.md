# De sectietabbladen in het scherm Bedrijf

Het scherm Bedrijf bevat de volgende sectietabbladen:

| Sectietabblad | Beschrijving |
|---|---|
| [Contactpersonen][1] | Bevat informatie over de werknemers van het actieve bedrijf. |
| [Relaties][2] | Toont eventuele bestaande verbindingen tussen bedrijven en/of contactpersonen. |
| [Activiteiten][3] | Bevat alle activiteiten die verband houden met het actieve bedrijf. |
| [Projecten][4] | Toont alle projecten voor het actieve bedrijf. |
| [Verkoop][5] | Bevat informatie over verkopen met betrekking tot het bedrijf. |
| [Verzoeken][6] | Toont SuperOffice Service-verzoeken voor het actieve bedrijf. |
| [ERP][7] | Hier kunt u een SuperOffice-bedrijf koppelen aan een klant/leverancier in het ERP-systeem. |
