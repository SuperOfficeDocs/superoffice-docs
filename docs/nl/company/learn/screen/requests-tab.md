# Tabblad Verzoeken

Het sectietabblad **Verzoeken** geeft verzoeken van SuperOffice Service weer voor het actieve bedrijf.

![Sectie Verzoeken in het bedrijf -screenshot][img1]

U kunt dubbelklikken op een verzoek om het in een apart venster te openen. Klik op de knop **Bewerken** om in SuperOffice Service aan het verzoek te werken.
