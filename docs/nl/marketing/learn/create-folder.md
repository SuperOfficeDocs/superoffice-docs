# Een map maken

Het is raadzaam een goede mapstructuur te definiëren voor een beter overzicht van uw mailings, formulieren, flows, sjablonen, koppelingen en afbeeldingen in SuperOffice Marketing.

## Stappen

1. Selecteer een van de hoofdtabbladen in SuperOffice Marketing.

2. Klik op de knop **Map toevoegen**.

3. Voer in het veld **Naam** een beschrijvende naam in voor de map.

4. Klik op **OK**. Als de nieuwe map niet in de lijst verschijnt, selecteer dan het tabblad **Mijn ...** of **Alle ...**.

## Submappen

Je kunt in alle mappen submappen maken, en je kunt het juiste marketingmateriaal in elk van deze mappen plaatsen.

Doe een van de volgende dingen:

* Open een bestaande map voordat je op **Map toevoegen** klikt. Mogelijk moet je eerst het tabblad **Alle ...** selecteren.

* Klik eerst op **Map toevoegen** en selecteer vervolgens een bovenliggende map uit de lijst **Maak als submap van**.
