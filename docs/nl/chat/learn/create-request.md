# Een verzoek maken in een actieve chat

Als u het probleem niet kunt oplossen via de chat, kunt u een verzoek maken dat door een andere specialist moet worden behandeld.

1. Klik op <i class="ph ph-list" aria-label="Task menu"></i> en selecteer **Verzoek maken**.
2. Vul in het dialoogvenster de relevante velden in.
3. Het chatlogboek wordt aan het verzoek toegevoegd, maar u kunt hier indien noodzakelijk meer gegevens toevoegen.
4. Klik op **Opslaan**.
5. Een kort bericht met de verzoek-ID wordt toegevoegd aan het tekstveld in de chat. Klik op **Verzenden** om dit naar de klant te verzenden.
