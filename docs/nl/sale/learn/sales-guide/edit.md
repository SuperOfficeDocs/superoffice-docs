# Een verkoopgids bewerken

[!include[Must be admin](../../../learn/includes/req-admin.md)]

1. Klik op de knop **Workflow** in de navigator (![pictogram][img1]).
2. Selecteer het tabblad **Verkoopgids**.
3. Selecteer het gewenste verkooptype in de keuzelijst **Verkooptype**.
4. Klik op **Bewerken**.
5. [Bewerk de afzonderlijke fasen][3].
6. Bewerk vervolgactiviteiten en documenten door er in de lijsten onderin op te dubbelklikken.

    * [Een vervolgactiviteit toevoegen voor een fase][1]
    * [Een document toevoegen voor een fase][2]

    De wijzigingen worden automatisch opgeslagen.
