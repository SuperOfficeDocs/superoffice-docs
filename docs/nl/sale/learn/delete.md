# Een verkoop verwijderen

Mogelijk wilt u een verkoop verwijderen als deze niet langer in de database hoeft te worden opgeslagen.

1. Selecteer de betreffende verkoop.
2. Klik op de knop **Taak** (![pictogram][img1]).
3. Selecteer **'Verkoopnaam' verwijderen**.
4. Klik op **Ja** om het verwijderen van de verkoop te bevestigen.
