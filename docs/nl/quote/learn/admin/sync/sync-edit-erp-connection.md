# ERP-verbinding bewerken (Sync Connector)

1. Ga naar het tabblad **Synchroniseren** op het scherm Offerte/Synchroniseren.
2. Schakel het selectievakje in de kolom **Actief** uit voor de ERP-verbinding die u wilt bewerken.
3. Dubbelklik op de ERP-verbinding.
4. In het dialoogvenster **ERP-verbinding bewerken** kunt u [de ERP-verbinding bewerken][1].
