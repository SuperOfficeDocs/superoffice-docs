# Een Sync Connector toevoegen

1. Ga naar het tabblad **Synchroniseren** op het scherm Offerte/Synchroniseren.
2. Klik op **Toevoegen** onder aan de lijst **Sync Connectors**.
3. Typ in het dialoogvenster de naam en URL van de Sync Connector.
4. Klik op **OK**.

    De Sync Connector wordt weergegeven in de lijst.
