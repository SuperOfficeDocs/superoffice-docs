# Synchroniseren

Het tabblad **Sync** op het scherm Offerte/Synchroniseren geeft Sync Connectors en gekoppelde ERP-verbindingen weer. Sync Connectors regelen de communicatie tussen SuperOffice en het ERP-systeem, terwijl de ERP-verbindingen de koppeling verzorgen tussen SuperOffice en Sync Connectors.
