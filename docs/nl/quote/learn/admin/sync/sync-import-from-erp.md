# Importeren vanuit ERP

U kunt bedrijven, personen en producten importeren vanaf een ERP-verbinding.

1. Ga naar het tabblad **Synchroniseren** op het scherm Offerte/Synchroniseren.
2. Selecteer **Uit** om automatische synchronisatie uit te schakelen.
3. Selecteer de ERP-verbinding en klik op de knop **Importeren vanuit ERP**.
4. Het scherm **Import** wordt geopend.[!include[Select source](../../../../admin/import/learn/includes/select-source.md)]
5. Zie [Importeren vanuit ERP][1] voor meer informatie.
