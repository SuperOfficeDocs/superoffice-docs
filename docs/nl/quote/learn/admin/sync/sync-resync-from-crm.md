# Opnieuw synchr. vanuit CRM

Deze functie wordt gebruikt om problemen op te lossen, met name als u weet dat de gegevens in SuperOffice CRM van betere kwaliteit zijn dan de ERP-gegevens. De functie werkt als een normale synchronisatie, maar de timestamps worden genegeerd en de synchronisatie wordt slechts in één richting (CRM naar ERP) uitgevoerd.

[!include[Back up database before import](../../../../admin/import/learn/includes/caution-backup-before-import.md)]

1. Ga naar het tabblad **Synchroniseren** op het scherm Offerte/Synchroniseren.
2. Klik op **Opnieuw synchroniseren vanuit CRM** onder aan de lijst **ERP-verbindingen**.
3. Klik ter bevestiging op **Ja**.
