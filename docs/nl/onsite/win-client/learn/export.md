# Exporteren

U exporteert gegevens uit de SuperOffice-database via het scherm Selectie of Rapporten in SuperOffice CRM.

Om een bestand te exporteren om het vervolgens te importeren via het scherm **Import** in Windows client-instellingen, moet u het exporteren als een [ANSI-bestand][1]. De importstappen voor ANSI-bestanden zijn hetzelfde als bij [het importeren van bestanden][2].
