# Import

[!include[Back up database before import](../../../../admin/import/learn/includes/caution-backup-before-import.md)]

Via het scherm **Import** kunt u belangrijke gegevens invoeren in de SuperOffice-database. U kunt bijvoorbeeld een klantenindex importeren zodat u deze kunt gebruiken in SuperOffice CRM. In de standaardversie van Windows client-instellingen kunt u de volgende gegevens importeren in de SuperOffice-database:

* Bedrijven met gekoppelde, door gebruikers gedefinieerde velden en interesses
* Contactpersonen met gekoppelde, door gebruikers gedefinieerde velden en interesses
* Projecten met gekoppelde, door gebruikers gedefinieerde velden
* Vervolgactiviteiten
* Verkoop
* Producten
* Valutaâ€™s
* Postcodes
* Datums voor herinneringen en aanmaningen

U kunt ook [bedrijven/personen en producten importeren in Instellingen en onderhoud][9].

Zie **Bestanden importeren** voor meer informatie over de velden in het scherm [Import][2] en de manier waarop u deze gebruikt. Lees voordat u begint met importeren ook [Importroutines][1]
