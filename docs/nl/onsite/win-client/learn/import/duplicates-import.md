# Duplicaten

Er is sprake van een duplicaat als dezelfde record zowel in het importbestand als in de database voorkomt. U moet in een [importbeschrijving][1] altijd regels opgeven voor de manier waarop duplicaten worden verwerkt tijdens de import. Wanneer u de importbeschrijving opslaat, worden deze regels hierin vervolgens opgenomen. Dit geldt voor alle gegevenstypen. In **Verwerking van duplicaten** vindt u de volgende kolommen:

* **Tabel**: geeft aan voor welk gegevenstype deze regel voor verwerking van duplicaten geldt. De inhoud in deze kolom is gebaseerd op de informatie die u hebt opgegeven in de kolom **Veld in SuperOffice CRM** onder **Veldtoewijzing**.
* **Sleutel**: vermeldt een importsleutel, de criteria waaraan een record moet voldoen om als duplicaat te worden aangemerkt.
* **Actie**: laat zien wat er gebeurt als er een duplicaat is.
