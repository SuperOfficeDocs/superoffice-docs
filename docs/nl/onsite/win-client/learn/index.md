# Windows Client-instellingen

Windows client-instellingen is de Windows-gebaseerde beheerclient voor SuperOffice CRM.

Het merendeel van de beheersfuncties is beschikbaar in [Instellingen en onderhoud][3].

## Aanmelden

U kunt Windows Client-instellingen openen vanuit het menu **Start** in Windows of rechtstreeks vanuit Windows Verkenner.

Hiervoor dient u een rol op **gebruikersniveau 0** te hebben, of een andere rol waarmee u aanmeldrechten hebt voor Windows Client-instellingen. Gebruikers-ID's, wachtwoorden en gebruikersniveaus worden gedefinieerd via het scherm Gebruikers in Instellingen en onderhoud

U moet zich ook bij Windows Client aanmelden als u al vanaf dezelfde pc bent aangemeld bij SuperOffice CRM.

## De hoofdsecties

* [Het scherm Status][1]
* [Import][2]

## Knoppen

Door slechts eenmaal te klikken op de knoppen in de Windows client-instellingen, start u de bijbehorende functies. Zo kunt u snel navigeren tussen de verschillende onderdelen van het programma.

## De navigator

De linker navigator bevat de volgende knoppen:

| Knop | Functie | Lees meer |
|:-:|---|---|
| ![pictogram][img1] | **Status**: hiermee opent u het scherm **Status**, waarin bijgewerkte informatie over het systeem wordt weergegeven. | [Scherm Status][1] |
| ![pictogram][img2] | **Import**: hiermee opent u het scherm **Import**, waarin u gegevens kunt importeren in de SuperOffice-database. | [Scherm Import][2] |
