# Informatie in de database bijwerken

Op een gegeven moment is het nodig om de database bij te werken. Sommige organisaties werken bijvoorbeeld de klantenindex eenmaal per maand bij, of wanneer dat nodig is.

Met de [importfunctie][1] in Windows Client-instellingen kunt u nieuwe updates uitvoeren, bijvoorbeeld wanneer u nieuwe informatie in een veld wilt invoeren voor alle bedrijven in de database.
