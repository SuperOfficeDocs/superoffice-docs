# Gebruikers afmelden (onsite)

Als een gebruiker geen toegang krijgt tot SuperOffice CRM, bijvoorbeeld na een programmacrash, kan de beheerder de gebruiker in Instellingen en onderhoud afmelden, zodat de gebruiker zich weer op de gebruikelijke manier kan aanmelden.

1. [!include[Click Users](../../../admin/user-management/learn/includes/goto-users.md)]

2. Selecteer het tabblad **Medewerkers**.

3. Selecteer het gewenste bedrijf in de keuzelijst **Bedrijf**.

4. Selecteer **Aangemeld bij Windows-client** in de keuzelijst onder de lijst met gebruikers.

5. [!include[Select user](../../../admin/user-management/learn/includes/select-user.md)]

6. Klik op de knop **Taak** en selecteer **Gebruiker afmelden**. De gebruiker wordt afgemeld.

> [!NOTE]
> Als de gebruiker niet is aangemeld of er iets anders dan **Aangemeld bij Windows-client** in de keuzelijst **Weergeven** is geselecteerd, is deze knop uitgeschakeld.
