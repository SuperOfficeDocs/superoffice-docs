Dit is de standaardmanier om je e-mailaccount, documenten en agenda te verbinden met SuperOffice.

> [!NOTE]
> Deze instructies zijn alleen voor Windows. Gebruik je een Mac? Zie de [instructies voor macOS][9].

## <a id="step-2"></a>E-mail

[!include[Waarom e-mail](includes/why-connect-email.md)]

Kies een van de volgende opties:

* [WebTools][2] (standaard).

* [SuperOffice Inbox][1] - als je geen WebTools kunt gebruiken.

* [!include[SuperOffice voor Outlook](includes/why-superoffice-for-outlook.md)]

De standaard documentintegratie is de **SuperOffice documentbibliotheek** met [WebTools][2]. Als je in de vorige stap WebTools voor e-mail hebt gekozen, ben je al klaar.
