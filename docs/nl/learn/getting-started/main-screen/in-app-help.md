# In-app help gebruiken

## De Help-bestanden gebruiken

Deze Help is ontworpen om u uitgebreide en gemakkelijk toegankelijke informatie te geven over het gebruik van SuperOffice CRM. Waar u zich ook bevindt in het programma en welke functie u ook uitvoert, u kunt altijd Help opvragen over de taak waarmee u bezig bent.

U kunt de Help op verschillende manieren openen:

* Selecteer **Help-bestanden (F1)** in het menu **Help**.
* Druk op **F1**. Het Help-onderwerp voor het huidige scherm wordt weergegeven.
* Klik op de knop **Help** die wordt weergegeven in de afzonderlijke vensters.
