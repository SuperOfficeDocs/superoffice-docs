U maakt een nieuwe map als volgt:

1. Ga naar de map waaronder u een submap wilt maken.
2. Klik op de knop **Nieuwe map** boven in het scherm.
3. In het veld **Map** wordt automatisch de naam weergegeven van de gekozen bovenliggende map. U kunt hier een andere map selecteren.
4. Voer in het veld **Naam** een beschrijvende naam in voor de map.
5. Voer in het veld **Beschrijving** een korte beschrijving in van de inhoud van de map.
6. Klik op **OK**. De nieuwe map wordt gemaakt.
