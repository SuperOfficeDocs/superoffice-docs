De kolommen op het sectietabblad **Project** bevatten de volgende gegevens:

* Vinkje dat aangeeft of het project is voltooid.
* De naam van het project.
* Het type project.
* De status van het project.
* Het projectnummer
* De projectbeschrijving (indien voorhanden). Houd de muisaanwijzer boven het ![pictogram][img1] paperclippictogram om de beschrijving weer te geven.
* Een eventuele koppeling met een webpagina waaraan het project is gekoppeld (een webadres (URL) waarop u kunt klikken om naar de betreffende webpagina te gaan).
* De gebruikers-ID van de persoon die het project heeft gemaakt.
