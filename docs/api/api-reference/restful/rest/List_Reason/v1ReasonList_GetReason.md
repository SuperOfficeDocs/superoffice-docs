# GET List/Reason/Items/{id}

```http
GET /api/v1/List/Reason/Items/{id}
```

Gets a Reason  item by id.






| Path Part | Type | Description |
|-----------|------|-------------|
| id | int32 | The id of the Reason to return. **Required** |
