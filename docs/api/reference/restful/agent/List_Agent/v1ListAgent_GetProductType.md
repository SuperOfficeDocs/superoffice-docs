# POST Agents/List/GetProductType

```http
POST /api/v1/Agents/List/GetProductType
```

Gets a ProductType object.

## Query String Parameters

| Parameter Name | Type |  Description |
|----------------|------|--------------|
| productTypeId | int32 | **Required** The primary key. |
| $select | string |  Optional comma separated list of properties to include in the result. Other fields are then nulled out to reduce payload size: "Name,department,category". Default = show all fields. |

```http
POST /api/v1/Agents/List/GetProductType?productTypeId=533
POST /api/v1/Agents/List/GetProductType?$select=name,department,category/id
```

## Response: object

The built-in product type list. Used by the SuperOffice Quote Connector to categorize products.

Carrier object for ProductType.
Services for the ProductType Carrier is available from the <see cref="T:SuperOffice.CRM.Services.IListAgent">List Agent</see>.
