# "SimpleInvitation"

This provider name is implemented by the class <see cref="T:SuperOffice.CRM.ArchiveLists.SimpleInvitationProvider">SuperOffice.CRM.ArchiveLists.SimpleInvitationProvider</see> inside NetServer's SODatabase assembly.

This is the query provider for invitations, of subtype 'Simple'

A simple invitation is one that is not recurring and has the status <see cref="F:SuperOffice.Data.AppointmentStatus.Booking" />.
This entity is mandatory. Simple bookings have no 'overdue' checks.
