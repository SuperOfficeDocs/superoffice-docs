# SuperOffice Web Tools

SuperOffice Web Tools bridges document editing in SuperOffice CRM Web with the associated applications on the user's desktop.

SuperOffice Web Tools has 4 features:

* Download, edit and upload documents
* Alarm notifications
* Quick-login to SuperOffice by menu or double-clicking on the icon in the notification area
* Bridge between user-initiated actions in SuperOffice CRM Web and SuperOffice Mail Link

How it works is detailed in the [technical overview][2].

## Distribution

* SuperOffice Web will automatically detect new versions of Web Tools and will offer the user to update. Web Tools can also be downloaded manually from the Download menu inside SuperOffice.

* SuperOffice Windows users will receive new versions upon upgrading the Windows client or by manually downloading and installing from theÂ [Download Service][1].
