Blank when not a video meeting. Filled with Join Meeting URL when created.

**Parameter:** 
 - **joinVideomeetUrl** String

```crmscript
NSAppointmentEntity thing;
String joinVideomeetUrl;
thing.SetJoinVideomeetUrl(joinVideomeetUrl);
```
