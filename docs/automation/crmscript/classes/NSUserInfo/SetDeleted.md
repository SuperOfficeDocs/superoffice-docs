intellisense: NSUserInfo.SetDeleted
keywords: NSUserInfo, GetDeleted
so.topic: reference
---

If true, the user is retired and should have no rights, not appear in lists, etc.

**Parameter:** 
 - **deleted** Bool

```crmscript
NSUserInfo thing;
Bool deleted;
thing.SetDeleted(deleted);
```
