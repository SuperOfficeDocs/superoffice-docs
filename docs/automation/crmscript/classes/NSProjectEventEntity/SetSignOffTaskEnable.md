If not 0, a task should be created when the external user Signs Off an event

**Parameter:** 
 - **signOffTaskEnable** Bool

```crmscript
NSProjectEventEntity thing;
Bool signOffTaskEnable;
thing.SetSignOffTaskEnable(signOffTaskEnable);
```
