Status if this appointment is in the process of being assigned to someone else

**Parameter:** 
 - **assignmentStatus** Integer
     - Enum: 0 = Unknown 
     - Enum: 1 = None 
     - Enum: 11 = Assigning 
     - Enum: 12 = Seen 
     - Enum: 13 = Declined
