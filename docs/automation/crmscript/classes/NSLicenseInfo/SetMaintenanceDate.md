Expiration date for maintenance. After this date, upgrades should refuse to install.

**Parameter:** 
 - **maintenanceDate** DateTime

```crmscript
NSLicenseInfo thing;
DateTime maintenanceDate;
thing.SetMaintenanceDate(maintenanceDate);
```
