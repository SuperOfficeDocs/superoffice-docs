Lowest allowed setting level, set by user, but not lower than sysMaxLevel

**Parameter:** 
 - **maxLevel** PreferenceLevel
     - Enum: 0 = Undefined 
     - Enum: 1 = HardDefault 
     - Enum: 2 = SystemWide 
     - Enum: 3 = Database 
     - Enum: 4 = Group 
     - Enum: 5 = Individual 
     - Enum: 6 = PC
