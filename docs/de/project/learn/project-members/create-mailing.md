# Dokumentenkampagne an Projektmitglieder erstellen

Wenn Sie ein Dokument an alle Mitglieder eines Projekts senden möchten, können Sie dazu die Funktion ![Symbol][img1] **Marketing** in SuperOffice verwenden. Führen Sie einen der folgenden Schritte aus:

* Wählen Sie das ![Symbol][img2] **Selektionen** im Navigator aus, gehen Sie zur Registerkarte **Aufgabe** und klicken Sie auf **Dokumentenkampagne erstellen**.

* Wählen Sie das ![Symbol][img1] **Marketing** im Navigator, die Registerkarte [Kampagnen][1] und die Option **Neue Dokumentkampagne erstellen** aus.

> [!NOTE]
> In **Schritt 4: Empfänger** Sie müssen das Projekt auswählen, um die Projektmitglieder als Empfänger hinzuzufügen.
