# Projekt - Typ, Status

Die Registerkarte **Projekt** im Projekt enthält das Feld **Typ**. Dieses Feld verfügt über eine vordefinierte Optionsliste, die in der Ansicht Listen in Einstellungen und Verwaltung festgelegt wird.

Jedes Projekt in SuperOffice CRM muss einen Projekttyp aufweisen und jeder Projekttyp kann verschiedene [Status][1] enthalten. Sie können auch festlegen, dass für bestimmte Projekttypen ein [Projektleitfaden][3] zu verwenden ist, der Sie durch die verschiedenen Projektstatus führt und entsprechende Folgeaufgaben und Dokumente vorschreibt.
