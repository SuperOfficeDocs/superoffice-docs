# Projekt erstellen

Sehen Sie sich dieses Video an oder befolgen Sie die folgenden Schritte, um zu lernen, wie Sie ein Projekt erstellen (Videolänge – 2:49):

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/zLJwNll223w]

## Schritte

1. Klicken Sie auf **Neu** auf der oberen Leiste und wählen Sie **Projekt**.

2. Geben Sie in der leeren Projektkarte die erforderlichen Information in die Felder auf der Registerkarte **Projekt** ein. Für weitere Details siehe unten.

3. Gehen Sie anderenfalls in eine der Registerkarten, um weitere Informationen einzugeben.

4. Klicken Sie auf **Speichern**. Als Nächstes könnten Sie zum Beispiel [Projektmitglieder hinzufügen][5].
