# Standardberichte

In der Ansicht **Berichte** können Sie verschiedene Arten von Standardberichten ausführen. Diese Berichte sind nach Typ/Kategorie geordnet. Sie können sie anzeigen, indem Sie auf die Registerkarten oben in der Ansicht Berichte klicken (siehe [Berichtseigenschaften anzeigen][1]).

SuperOffice CRM beinhaltet bereits einige vordefinierte Standardberichte, aber eine Organisation kann auch eigene Berichte in im optionalen Add-on-Modul SuperOffice Reporter Studio erstellen. Wenn Sie die Ansicht Berichte verwenden, werden Sie jedoch keinen Unterschied bezüglich der Herkunft der Berichte feststellen. Aus diesem Grund behandeln wir alle Berichte als Standardberichte.

> [!TIP]
> Sie können häufig verwendete Standardberichte als [Favoriten][2] speichern, um schneller darauf zugreifen zu können. Darüber hinaus können Sie Standardberichte bearbeiten und ebenfalls als Favoriten speichern.

## <a id="type"></a>Typen

| Typ | Beschreibung |
|---|---|
| Firma/Person | Zeigt Firmen-/Personenberichte an. |
| Projekt | Zeigt Projektberichte an. |
| Verkauf | Zeigt Verkaufsberichte an. |
| Aktivität | Zeigt Berichte für Folgeaufgaben und Dokumente an. |
| Selektion | Zeigt Selektionsberichte an. |

## Standardberichte bearbeiten und löschen

Sie können Standardberichte selbst weder bearbeiten noch löschen. Sie können jedoch einen [Standardbericht als Favorit speichern][3] und anschließend die [Suchkriterien nach Bedarf anpassen][4].
