# Bericht auszuführen

Für das Erstellen von Berichten anhand der Daten aus SuperOffice CRM können Sie eigene Berichte oder die veröffentlichten Berichte aus der Ansicht **Berichte** verwenden.

> [!NOTE]
> Die Berichtserstellung basiert auf denselben Zugriffsrechten wie SuperOffice CRM. Das bedeutet, dass Informationen in den von Ihnen erstellten Berichten nicht angezeigt werden, wenn Sie für diese in SuperOffice CRM keine Berechtigungen besitzen.

Wenn Sie einen Bericht in der Ansicht **Berichte** ausführen möchten, können Sie ihn auf dem Bildschirm anzeigen, auf einem Drucker ausgeben oder als Datei speichern. In den beiden letzteren Fällen ist es ratsam, den Bericht zunächst auf dem Bildschirm anzuzeigen, um sicherzustellen, dass er so aussieht wie von Ihnen gewünscht.

Damit Sie in SuperOffice CRM einen [Bericht als Dokument speichern][1] können, müssen Sie auf die Schaltfläche **Speichern** in der Anzeige **Berichte** klicken.
