# Mit Formularvorlagen arbeiten

## Ordner zur besseren Organisation verwenden

[Wie man einen Kampagnenordner erstellt][19]

## Verwenden von Formularvorlagen

1. Wählen Sie die Registerkarte **Formularvorlagen**.
2. Klicken Sie auf **Alle Vorlagen**.
3. Klicken Sie auf <i class="ph ph-list" aria-label="Aufgabe"></i> neben der Vorlage, die Sie verwenden möchten.
4. Wählen Sie **Kopie in neues Formular**. Die Vorlage wird im Dialogfeld **Formular bearbeiten** geöffnet. Siehe [Ein neues Formular erstellen][1].

## Vollkommen neue Formularvorlage erstellen

1. Gehen Sie zur Registerkarte **Formularvorlagen**.
2. Falls erforderlich, klicken Sie auf **Ordner hinzufügen**, um einen Ordner für die Vorlage zu erstellen.
3. Klicken Sie auf **Formular hinzufügen**. Daraufhin wird die Ansicht **Formular bearbeiten** angezeigt.
4. Geben Sie einen aussagekräftigen Namen für die Vorlage in das Feld **Formularname** ein.
5. Füllen Sie das Dialogfeld **Formular bearbeiten** wie in [Ein neues Formular erstellen][1] beschrieben aus.
6. Klicken Sie, sobald Sie fertig sind, auf **Speichern**. Die neue Formularvorlage wird in der Vorlagenliste angezeigt.

## Eine Vorlage auf der Grundlage eines vorhandenen Formulars erstellen

1. Gehen Sie zur Registerkarte **Formulare**.
2. Klicken Sie auf <i class="ph ph-list" aria-label="Aufgabe"></i> neben dem Formular, das Sie als Grundlage für die Vorlage verwenden möchten.
3. Wählen Sie **Kopie in Vorlage**.
4. Geben Sie in der Anzeige **Formular bearbeiten** einen aussagekräftigen Namen für die Vorlage in das Feld **Formularname** ein.
5. Falls erforderlich, klicken Sie auf **Ordner**, um einen Ordner für die Vorlage auszuwählen.
6. [Führen Sie die erforderlichen Änderungen aus][1].
7. Klicken Sie auf **Speichern**.
