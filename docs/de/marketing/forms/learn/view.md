# Formulare anzeigen

Auf der Registerkarte **Formulare** finden Sie alle vorhandenen Formulare.

Wenn Sie ein vorhandenes Formular anzeigen möchten, klicken Sie in der Liste auf das Formular. Die Ansicht **Formular anzeigen** wird angezeigt.

## Optionen anzeigen

* **Kürzlich**: Zeigt die Formulare an, an denen Sie zuletzt gearbeitet haben.
* **Meine Formulare**: Zeigt die von Ihnen erstellten Formulare an.
* **Aktive Formulare**: Zeigt nur die aktiven Formulare an. Formulare können manuell oder zu einem bestimmten Datum bzw. bei Erreichen einer bestimmten Anzahl von Antworten automatisch deaktiviert werden.
* **Alle Formulare**: Zeigt alle verfügbaren Formulare an.
* **In Formularen suchen**: Geben Sie einen Suchbegriff ein und drücken Sie die **EINGABETASTE** oder klicken Sie auf das Suchsymbol ![Symbol][img2].

## Die Ansicht Formular anzeigen

Oben in der Ansicht stehen folgende Optionen zur Verfügung:

* **Inaktiv**/**Aktiv**: Klicken Sie auf den Schieberegler, um das Formular zu aktivieren bzw. zu deaktivieren.
* **Aktualisieren**: Klicken Sie auf diese Schaltfläche, um die Ansicht zu aktualisieren.
* **Bearbeiten**: Klicken Sie auf diese Schaltfläche, um das Formular zu bearbeiten.
* ![Symbol][img1]: Klicken Sie auf diese Schaltfläche, um das Formular als [Favorit][1] festzulegen.

## Registerkarten

Die Ansicht **Formular anzeigen** besteht aus den folgenden Registerkarten:

### Zusammenfassung

Hier können Sie grundlegende Informationen zum Formular wie eine Vorschau auf das Formular und verschiedene Statistiken zur Übermittlung sehen:

* **Anfragen insgesamt**: Anzahl der übermittelten Formulare.
* **Letzte Anfrage**: Zeitpunkt, an dem zuletzt ein Formular übermittelt wurde.
* **In letzten 7 Tagen ausgefüllt**: Anzahl der in den letzten 7 Tagen pro Tag übermittelten Formulare.
* **Ausfüllungsrate**: Status der übermittelten Formulare.

Neben **Kampagne** befindet sich Link zur Formularantwortkampagne, die für dieses Formular verwendet wird. Klicken Sie auf den Link, um Informationen und Statistiken zur Formularantwort anzuzeigen. Weitere Informationen hierzu finden Sie unter [Kampagnen überwachen und Statistiken anzeigen][2].

### Veröffentlichen

Hier finden Sie den Link und das Skript zur [Veröffentlichung des Formulars][3] auf einer Webseite oder in einer Kampagne.

Klicken Sie auf die Schaltfläche **In neuem Fenster öffnen**, um eine Vorschau des Formulars anzuzeigen.

**Mit vorausgefüllten Feldern anzeigen**: Wählen Sie diese Option, um Feldnamen zum Link und zu JavaScript hinzuzufügen. Wenn Sie Vorlagenvariablen zur Formularverknüpfung in Kampagnen hinzufügen, können Sie die Formularfelder für erfasste Personen bereits [mit vorhandenen Daten, wie Name und E-Mail, vorausfüllen][4].

Auf der linken Seite der Ansicht können Sie verschiedene Typen von Veröffentlichungsoptionen für das Formular auswählen.

### Berichte

Diagramme zeigen Statistiken für das Formular (zur Anzeige der genauen Zahlen mit dem Mauszeiger über das Element fahren).

### Formularübermittlungen

Hier können Sie [alle übermittelten Formulare verarbeiten][5] und anzeigen. Mit den Filteroptionen unten in der Liste können Sie gezielt übermittelte Formulare mit einem bestimmten Status anzeigen.

### Antwort-E-Mail

Diese Registerkarte wird angezeigt, wenn **E-Mail-Antwort** für dieses Formular wurde (siehe [Ein neues Formular erstellen – Aktionen][6]). Hier können Sie eine Vorschau der mit dem Formular verbundenen Formularantwort (Kampagne) anzeigen.
