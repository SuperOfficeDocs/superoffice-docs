# Merge Tags zu Ihrer Mitteilung hinzufügen

Es gibt nichts Besseres als eine persönliche Note in der Branche. Um Ihre Kommunikation zu personalisieren, können Sie mit Hilfe von Zusammenführen-Tags (auch Vorlagenvariablen genannt) kundenspezifische Informationen in Ihr Kampagne einfügen.

Zusammenführen-Tags gibt es in vielen Formen: Dies können der Vorname des Kunden, sein Firmenname, die Anrede, der Name des Verkäufers, mit dem der Kunde zu tun hatte, usw. sein. Zusammenführen-Tags sind also Ihre benutzerfreundlichen Werkzeuge, mit denen Sie den Inhalt für jeden individuellen Kunden personalisieren können.

![Personalisieren Sie die Kampagne, indem Sie Textfeldern mit Zusammenführungs-Tags hinzufügen -screenshot][img2]

## Schritte

1. Wählen Sie einen Text in der Nachricht aus.

2. Geben Sie beispielsweise "Sehr geehrte(r) " ein.

3. Positionieren Sie den Mauszeiger hinter dem Text und klicken Sie auf **Zusammenführungs-Tags** in der Werkzeugleiste.

4. Wählen Sie in der angezeigten Liste **Person** und **Vorname** aus.

5. Fahren Sie mit der Eingabe des Textes fort und fügen die erforderlichen Seriendrucktags ein. Für eine Liste siehe die [Variablenreferenz][2].

> [!TIP]
> Bevor Sie eine Kampagne versenden, müssen Sie [testen, ob die Kundendaten richtig in die Mitteilung eingefügt werden][1].
