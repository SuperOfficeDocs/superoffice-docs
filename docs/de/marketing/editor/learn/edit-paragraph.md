# Inhalt bearbeiten

![Edit content -screenshot][img1]

> [!TIP]
> Um globale Eigenschaften und das Layout der Mitteilung zu bearbeiten, wählen Sie den **Textbereich** in der Seitenleiste aus. Wenn Sie für einzelne Bereiche bestimmte Eigenschaften definiert haben, werden diese von den globalen Eigenschaften nicht ersetzt.
>
> Verwenden Sie **Desktop** und (<i class="ph ph-monitor" aria-hidden="true"></i>) und **Mobil** (<i class="ph ph-device-mobile" aria-hidden="true"></i>) oben in der Seitenleiste, um die Eigenschaften für die Desktop- und Mobil-Versionen der Mitteilung zu bearbeiten.

## Textinhalt (Absatzblöcke) in einer Mitteilung bearbeiten

1. Gehen Sie zum Schritt **Inhalt** in der Kampagne oder öffnen Sie eine Vorlage zum Bearbeiten.

2. Klicken Sie auf den Text, den Sie bearbeiten möchten, damit die Textbearbeitungssymbolleiste angezeigt wird.

3. Bearbeiten Sie den Text mit den Schaltflächen in der Symbolleiste.

4. Klicken Sie auf **Merge Tags** (wie Vorlagenvariablen), um [Merge Tags in der Mitteilung einzufügen][2].

5. [Überwachte Links zu einer Mitteilung hinzufügen][8]. Sie können z. B. einen Link zu einer Informationsseite für ein Produkt hinzufügen. Sie können einen neuen Link erstellen oder einen vorhandenen Link hinzufügen.

    So fügen Sie einen regulären Link hinzu (nicht überwacht): Wählen Sie einen Text aus und klicken auf <i class="ph ph-link" aria-label="Link"></i> in der Werkzeugleiste. Geben Sie den Link in das Feld **URL** ein.

6. Bearbeiten Sie die Formatierung des Textes mithilfe der Felder in der Seitenleiste.

7. Klicken Sie auf **Übernehmen**.

## Andere Inhaltstypen wie Bilder oder Social-Media-Schaltflächen bearbeiten

1. Gehen Sie zum Schritt **Inhalt** in der Kampagne oder öffnen Sie eine Vorlage zum Bearbeiten.
2. Klicken Sie in der Nachricht auf den Inhalt, den Sie bearbeiten möchten. In der Seitenleiste werden alle Bearbeitungsoptionen für den ausgewählten Inhalt angezeigt.
3. Bearbeiten Sie die Optionen in der Seitenleiste.
4. Klicken Sie auf **Übernehmen**.

## Bearbeiten von Zeilen und Spalten

Klicken Sie auf eine Zeile in einer Nachricht, um die Reiheneigenschaften in der Seitenleiste zu bearbeiten.

Sie können die folgenden Bereiche bearbeiten:

* **Spalte**: Hier können Sie die Anzahl der Spalten und die Breite der Spalten in der Zeile ändern.

* **Spalteneigenschaften**: Hier können Sie die Eigenschaften für jede Spalte in der Zeile bearbeiten.

* **Zeileneigenschaften**: Hier können Sie die Eigenschaften der Zeile bearbeiten, z. B. Hintergrundfarbe und [Hintergrundbild][4].

* **Responsives Design**: Verwenden Sie diese Option, um bestimmte Zeilen in der Desktop- oder Mobilversion der Nachricht auszublenden.

> [!TIP]
> Um eine Farbe (Hintergrund- oder Rahmenfarbe) zu entfernen, klicken Sie in der Ecke des Farbfelds auf das X.

Klicken Sie auf **Weiter,** wenn Sie mit der Bearbeitung fertig sind.
