# Favoriten hinzufügen

Favoriten ermöglichen Ihnen den schnellen Zugriff auf für Sie besonders wichtige Kampagnen oder Formulare.

## Kampagne als Favoriten hinzufügen

Öffnen Sie die Kampagne oder das Formular, die bzw. das Sie zu den Favoriten hinzufügen möchten, und klicken Sie auf das Sternsymbol ![Symbol][img1] oben rechts. Der Stern wird daraufhin in Gelb ![Symbol][img2] angezeigt und die Erstellungs-Kampagne/das Formular wird sofort zu den Favoriten hinzugefügt.

In SuperOffice Service können Sie jetzt schnell auf den Datensatz zugreifen, indem Sie im Navigator auf **Marketing** und dann auf ![Symbol][img3] klicken.

> [!TIP]
> Wenn Sie mit dem Mauszeiger auf eine Kampagne oder ein Formular in der Favoritenliste zeigen, wird eine QuickInfo mit zusätzlichen Informationen zur Kampagne bzw. zum Formular eingeblendet.

## Kampagnen oder Formulare aus der Favoritenliste entfernen

1. Öffnen Sie die Kampagne oder das Formular.
2. Klicken Sie auf das Favoritensymbol ![Symbol][img2] in der Kopfzeile. Das Symbol wechselt zu ![Symbol][img1].
