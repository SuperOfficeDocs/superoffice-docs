# Der Navigator

Der Navigator auf der linken Seite ermöglicht Ihnen die Navigation zwischen den Hauptbereichen von SuperOffice CRM.

> [!TIP]
> Sie können die Breite des Navigators ändern, indem Sie auf <i class="ph ph-sidebar-simple" aria-label="Seitenleiste"></i> über dem Navigator klicken.

## SuperOffice-Logo ![SuperOffice-logo][img1]

Oben links im Navigator befindet sich das SuperOffice-Logo. Klicken Sie auf das Logo, um die [SuperOffice Community-Website][30] zu öffnen. Wenn Sie beim Klicken auf das Logo die **STRG**-Taste gedrückt halten, gelangen Sie direkt zur Firmenkarte Ihres Unternehmens. (Die Hauptfirma ist eine [Voreinstellung][31], die Sie ändern können.)

## Der Navigator im Standard-CRM

Der Navigator bietet schnellen Zugriff auf die wichtigsten Funktionen in SuperOffice CRM. Jedes Symbol öffnet eine bestimmte Ansicht.

| Symbol | Ansicht | Beschreibung |
|:-:|---|---|
| <i class="ph ph-gauge" aria-label="Dashboard-Symbol"></i> | **[Dashboard][1]** | Anzeigen von **Berichten und Diagrammen**, die CRM-Daten zusammenfassen. |
| <i class="ph ph-buildings" aria-label="Firmen-Symbol"></i> | **[Firma][2]** | Verwaltung von **Firmendatensätzen** und zugehörigen Informationen. |
| <i class="ph ph-user-circle" aria-label="Person-Symbol"></i> | **[Person][3]** | Zugriff und Verwaltung von **individuellen Personendaten**. |
| <i class="ph ph-calendar" aria-label="Kalender-Symbol"></i> | **[Kalender][4]** | Planung und Verwaltung von **Folgeaufgaben**. |
| <i class="ph ph-currency-circle-dollar" aria-label="Verkaufs-Symbol"></i> | **[Verkauf][5]** | Verfolgung und Verwaltung von **Verkaufschancen**. |
| <i class="ph ph-clipboard-text" aria-label="Projekt-Symbol"></i> | **[Projekt][6]** | Organisation und Überwachung von **Projektaktivitäten**. |
| <i class="ph ph-subtract-square" aria-label="Selektionen-Symbol"></i> | **[Selektion][7]** | Erstellung und Verwaltung von **Selektionen** für Kampagnen oder Workflows. |
| <i class="ph ph-at" aria-label="E-Mail-Symbol"></i> | **[Posteingang][8]** | Lesen und Beantworten von **eingehenden Nachrichten**. |
| <i class="ph ph-chat-teardrop-text" aria-label="Chat-Symbol"></i> | **[Chat][9]** | Beantwortung von **Chat-Anfragen** von Ihrer Webseite. |
| <i class="ph ph-target" aria-label="Marketing-Symbol"></i> | **[Marketing][11]** | Erstellung und Versand von **E-Mail- und SMS-Kampagnen**. Formulare erstellen. Automatisierung. |
| <i class="ph ph-ticket" aria-label="Service-Symbol"></i> | **[Anfragen][34]** | Verwaltung von **Service-Anfragen**. |
| <i class="ph ph-wrench" aria-label="Werkzeug-Symbol"></i> | **Werkzeuge** | Zugriff auf externe **Anwendungen und Webseiten**. |

## <a id="navigator"></a>Navigatoroptionen

Beim Klicken auf eine Navigatorschaltfläche erscheinen folgende Optionen:

| Symbol | Funktion | Beschreibung |
|:-:|---|---|
| <i class="ph ph-clock" aria-label="Uhr-Symbol"></i> | **[Verlauf][16]** | Zeigt kürzlich geöffnete Datensätze wie Anfragen, Personen oder Selektionen. |
| <i class="ph ph-star" aria-label="Stern-Symbol"></i> | **[Favoriten][15]** | Zeigt Ihre favorisierten Datensätze. |
| <i class="ph ph-magnifying-glass" aria-label="Such-Symbol"></i> | **Suchen** | Zeigt Ergebnisse der letzten Suche. Die Suchleiste ist dynamisch. |

> [!NOTE]
> Durch Klicken auf **Dashboard** wird nur das Dashboard geöffnet.

Wenn Sie den Mauszeiger über einen Datensatz bewegen, erscheint eine QuickInfo mit zusätzlichen Details.
