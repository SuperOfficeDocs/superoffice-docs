# Objekte löschen und wiederherstellen

Um mit Kunden und Personen effizient arbeiten zu können, ist es wichtig, eine "aufgeräumte" Umgebung zu haben. Wenn Sie neue Firmen und Personen hinzufügen, müssen Sie überprüfen, dass diese nicht zuvor hinzugefügt wurden. Natürliche müssen sie auch sicherstellen, dass Sie korrekte Informationen zu Firmen und Personen hinzufügen. Löschen Sie also unbenutzte Firmen und Personen und führen Sie doppelte Firmen und Personen zusammenführen.

## Firmen und Personen zusammenführen

Wenn Sie doppelte Informationen finden, führen Sie diese so schnell wie möglich zusammen, um sicherzustellen, dass es einfacher ist, die richtige Firma und die richtige Person zu finden. Sie können einen Kontakt auch verschieben, wenn er auf der falschen Firmenkarte hinzugefügt wurde. Dadurch können in Zukunft Duplikate vermieden werden.

> [!NOTE]
> Für diese Funktion sind Benutzerrechte erforderlich, wenden Sie sich daher an Ihren Administrator, um die erforderlichen funktionellen Rechte zu erhalten.

**Aufgaben:**

* [Personen zusammenführen][4]
* [Person verschieben][5]
* [Firmen zusammenführen][1]

## Firma und Person löschen

Wenn Sie falsche oder doppelte Informationen hinzugefügt haben, wird empfohlen, eine Person zu löschen. Formulare können auf verschiedene Weise gesucht werden:

* [Firma und zugehörige Daten löschen][2]
* [Person löschen][3]
* [Massenlöschung von Personen und/oder Firmen mit Selektion][6]

Wenn Sie ein Element in SuperOffice CRM löschen, wird je nach zu löschendem Element das Dialogfeld **Folgeaufgabe löschen**, **Verkauf löschen**, **Selektion löschen** usw. angezeigt. Das Dialogfeld zeigt den Namen des zu löschenden Elements an, sodass Sie Ihre Auswahl prüfen können.

Wenn Sie viele Personen oder Firmen löschen müssen, wird eine Massenlöschung empfohlen, für die Sie sie zu einer neuen Selektion hinzufügen und die Option Aufgaben löschen im Menü Aufgabe verwenden. Klicken Sie bei der Selektion auf die Schaltfläche Aufgabe und wählen Sie eine der gewünschten Löschfunktionen aus.

### Warum ist die Schaltfläche Löschen deaktiviert (ausgegraut)?

Wenn die Schaltfläche **Löschen** deaktiviert ist (hellgrau), verfügen Sie nicht über die zum Löschen des jeweiligen Elements erforderlichen Rechte.

Beispiele:

* Sie können keine Person löschen, die auch Mitarbeiter einer SuperOffice-Firma ist.
* Sie können Folgeaufgabe, Dokumente, Verkäufe, Projekte usw., die Ihnen nicht gehören, nur löschen, wenn Sie über die entsprechenden Benutzerrechte verfügen.

## <a id="restore" />Gelöschte Firmen und Personen wiederherstellen

Wenn Sie etwas gelöscht haben, das für Firmen und Personen nicht gelöscht werden sollte, können Sie es immer noch aus Ihrem Papierkorb wiederherstellen. Das Löschen wird weniger beängstigend, wenn Sie wissen, dass Sie noch eine Wiederherstellungsoption haben.

![Sie können gelöschte Objekte vom Papierkorb im Menü Persönliche Einstellungen wiederherstellen -screenshot][img3]

Gelöschte Firmen und Personen bleiben standardmäßig 14 Tage lang im Papierkorb erhalten. Danach werden sie endgültig gelöscht.

Sie haben auch die Möglichkeit, die gelöschten Firmen und Personen anderer Benutzer wiederherzustellen. Aktivieren Sie hierzu einfach EIN für die Option Gelöschte Firmen für alle Benutzer anzeigen.

1. Wählen Sie ![Symbol][img1] **Persönliche Einstellungen** > **Papierkorb** aus.

2. Wählen Sie im Dialogfeld die Registerkarte **Firma** oder **Person** aus. Von Ihnen gelöschte Firmen und Personen werden angezeigt.

3. Optional: Wählen Sie **Gelöschte Firmen/Personen für alle Benutzer anzeigen**. Für diese Option sind funktionelle Rechte erforderlich.

4. Wählen Sie die Firmen oder Personen aus, die Sie wiederherstellen möchten.
    * Beim Wiederherstellen einer Firma werden auch alle Personen wiederhergestellt, die zu dieser Firma gehören.
    * Beim Wiederherstellen einer Person, deren Firma ebenfalls gelöscht wurde, werden die Firma und die ausgewählte Person wiederhergestellt.

    ![Firmen und Personen vom Papierkorb wiederherstellen -screenshot][img2]

5. Klicken Sie auf **Wiederherstellen**. Die Firmen und Personen werden in SuperOffice wiederhergestellt.

## Schaltflächen

| Schaltflächen | Wirkung |
|---|---|
| Ja | Bestätigt das Löschen eines einzelnen Elements. |
| Ja alle | Bestätigt das Löschen aller ausgewählten Elemente. Wenn Sie z. B. fünf Elemente zum Löschen ausgewählt haben, werden durch Klicken auf diese Schaltfläche automatisch alle fünf gelöscht. |
| Nein | Bricht das Löschen eines einzelnen Elements ab. Das Dialogfeld wird geschlossen, ohne dass ein Element gelöscht wird. |
| Abbrechen | Bricht das Löschen aller ausgewählten Elemente ab. Das Dialogfeld wird geschlossen, ohne dass ein Element gelöscht wird. |

* So blenden Sie das Dialogfeld aus: Aktivieren Sie unten im Löschdialogfeld das Kontrollkästchen **Diesen Dialog nicht mehr anzeigen**.
* So blenden Sie das Dialogfeld ein: Gehen Sie zu ![Symbol][img1] **Persönliche Einstellungen** > **Einstellungen** > **Funktionen** und stellen Sie die Option **Dialogfeld Bestätigung anzeigen** auf **Nein**.

> [!NOTE]
> Wenn Sie eine Firma, eine Selektion oder ein Projekt löschen, werden Sie immer um Bestätigung gebeten. Daher steht die Option **Diesen Dialog nicht mehr anzeigen** hier nicht zur Verfügung.
