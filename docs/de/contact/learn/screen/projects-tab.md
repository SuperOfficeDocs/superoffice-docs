# Registerkarte Projekte

Auf der Bereichsregisterkarte **Projekte** werden alle Projekte angezeigt, bei denen die aktive Person ein Projektmitglied ist.

Klicken Sie auf **Zu Projekt hinzufügen**, um Projektmitglieder zu einem Projekt hinzuzufügen.

Doppelklicken Sie auf ein Projekt in der Liste, um die Anzeige Projekt zu öffnen, welche die Informationen über das aktive Projekt enthält.
