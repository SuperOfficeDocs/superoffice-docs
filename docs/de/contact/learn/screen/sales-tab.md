# Registerkarte Verkäufe

Auf der Bereichsregisterkarte **Verkäufe** werden die Verkäufe, die im Zusammenhang mit der Person stehen, angezeigt.

[!include[Contents of the Sales section tab](../../../learn/includes/sales-tab.md)]

[!include[Conf. columns](../../../learn/includes/tip-configure-columns.md)]

Wenn Sie auf einen Verkauf in der Liste doppelklicken, wird die Ansicht Verkauf mit den Informationen zum Verkauf geöffnet.

Klicken Sie auf **Hinzufügen**, um einen neuen [Verkauf][1] zu erstellen.

Klicken Sie auf das ![Symbol][img1] der Schaltfläche **Filter**, damit Sie nach Datum, Benutzern oder Gruppen die [Liste filtern][3] können.

Wenn Sie **Beteiligte einschließen** unten auf der Bereichsregisterkarte aktivieren, werden auch die Verkäufe angezeigt, an denen die [Person als Beteiligter mitwirkt][2].
