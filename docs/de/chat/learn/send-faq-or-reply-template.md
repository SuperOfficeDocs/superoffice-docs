# FAQs, Antwortvorlagen und Schnellantworten in einem Chat senden

Kunden stellen oft dieselben Fragen. Mit vordefinierten Texten können Sie Zeit sparen, da Sie Ihre Antwort nicht vollständig eingeben müssen. Sie können aus FAQs und Antwortvorlagen in SuperOffice Service und [von Ihnen erstellte Schnellantworten][1] auswählen.

1. Klicken Sie auf <i class="ph ph-article" aria-label="Insert"></i> im unteren Teil des Chat-Fensters.

2. Wählen Sie eine der folgenden Optionen und blättern Sie in den Ordnern. Sie können auch einen Suchtext in das obere Feld eingeben oder auf die entsprechenden Symbole klicken, um auf den Verlauf und die Favoriten zuzugreifen.
    * **FAQ**: Zum Einfügen einer Verknüpfung zu einer FAQ im SuperOffice Kundenzentrum. Nur FAQs, die allgemein zugänglich sind und veröffentlicht wurden, stehen hier zur Verfügung.
    * **Antwortvorlagen**: Zum Einfügen von Vorlagentext.
    * **Schnellantworten**: Zum Einfügen einer Schnellantwort. Um die Option [Schnellantworten erstellen][1] auszuführen, klicken Sie auf **Bearbeiten** (<i class="ph ph-pencil-simple" aria-hidden="true"></i>).

3. Drücken Sie die **EINGABETASTE** oder klicken Sie auf **Senden**.
