# Anfrage in einem aktiven Chat erstellen

Wenn Sie das Problem nicht über den Chat lösen können, können Sie eine Anfrage erstellen, die von einem anderen Spezialisten bearbeitet wird.

1. Klicken Sie auf <i class="ph ph-list" aria-label="Aufgabe"></i> und wählen Sie **Anforderung erstellen** aus.
2. Füllen Sie im Dialogfeld die relevanten Felder aus.
3. Das Chatprotokoll wird der Anfrage hinzugefügt, aber Sie können hier bei Bedarf weitere Informationen hinzufügen.
4. Klicken Sie auf **Speichern**.
5. Eine kurze Nachricht mit der Anfrage-ID wird zum Textfeld im Chat hinzugefügt. Klicken Sie auf **Senden**, um diese an den Kunden zu senden.
