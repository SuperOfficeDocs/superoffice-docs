# Einstellungen für Gmail Link bearbeiten

Im Dialogfeld **SuperOffice Gmail Link - Einstellungen** können Sie Archiv-, Konto- und allgemeine Optionen einstellen und Versionsinformationen zu SuperOffice Mail Link anzeigen.

Öffnen Sie das Dialogfeld **SuperOffice Gmail Link - Einstellungen**, indem Sie in der Ecke rechts oben im Browser auf das SuperOffice-Symbol ![Symbol][img1] klicken.

## Archivdialogfeld immer anzeigen

Wenn diese Option deaktiviert ist, wird das Dialogfeld **E-Mail in SuperOffice archivieren** nicht angezeigt, wenn Sie durch Klicken auf die Schaltfläche **Archiv** eine E-Mail archivieren. Bei dieser "Schnellarchivierung" werden für die Einstellungen alle Standardwerte verwendet, auch wenn das Archivierungsdialogfeld nicht angezeigt wird. Nach der Archivierung der E-Mail-Mitteilung wird eine Popup-Mitteilung angezeigt.

Manchmal wird immer das Dialogfeld **E-Mail archivieren in SuperOffice** angezeigt:

* Wenn die E-Mail-Adresse des Absenders nicht in SuperOffice erfasst wurde.
* Wenn die E-Mail-Adresse des Absenders für mehrere Personen/Firmen in SuperOffice erfasst wurde.

## Andere als E-Mail-Vorlagen in Vorlagenliste hinzufügen

![Symbol][img2] aktiviert: Alle Dokumentvorlagen werden in der Liste **Typ** im Dialogfeld **E-Mail in SuperOffice archivieren** angezeigt.

![Symbol][img3] deaktiviert: Es werden nur Dokumentvorlagen des Typs **E-Mail** angezeigt.

## Standardvorlage für ausgehende E-Mail

Wählen Sie die Standarddokumentvorlage aus, wenn die Option [ausgehende E-Mail-Nachrichten archivieren][4] verwendet werden soll.

> [!TIP]
> Geben Sie beim Suchen einer Dokumentvorlage die Anfangsbuchstaben in die Liste ein.

## Standardvorlage für eingehende E-Mail

Wählen Sie die Standarddokumentvorlage aus, wenn die Option [eingehende E-Mail-Nachrichten archivieren][3] verwendet werden soll.

## Standardaktivitätstyp

Wählen Sie den Standardaktivitätstyp aus, der zum [Erstellen einer Aktivität][5] auf der Grundlage einer eingehenden E-Mail verwendet werden soll.

## SuperOffice-Kontoinformationen

Unten im Dialogfeld **SuperOffice Gmail Link - Einstellungen** werden folgende Informationen angezeigt:

* **Firmenname**: Der Name Ihrer Firma in SuperOffice.
* **URL**: Die SuperOffice-Installation, mit der Gmail Link verbunden ist. Klicken Sie auf den Link, um SuperOffice aufzurufen.
* **Benutzer**: Der Name des SuperOffice-Benutzers.
* **Server-Version**: Die Version der SuperOffice-Serverinstallation.

## SuperOffice Benutzer ändern

Klicken Sie auf diese Schaltfläche, um die Verbindung zu einer anderen SuperOffice-Installation herzustellen. Befolgen Sie die Anweisungen auf dem Bildschirm.
