# SuperOffice Gmail Link aktivieren

Sehen Sie sich dieses Video an, um zu lernen wie Sie Ihren Gmail-Link einrichten und Ihr Gmail-Konto mit SuperOffice verbinden (Videolänge - 2:53) oder führen Sie die folgenden Schritte aus:

<!-- markdownlint-disable-next-line MD034 DOCSMD007 -->
> [!Video https://www.youtube.com/embed/kiQyO9KwiEs]

## Schritte

1. Gehen Sie zum App-Eintrag für Gmail Link im [SuperOffice App Store][1].

1. Klicken Sie auf **Installieren** und wählen Sie den Browser aus, für den Sie die Erweiterung installieren möchten.

    **Chrom:** Wenn Sie Chrome auswählen, werden Sie zum Chrome Web Store weitergeleitet. Dort klicken Sie auf die Schaltfläche **Zu Chrome hinzufügen** und bestätigen Sie, dass Sie diese Erweiterung hinzufügen möchten.
    * **Firefox**: Wenn Sie Firefox auswählen, wird das Firefox-Browser-Installationsprogramm geöffnet und Sie werden aufgefordert, das Browser-Plugin für Firefox zuzulassen. Bestätigen Sie mit einem Klick auf **Weiter zur Installation**.

1. Der letzte Schritt für beide Browser besteht im Einloggen bei Ihrem SuperOffice CRM-Konto. SuperOffice und Gmail stellen automatisch fest, dass die Erweiterung hinzugefügt wurde.
