# Ausgehende E-Mails archivieren

Wenn Sie eine E-Mail senden, können Sie diese (einschließlich sämtlicher Anhänge) ebenfalls als Dokument in SuperOffice CRM archivieren. Damit können Sie Ihre ausgehende Korrespondenz bequem nachverfolgen.

Gehen Sie wie folgt vor:

1. Klicken Sie in Gmail auf **Schreiben**.

2. Schreiben Sie die E-Mail-Mitteilung.

    > [!NOTE]
    > Wenn Sie die E-Mail an mehrere Empfänger senden wollen, wird sie nur für den ersten Empfänger archiviert.

3. Klicken Sie auf **Senden und archivieren**. Die E-Mail wird gesendet.

4. Im Dialogfeld **E-Mail in SuperOffice archivieren** werden die Felder **Firma** und **Person** automatisch ausgefüllt, wenn die E-Mail-Adresse des Empfängers in SuperOffice erfasst wurde. Geben Sie die relevanten Informationen in die folgenden Felder ein:

    [!include[Archiving options for Gmail](includes/gmail-archiving-options.md)]

    Wenn der Empfänger der E-Mail nicht in SuperOffice CRM aufgelistet wird, wird oben im Dialogfeld eine Warnung angezeigt. Eine Warnung wird auch angezeigt, wenn der Empfänger in SuperOffice CRM für mehrere Firmen erfasst wurde.

5. Klicken Sie auf **Speichern**. Die E-Mail (einschließlich sämtlicher Anhänge) wird als neues Dokument in SuperOffice CRM archiviert. Sie können die E-Mail dann öffnen, indem Sie in der Bereichsregisterkarte **Aktivitäten** darauf doppelklicken.

Klicken Sie auf den Link in der Popup-Meldung, um das Dokument in SuperOffice zu öffnen.

> [!TIP]
> Wenn Sie die E-Mail nicht beim Senden archivieren, können Sie immer noch die Option [Gehe zum Ordner mit den gesendeten E-Mails und die Mitteilung archivieren][3] ausführen.
