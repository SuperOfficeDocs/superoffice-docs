# Eingehende E-Mails archivieren

Sie können eine eingehende E-Mail (einschließlich sämtlicher Anhänge) als Dokument in SuperOffice CRM archivieren. Damit können Sie Informationen, die Sie per E-Mail erhalten, bequem zusammen mit anderen Firmen-/Personeninformationen in SuperOffice CRM speichern.

![Wenn Sie SuperOffice Gmail Link installiert haben, können Sie E-Mails archivieren, Aktivitäten oder Anfragen stellen -screenshot][img2]

## Schritte

1. Wählen Sie im Gmail-Posteingang die E-Mail aus, die archiviert werden soll.

2. Klicken Sie auf ![Symbol][img1] in der E-Mail-Werkzeugleiste.

    Wenn der Absender der E-Mail nicht in SuperOffice CRM aufgelistet wird, wird oben im Dialogfeld eine Warnung angezeigt. Eine Warnung wird auch angezeigt, wenn der Absender in SuperOffice CRM für mehrere Firmen erfasst wurde.

3. Im Dialogfeld werden die Felder **Firma** und **Person** automatisch ausgefüllt, wenn der Absender in SuperOffice erfasst wurde. Geben Sie die relevanten Informationen in die folgenden Felder ein:

    [!include[Archiving options for Gmail](includes/gmail-archiving-options.md)]

4. Klicken Sie auf **Speichern**. Die ausgewählte E-Mail (einschließlich sämtlicher Anhänge) wird als neues Dokument in SuperOffice CRM archiviert. Sie können die E-Mail dann öffnen, indem Sie in der Bereichsregisterkarte **Aktivitäten** darauf doppelklicken.

Klicken Sie auf den Link in der Popup-Meldung, um das Dokument in SuperOffice zu öffnen.

> [!NOTE]
> Sie können auch [Anhänge als Dokumente archivieren][3].
