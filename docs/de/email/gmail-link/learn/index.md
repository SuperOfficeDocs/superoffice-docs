# SuperOffice Gmail Link

SuperOffice Gmail Link ist ein Plug-in, das den Google Gmail-Client mit SuperOffice verknüpft. Das heißt, Sie können unter anderem:

* Archivieren Sie ein- und ausgehende E-Mails und Anhänge von Ihrem Gmail-Konto als Dokumente in SuperOffice.
* Erstellen Sie anhand von E-Mails in Gmail Kalenderaktivitäten in SuperOffice.
* Erstellen Sie anhand von E-Mails in Gmail Anfragen.
* Absender in SuperOffice nachschlagen: Diese leicht verständlichen Assistenten helfen Ihnen, eine neue Firma, eine neue Person oder eine neue E-Mail-Adresse in Ihrer SuperOffice CRM-Datenbank zu erstellen, wenn der Absender nicht unbekannt ist.
