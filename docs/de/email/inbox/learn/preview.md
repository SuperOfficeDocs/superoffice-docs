# E-Mail-Nachrichten anzeigen

Wenn Sie unter E-Mail eine Nachricht auswählen, wird eine Vorschau der Nachricht auf der rechten Seite der Ansicht angezeigt.

Die Registerkarte **Vorschau** enthält folgende Felder:

| Feld | Beschreibung |
|---|---|
| Experten | Zeigt den Betreff der E-Mail an. |
| Von | Zeigt den Absender an. Wenn der Absender in SuperOffice CRM erfasst ist, können Sie mit dem Mauszeiger auf den Namen zeigen, um die Personenkarte anzuzeigen. |
| Datum | Zeigt das Datum und die Uhrzeit der E-Mail an. |
| An | Zeigt den oder die Empfänger der E-Mail an. |
| Anhänge | Zeigt alle Anhänge der Nachricht an. |

> [!NOTE]
> Gehen Sie zum ![Symbol][img1] **Persönliche Einstellungen** &gt; **E-Mail-Optionen**. Öffnen Sie die Registerkarte **Allgemein**. Wenn Sie die Option **Inline-Bilder anzeigen** aktivieren, werden Inline-Bilder als Vorschau angezeigt.
