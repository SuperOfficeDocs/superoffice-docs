# E-Mails von SuperOffice CRM erstellen

Es gibt zwei Möglichkeiten, eine E-Mail von SuperOffice CRM zu erstellen. Sie können die Schaltfläche **E-Mail** in der Werkzeugleiste am unteren Rand des SuperOffice CRM-Fensters verwenden. Dies funktioniert mehr oder weniger auf die gleiche Weise wie unter [E-Mails von Microsoft Outlook erstellen][1] beschrieben, nur dass vordefinierte E-Mail-Textvorlagen verwendet werden können.

Außerdem können Sie auch die Schaltfläche **Schreiben** in der Werkzeugleiste verwenden.

1. Klicken Sie auf die Schaltfläche **Schreiben** in der Werkzeugleiste am unteren Rand des SuperOffice CRM-Fensters.
2. Wählen Sie im Feld **Typ** oben im Dialogfeld **Dokument** eine E-Mail-Vorlage aus.
3. Geben Sie die erforderlichen Informationen in den restlichen Feldern ein.
4. Klicken Sie auf die Schaltfläche **Erstellen**, um die neue E-Mail in Ihrem Standard-E-Mail-Client zu öffnen.
5. Wenn SuperOffice CRM die Firma/Person, die Sie im Dialogfeld **Dokument** ausgewählt haben, mit einer E-Mail-Adresse enthält, wird diese Adresse automatisch in das Feld **An** eingefügt.
    Geben Sie die E-Mail-Adresse aller anderen Empfänger ein, die Sie hinzufügen möchten. Sie können auch das Dialogfeld **Empfänger auswählen** verwenden (siehe [Empfänger auswählen][4]).
6. Schreiben Sie den Text der Nachricht.
7. Fügen Sie alle Anhänge hinzu, die Sie einbeziehen möchten. Sie können auch [Dokumente von SuperOffice CRM hinzufügen][5].
8. Klicken Sie auf **Senden**, um die Nachricht an den/die ausgewählten Empfänger zu senden.

## E-Mail mit einer vordefinierten E-Mail-Textvorlage erstellen

Wenn Sie eine E-Mail von SuperOffice CRM erstellen, können Sie vordefinierte SuperOffice-Vorlagen nur für den E-Mail-Text verwenden.

1. Stellen Sie sicher, dass eine E-Mail-Textvorlage verfügbar ist und dass die Option **SuperOffice-E-Mail-Text verwenden** im Dialogfeld **SuperOffice CRM** ausgewählt wurde.

2. Klicken Sie auf die Schaltfläche **E-Mail** in der Werkzeugleiste am unteren Rand des SuperOffice CRM-Fensters. (Dies gilt auch, wenn Sie in SuperOffice CRM auf eine E-Mail-Adresse klicken, eine E-Mail über die Registerkarte **Personen** senden oder eine E-Mail über die Bereichsregisterkarte **Aktivitäten** beantworten/weiterleiten.)

3. Geben Sie im Feld **An** die E-Mail-Adresse(n) der/des Empfänger(s) ein. Sie können auch das Dialogfeld **Empfänger auswählen** verwenden (siehe [Empfänger auswählen][4]).

    > [!TIP]
    > Wenn Sie mehrere Kontakte in einer Liste in SuperOffice CRM auswählen, mit der rechten Maustaste klicken und **E-Mail senden** auswählen, werden alle Personen als Empfänger der E-Mail festgelegt.

4. Schreiben Sie den Text der Nachricht.

5. Fügen Sie alle Anhänge hinzu, die Sie einbeziehen möchten. Sie können auch [Dokumente von SuperOffice CRM hinzufügen][5].

6. Klicken Sie auf **Senden**, um die Nachricht an den/die ausgewählten Empfänger zu senden.
