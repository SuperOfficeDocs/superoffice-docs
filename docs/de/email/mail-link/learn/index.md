# SuperOffice Mail Link

SuperOffice Mail Link ist ein Plug-in, der den Microsoft Outlook E-Mail-Client mit SuperOffice CRM und/oder SuperOffice Service verknüpft. Das heißt, Sie können unter anderem:

* E-Mails und Anhänge in SuperOffice CRM archivieren.
* Eine E-Mail als Dokument oder Aufgabe in SuperOffice CRM archivieren.
* Eine E-Mail als Anfrage in SuperOffice Service archivieren.
* Den Absender einer E-Mail suchen.
* Funktionen für die Standardarchivierung festlegen.
* SuperOffice-Vorlagen oder vordefinierte E-Mail-Textvorlagen zum Erstellen von E-Mails verwenden.
* Informationen über Firmen und Personen in SuperOffice CRM anhand von erhaltenen E-Mails aktualisieren.

## Schaltflächen auf der E-Mail-Client-Symbolleiste

Diese Schaltflächen werden auf der Microsoft Outlook-Werkzeugleiste angezeigt:

| Schaltflächen | Funktion |
|:-:|---|
| ![Symbol][img1] | E-Mails in SuperOffice CRM archivieren |
| ![Symbol][img2] | Nach dem Absender einer E-Mail suchen |
| ![Symbol][img3] | Klicken Sie auf den kleinen Pfeil rechts neben der Schaltfläche **SuperOffice**, um das Dialogfeld **SuperOffice CRM-Optionen** zu öffnen. |
