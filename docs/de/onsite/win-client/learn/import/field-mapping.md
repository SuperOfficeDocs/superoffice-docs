# Feldzuordnung

Mit Feldzuordnung ist die Beziehung zwischen den Feldern der Importdatei und den Feldern in der Datenbank gemeint. Diese Beziehung wird in der Tabelle **Feldzuordnung** widergespiegelt, die aus den folgenden Spalten besteht:

* **Feld in Datei**: Wenn Sie eine Datei im Feld **Importdatei** festlegen, wird der Inhalt des ersten Datensatzes in der importierten Datendatei in diese Spalte geladen.

* **Feld in SuperOffice CRM**: zeigt an, in welche Felder der SuperOffice-Datenbank die Informationen in der Spalte **Feld in Datei** importiert werden.
