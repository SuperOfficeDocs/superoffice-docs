# Import

[!include[Back up database before import](../../../../admin/import/learn/includes/caution-backup-before-import.md)]

Über die Ansicht **Import** können Sie wichtige externe Daten in die SuperOffice-Datenbank aufnehmen. Sie können zum Beispiel einen Kundenindex importieren, damit Sie ihn in SuperOffice CRM verwenden können. Genauer gesagt können Sie ein der Standardversion der Einstellungen für den Windows-Client folgende Daten in die SuperOffice-Datenbank importieren:

* Firmen mit dazugehörigen benutzerdefinierten Feldern und Interessen
* Personen mit dazugehörigen benutzerdefinierten Feldern und Interessen
* Projekte mit dazugehörigen benutzerdefinierten Feldern
* Folgeaufgaben
* Verkäufe
* Produkte
* Währungen
* Postleitzahlen
* Wichtige Tage

Sie können in [Firmen/Personen und Produkte in Einstellungen und Verwaltung importieren][9].

Die Felder in der Ansicht **Import** und ihre Verwendung werden unter [Dateien importieren][2] beschrieben. Bevor Sie mit dem Importieren beginnen, sollten Sie auch den Abschnitt [Importroutinen][1] lesen
