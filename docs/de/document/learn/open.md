# Dokumente öffnen

Sie können vorhandene Dokumente von den folgenden Orten aus öffnen und anzeigen:

<!-- markdownlint-disable MD032 -->
[!include[List of Activities section tabs](../../learn/includes/list-activities-section-tabs.md)]
* In der Ansicht Selektion auf der Bereichsregisterkarte **Dokumente**, wenn eine Dokumentselektion aktiv ist.
* In den Suchergebnissen auf der Registerkarte **Dokument** in der Ansicht **Suchen**.
<!-- markdownlint-restore -->

Sie können Dokumente im [Bearbeitungs- oder im Lesemodus][1] öffnen. Wenn Sie ein Dokument im Bearbeitungsmodus öffnen, können nur Sie selbst es bearbeiten. Andere Benutzer können das Dokument im Lesemodus öffnen.

> [!NOTE]
> Wenn Ihre Organisation eine Dokumentbibliothek wie SharePoint nutzt, können mehrere Personen ein Dokument gleichzeitig bearbeiten.

Das Dokument wird in dem Programm geöffnet, in dem es erstellt wurde.

Das Verfahren hängt davon ab, wie Dokumente in Ihrer SuperOffice-Installation verwaltet werden. Siehe [Dokumente bearbeiten][2].
