# Dashboards personalisieren

Sie können jedes Dashboard anpassen, um die Informationen anzuzeigen, die für Sie am wichtigsten sind. Wählen Sie aus vordefinierten Kacheln oder erstellen Sie eigene, um Aktivitäten, Kundensegmente, Produkttrends, Projektfortschritte oder Verkaufszahlen zu visualisieren.

## Dashboard-Einstellungen bearbeiten

Sie können Name, Beschreibung, Layout oder Sichtbarkeit eines bestehenden Dashboards aktualisieren.

1. Öffnen Sie die Ansicht **Dashboard**.

1. Wählen Sie das Dashboard aus, das Sie bearbeiten möchten.

1. Klicken Sie auf die Schaltfläche **Aufgabe** (<i class="ph ph-dots-three-circle-vertical" aria-hidden="true"></i>) und wählen Sie **Dashboard-Details bearbeiten** aus.

1. Nehmen Sie Ihre Änderungen vor:
    * Ändern Sie **Name** oder **Beschreibung**
    * Wählen Sie ein anderes **Design**
    * Passen Sie die Anzahl der **Spalten** an
    * Wenn Sie über die erforderlichen Rechte verfügen, aktualisieren Sie die Felder **Sichtbar für** und **Anheften für**

    > [!NOTE]
    > Der Ersteller des Dashboards ist der **Eigentümer**. Die Eigentümerschaft kann nicht übertragen werden.

1. Klicken Sie auf **Speichern**.

> [!NOTE]
> Wenn Sie die Anzahl der Spalten ändern, werden die Kacheln automatisch an das neue Layout angepasst. Keine Kacheln werden gelöscht, aber das Raster kann neu angeordnet werden, wodurch Lücken entstehen. Sie können die Kacheln anschließend verschieben oder ihre Größe ändern.

## Kacheln und Layout bearbeiten

So fügen Sie Kacheln hinzu, bearbeiten, verschieben oder entfernen Sie sie:

1. Öffnen Sie das gewünschte Dashboard.
1. Klicken Sie auf <i class="ph ph-dots-three-circle-vertical" aria-label="Aufgabe"></i> und wählen Sie **Dashboard-Kacheln bearbeiten**.
1. [Folgen Sie diesen Schritten, um Kacheln zu verwalten][2].

## <a id="delete"></a>Ein Dashboard löschen

Nur der **Eigentümer** eines Dashboards kann es löschen. Selbst Administratoren können keine fremden Dashboards löschen. Eigentümerschaft kann nicht geändert werden.

1. Öffnen Sie die Ansicht **Dashboard**.
1. Wählen Sie das Dashboard aus, das Sie löschen möchten.
1. Klicken Sie auf <i class="ph ph-dots-three-circle-vertical" aria-label="Aufgabe"></i> und wählen Sie **Dieses Dashboard löschen**.
1. Klicken Sie auf **OK**, um zu bestätigen.

> [!NOTE]
> Wenn das Dashboard für andere Benutzer angeheftet war, wird es auch aus deren Ansicht entfernt.
