# Registerkarte Projekte

![Projektbereich in Firma -screenshot][img1]

Auf der Bereichsregisterkarte **Projekte** werden alle zur aktiven Firma gehörenden Projekte angezeigt. Um hier ein Projekt anzeigen zu können, muss mindestens ein Mitarbeiter in der betreffenden Firma als Projektmitglied eingetragen sein.

Klicken Sie auf **Hinzufügen**, damit Sie ein [neues Projekt erstellen][2] können

Doppelklicken Sie auf ein Projekt in der Liste, um die Anzeige Projekt mit Informationen über das aktive Projekt zu öffnen.
