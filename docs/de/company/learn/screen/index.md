# Die Bereichsregisterkarten der Ansicht Firma

Die Ansicht Firma enthält die folgenden Bereichsregisterkarten:

| Bereichsregisterkarte | Beschreibung |
|---|---|
| [Personen][1] | Enthält Informationen über Mitarbeiter der aktiven Firma. |
| [Beziehungen][2] | Zeigt alle vorhandenen Verbindungen zwischen Firmen und/oder Personen an. |
| [Aktivitäten][3] | Enthält alle mit der aktiven Firma verknüpften Aktivitäten. |
| [Projekte][4] | Zeigt alle Projekte für die aktive Firma an. |
| [Verkäufe][5] | Enthält Informationen über alle mit der Firma verbundenen Verkäufe an. |
| [Anfragen][6] | Zeigt SuperOffice Service-Anfragen für die aktive Firma an. |
| [ERP][7] | Hier können Sie eine Firma aus SuperOffice mit einem Kunden/Lieferanten im ERP-System verbinden. |
