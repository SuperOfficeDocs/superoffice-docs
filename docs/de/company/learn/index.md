# Firma

Alle Informationen bezüglich einer Firma – egal, ob Kunde, Partner oder Lieferant – werden auf der Firmenkarte/der Anzeige gespeichert. Diese Karte bietet Ihnen einen sofortigen Überblick über die Firma auf einem Bildschirm.

![Ansicht Firma -screenshot][img3]

[!include[Intro](includes/company-screen-intro.md)]

In der Ansicht **Firma** wird jeder Datensatz in der Datenbank wie eine Indexkarte angelegt, die alle Informationen für jene Firma enthält.

Um zwischen Firmen zu wechseln, klicken Sie auf die Schaltflächen Zurück/Weiter ( (![Symbol][img2] ![Symbol][img1])  unten rechts auf den Karten.

Anhand von selbst festgelegten Kriterien können Sie nach bestimmten Firmen suchen. Außerdem können Sie Dokumente verfassen, Folgeaufgaben erstellen und neue Firmen und Personen einrichten.

## Bereichsregisterkarten auf Firma

> [!NOTE]
> [Personen][1], die nicht zu einer Firma gehören, finden Sie in der Ansicht Person.

Am unteren Rand der Ansicht befinden sich die folgenden [Standard-Bereichsregisterkarten][8]:

* Personen
* Beziehungen
* Projekte
* Aktivitäten
* Verkäufe
* Anfragen
