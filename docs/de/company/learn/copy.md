# Firmen kopieren

[!include[Requirement](../../learn/includes/note-req-manage-entities.md)]

In bestimmten Situationen ist es hilfreich, die Informationen einer bereits registrierten Firma für eine neue Firma zu kopieren. Zum Beispiel: Eine bereits registrierte Firma hat eine neue Niederlassung eröffnet. Die neue Niederlassung nutzt nahezu dieselben Informationen wie für die bereits vorhandene Firma.

> [!NOTE]
> Aktivitäten, Verkäufe, Interessen, Projekt- und Selektionszugehörigkeiten werden nicht kopiert.

## Zum Kopieren der Firma

[!include[Go to company](../../learn/includes/goto-company.md)]

1. Wählen Sie **Firmen verwalten** vom ![Symbol][img1] der Schaltfläche **Aufgabe** aus und klicken Sie auf **Firma kopieren**.
    Das Dialogfeld **Firma kopieren** wird geöffnet. Die Quellfirma wird unter **Von** angezeigt.

1. Geben Sie unter **An** den Namen der neuen Firma und ggf. eine Abteilung ein.

1. Wählen Sie unter **Personen mit allen Informationen kopieren** aus:

    * **Nicht kopieren**: Wenn Sie diese Option wählen, werden die Angaben zur Firma kopiert, nicht aber die Personen.
    * **Kopieren**: Wenn Sie diese Option wählen, werden die Angaben zur Firma und die Personen kopiert.

    [!include[Preview results](../../learn/includes/note-preview-results.md)]

1. Klicken Sie auf **Kopieren**. Die neue Firma wird mit den angegebenen Informationen registriert und mit einer eigenen Nummer und einem eigenen Code versehen. Anschließend können Sie bei Bedarf die [Firmeninformationen bearbeiten][2].
