# Eine Firma kopieren

[!include[Requirement](../../learn/includes/note-req-manage-entities.md)]

Sie können eine bestehende Firma kopieren, um eine neue zu erstellen. Wenn beispielsweise eine Firma eine neue Niederlassung eröffnet, können Sie die meisten ihrer Informationen für den neuen Datensatz wiederverwenden.

> [!NOTE]
> Aktivitäten, Verkäufe, Interessen, Projekt- und Selektionszugehörigkeiten werden nicht kopiert.

## Schritte

1. Klicken Sie auf die Schaltfläche **Aufgabe** (<i class="ph ph-dots-three-circle-vertical" aria-hidden="true"></i>) in der oberen rechten Ecke der Firmenkarte.
    Wählen Sie **Firmen verwalten** > **Firma kopieren** im Menü aus.
    Das Dialogfeld **Firma kopieren** wird geöffnet, wobei die Quellfirma unter **Von** angegeben ist.

1. Geben Sie im Abschnitt **An** den Namen der neuen Firma ein und fügen Sie ggf. eine Abteilung hinzu.

1. Wählen Sie unter **Personen mit allen Informationen kopieren** eine der folgenden Optionen aus:

    * **Nicht kopieren:** Nur Firmendetails werden kopiert.
    * **Kopieren:** Firmendetails und Personen werden kopiert.

1. Klicken Sie auf **Kopieren**. Die neue Firma wird mit eigener Nummer und eigenem Code erstellt. Sie können die [Firmendaten bearbeiten][2], falls erforderlich.
