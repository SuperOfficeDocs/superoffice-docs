# Projekttyp hinzufügen

[!include[Note difference](includes/different-edit-list-item-dialog.md)]

[!include[Edit list item](includes/edit-list-item.md)] wenn die Listenobjekte Projekttypen sind:

1. Geben Sie den Objektnamen in das Feld **Name** ein.

    > [!NOTE]
    > [!include[Mandatory](includes/note-mandatory-field.md)]

2. Geben Sie in das Feld **Geschätzte Zeit für diesen Projektfortschritt** eine Dauer für das Projekt ein (Anzahl der Wochen, Tage oder Monate). Wenn Sie ein neues Projekt für diesen Projekttyp erstellen, wird das Enddatum relativ zu der geschätzten Dauer ermittelt, die Sie hier eingeben (aktuelles Tagesdatum + geschätzte Zeit).

3. Wählen Sie in der Liste **Für diesen Projekttyp verwendete Status** die Status aus, die Sie in diesen Projekttyp einbeziehen möchten. Dieser Schritt ist optional. Wenn Sie aber den Projekttyp mit einem Projektleitfaden verknüpfen möchten, sollten Sie zumindest die letzten zwei Status aktivieren.

    [Wie man einen Projektstatus erstellt][2].

4. Aktivieren Sie die Option **Dieser Projekttyp ist mit einem Leitfaden verknüpft**, wenn alle Projekte dieses Projekttyps einem [Projektleitfaden][3] folgen sollen.

5. Wählen Sie optional **Vorschlag, das Projekt automatisch in den nächsten Status zu verschieben** auswählen. Nachdem alle Folgeaufgaben und Dokumente für einen Status abgeschlossen sind, werden Sie gefragt, ob Sie zum nächsten Status gehen möchten.

6. Geben Sie eine Beschreibung des Projekttyps ein.

7. Klicken Sie auf **Speichern**. Der Projekttyp wird als eigenständiges Objekt zur Liste **Objekte** hinzugefügt. Wiederholen Sie diesen Vorgang, wenn Sie weitere Projekttypen hinzufügen möchten.
