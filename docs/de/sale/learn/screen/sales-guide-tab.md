# Die Bereichsregisterkarte Verkaufsleitfaden

Wenn Sie einen Verkauf erstellen und einen Verkaufstyp auswählen, wird die Bereichsregisterkarte **Verkaufsleitfaden** und darunter die Aktivitätenliste angezeigt.

Die Schaltflächen für die unterschiedlichen Stufen werden oben auf der Bereichsregisterkarte angezeigt. Das Erscheinungsbild der Schaltflächen ändert sich, wenn Sie die einzelnen Stufen im Verkaufsleitfaden abschließen:

| Stufe | Symbol |
|---|---|
| Stufe Erledigt | ![Symbol][img1] |
| Aktuelle Stufe | ![Symbol][img2] |
| Zukünftige Stufe | ![Symbol][img3] |

Jede Stufe im Verkaufsleitfaden enthält unterschiedliche Folgeaufgaben und/oder Dokumente, die in der unteren Hälfte der Bereichsregisterkarte angezeigt werden. Sie sind in zwei Spalten eingeteilt: **Folgeaufgaben** und **Dokumente**. Wenn Sie eine Sekunde lang auf eine Schaltfläche klicken, zeigt die untere Hälfte der Bereichsregisterkarte an, welche Aktivitäten zu der Stufe gehören.

Klicken Sie auf **Erstellen** neben den Folgeaufgaben und Dokumenten, um diese zu erstellen. Sie können zum Erstellen auch auf den Titel der Folgeaufgabe oder des Dokuments klicken.

![Folgeaufgabe erstellen -screenshot][img4]

> [!TIP]
> Alle Folgeaufgaben und Dokumente, die Sie für einen Verkaufsleitfaden erstellt haben, werden auch in der Ansicht Verkauf auf der Bereichsregisterkarte **Aktivitäten** angezeigt. Dadurch erhalten Sie einen Überblick über alle – erledigten und nicht erledigten – Aktivitäten, die mit dem Verkauf verknüpft sind.

Nachdem eine Folgeaufgabe oder ein Dokument erstellt wurde, ist ein späteres Löschen oder Bearbeiten möglich, indem Sie mit der rechten Maustaste auf den Titel klicken und die entsprechende Option aus dem daraufhin angezeigten Menü auswählen. Auf die gleiche Weise können Sie auch eine neue Folgeaufgabe oder ein neues Dokument des gleichen Typs erstellen.

![Folgeaufgabe erstellen -screenshot][img5]

## Alle Folgeaufgaben und Dokumente anzeigen

Standardmäßig werden in der unteren Hälfte des Fensters die Aktivitäten für die aktuellen Verkaufsstufen angezeigt. Sie können jedoch festlegen, dass alle Aktivitäten und Dokumente für sämtliche Stufen angezeigt werden, indem Sie unten rechts auf der Bereichsregisterkarte die Option **Alle Stufen anzeigen** auswählen. Sie können außerdem die Folgeaufgaben und Dokumente für eine bestimmte Stufe anzeigen, indem Sie auf die Schaltfläche mit der Bezeichnung der entsprechenden Stufe klicken.

Sie können zur nächsten Stufe wechseln, indem Sie diese aus der Liste **Stufe** auf der Hauptkarte auswählen.

Ein Verkaufsleitfaden, in dem nur Folgeaufgaben und Dokumente für die aktuelle Stufe angezeigt werden:

Ein Verkaufsleitfaden, in dem alle Folgeaufgaben und Dokumente für alle Stufen angezeigt werden:
