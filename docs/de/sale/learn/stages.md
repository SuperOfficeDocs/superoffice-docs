# Verkäufe mit dem Status aufgeschoben, verkauft und verloren

Im Feld **Stufe** in der Ansicht Verkauf können Sie dem Verkauf den Status **Verkauft** oder **Verloren** zuweisen. Sie können auch das Kontrollkästchen **Aufgeschoben** unter diesem Feld aktivieren, wenn Sie den Verkaufsprozess für eine Weile zurückstellen möchten. In diesem Fall ändern sich die Felder darunter sowie das Symbol oben links in der Ansicht Verkauf.

Für individuelle Verkaufstypen wird auch eine Liste der Stufen angezeigt,[!include[SM](../../learn/includes/as-defined-sm.md)] [Weitere Infos über Verkaufsleitfäden][1].

## Verzögert

Wenn Sie die Verkäufe auf **Aufgeschoben** setzen, werden folgende Felder angezeigt:

* **Wiedereröffnungsdatum**: Hier können Sie das Datum eingeben, an dem der Verkauf wieder geöffnet werden soll.
* **Grund (aufgeschoben)**: Hier können Sie einen Grund für die Verzögerung eingeben.

![Verzögerter Auftrag -screenshot][img5]

Es öffnet sich das Dialogfeld **Termin**, wenn Sie auf **Speichern** klicken. Hier können Sie ein zukünftiges Gespräch oder einen Anruf einrichten, um den aufgeschobenen Verkauf zu verfolgen.

Wenn Sie dem Verkauf den Status **Verkauft** oder **Verloren** zuweisen, können Sie das Datum, die anderen Wettbewerber und das Gelingen/Fehlschlagen des Verkaufs eintragen.
