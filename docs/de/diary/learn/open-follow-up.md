# Folgeaufgaben öffnen

Sie können eine bereits eingetragene Folgeaufgabe öffnen, indem Sie darauf doppelklicken:

<!-- markdownlint-disable MD032 -->
[!include[List of Activities section tabs](../../learn/includes/list-activities-section-tabs.md)]
* Auf den verschiedenen Registerkarten der Ansicht Kalender.
<!-- markdownlint-restore -->

Je nach Art der Folgeaufgabe wird das Dialogfeld **Termin**, **Aufgabe** oder **Anruf** geöffnet.
