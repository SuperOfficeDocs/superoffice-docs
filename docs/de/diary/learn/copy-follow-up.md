# Folgeaufgabe in den Kalender kopieren

Sie können eine Folgeaufgabe im Kalender zu einer anderen Uhrzeit bzw. einem anderen Tag kopieren. Gehen Sie dazu wie bei [Folgeaufgabe verschieben][1] vor. Halten Sie jedoch beim Kopieren die **STRG**-Taste gedrückt, während Sie Ziehen und Ablegen.
