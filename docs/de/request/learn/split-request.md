# Anfrage aufteilen

Wenn eine Anfrage mehrere Anliegen enthält – beispielsweise eine technische Frage und eine Rechnungsanfrage – können Sie die Anfrage aufteilen und daraus eine neue Anfrage erstellen.

> [!TIP]
> Sie können auch eine [Nachricht aufteilen][1].

## Schritte

1. [Öffnen Sie die Anfrage][2].

1. Klicken Sie auf <i class="ph ph-dots-three-circle-vertical" aria-label="Aufgabenmenü"></i> und wählen Sie **Aufteilen**.

    Es öffnet sich ein Dialogfeld, in dem Sie Informationen zur neuen Anfrage angeben.

1. Gehen Sie zur Registerkarte **Eigenschaften der neuen Anfrage** und füllen Sie die relevanten Felder aus.

1. Geben Sie im Feld **Titel** eine aussagekräftige Bezeichnung für die neue Anfrage ein.

1. Wählen Sie eine passende **Kategorie** für die neue Anfrage aus.

1. Wählen Sie im Feld **Priorität** eine der folgenden Optionen:
    * **Niedrig**, **Mittel** oder **Hoch** – je nach Dringlichkeit der Anfrage.
    * Alternativ kann die Priorität automatisch anhand definierter [Eskalationsstufen][3] für Kunden oder Firmen gesetzt werden.

1. Wählen Sie im Feld **Zugriffsstufe** eine der folgenden Optionen:
    * **Extern**: Der Kunde kann die Anfrage über das Kundenzentrum einsehen.
    * **Intern**: Der Kunde hat keinen Zugriff auf die Anfrage.

1. Wählen Sie im Feld **Status** eine der folgenden Optionen:
    * **Offen**: Die Anfrage wird aktiv bearbeitet.
    * **Geschlossen**: Die Anfrage ist abgeschlossen.
    * **Auf Termin gelegt**: Die Bearbeitung der Anfrage wird aufgeschoben. Wählen Sie ein Datum, an dem die Anfrage automatisch auf **Offen** zurückgesetzt wird.

1. Wechseln Sie zur Registerkarte **Nachrichten**, wo eine Liste der Nachrichten innerhalb der Anfrage angezeigt wird. Wählen Sie die Nachrichten aus, die in die neue Anfrage verschoben werden sollen.

1. Klicken Sie auf **OK**. Die Anfrage wird aufgeteilt, und die neue Anfrage wird erstellt.
