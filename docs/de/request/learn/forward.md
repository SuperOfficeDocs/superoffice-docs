# Anfrage an externen Kontakt weiterleiten

Als Bearbeiter einer Anfrage können Sie nicht immer selbst eine Antwort geben – selbst mit der Unterstützung Ihrer Kollegen. In manchen Fällen müssen Sie externe Hilfe einholen, z. B. von einem Lieferanten, Partner oder Branchenspezialisten.

Sie können eine komplexe oder problematische Anfrage per E-Mail an eine externe Person weiterleiten. Die Antwort dieser Person wird automatisch in der Anfrage in SuperOffice gespeichert. Der Kunde erhält keine dieser E-Mail-Kommunikationen. Sobald Sie die Lösung haben, können Sie dem Kunden direkt in der Anfrage antworten.

## Mitteilung weiterleiten

1. [Öffnen Sie die Anfrage][1] und wechseln Sie zur Registerkarte **Mitteilungen**.

1. Klicken Sie auf das Symbol **Weiterleiten** (<i class="ph ph-arrow-bend-up-right" aria-label="Weiterleiten-Symbol"></i>) oder öffnen Sie das **Aufgabenmenü** (<i class="ph ph-dots-three-circle-vertical" aria-label="Aufgabenmenü"></i>) in der Mitteilung, die Sie weiterleiten möchten, und wählen Sie **Weiterleiten**. Eine neue Mitteilung öffnet sich im Bearbeitungsmodus.

    ![Weiterleiten einer Anfrage an eine externe Person -screenshot][img3]

1. Geben Sie im Feld **An** den Empfänger ein. Klicken Sie auf **Cc/Bcc**, um Empfänger in Kopie oder Blindkopie hinzuzufügen.

    ![Empfänger für Weiterleitung hinzufügen -screenshot][img2]

1. [Überarbeiten Sie den Mitteilungentext][2], falls erforderlich.

1. Klicken Sie auf <i class="ph ph-paperclip" aria-label="Anhänge"></i> in der Werkzeugleiste, um relevante Dokumente als Anhang hinzuzufügen.

1. Klicken Sie auf **Senden**, um die Mitteilung weiterzuleiten.

## Mehrere Anfragen weiterleiten

1. Öffnen Sie eine Liste oder gespeicherte Selektion mit den gewünschten Anfragen und wählen Sie die Anfragen aus.

1. Öffnen Sie das **Aufgabenmenü**:
    * **Gespeicherte Selektion:** Klicken Sie auf <i class="ph ph-dots-three-circle-vertical" aria-label="Aufgabenmenü"></i> und wählen Sie **Massenweiterleitung**.
    * **Anfragen suchen:** Klicken Sie auf das <i class="ph ph-list" aria-hidden="true"></i> **Aufgaben**-Symbol in der unteren linken Ecke und wählen Sie **Anfragen weiterleiten**.

1. Fügen Sie den Empfänger im Feld **An** hinzu. Optional können Sie über **Cc/Bcc** weitere Empfänger hinzufügen.

1. (Optional) Aktivieren Sie **Nach Weiterleitung schließen**, wenn die weitergeleiteten Anfragen automatisch als "geschlossen" markiert werden sollen.

1. Unter **Kommentar** können Sie eine Anmerkung hinzufügen, die sowohl in die E-Mail aufgenommen als auch als interne Mitteilung zur Anfrage gespeichert wird.

1. Klicken Sie auf **Weiterleiten**, um die Anfragen zu senden.
