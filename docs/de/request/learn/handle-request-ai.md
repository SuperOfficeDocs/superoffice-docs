# Effizientes Bearbeiten von Anfragen mithilfe von KI

> [!NOTE]
> Für KI Services ist eine separate Lizenz erforderlich. SuperOffice AI steht nur in der Cloud zur Verfügung und ist ein separates Add-on.

Die Einführung von Künstlicher Intelligenz (KI) in SuperOffice 10 bietet Bearbeitern von Anfragen/Supportmitarbeitern ein neues Set intelligenter Tools für einen effizienteren Arbeitstag.

## Spracherkennung

Mit dieser Funktion wird die Sprache einer eingehenden Anfrage automatisch erkannt. Die erkannte Sprache wird unter der Nachricht angezeigt.

## Übersetzung

Wenn Sie eine Mitteilung in einer Sprache erhalten, die Sie nicht lesen können, können Sie unter der Mitteilung auf **Übersetzen aus** ***Sprache*** klicken. Betreff und Text der Nachricht werden übersetzt und unter der ursprünglichen Nachricht hinzugefügt.

## Sentiment-Analyse

Ist die Person glücklich oder wütend?

* Negatives <i class="ph ph-smiley-sad" aria-hidden="true"></i>
* Neutrales <i class="ph ph-smiley-meh" aria-hidden="true"></i>
* Positives <i class="ph ph-smiley" aria-hidden="true"></i>

Wenn diese Funktion aktiv ist, werden eingehende E-Mail-Anfragen analysiert, um die Stimmung der Person festzustellen. Ein Emoticon unter der Nachricht zeigt die erkannte Stimmung (zufrieden, unzufrieden oder neutral) an.

### Automatische Kategorisierung

Lassen Sie das System Anfragen für Sie kategorisieren und sparen Sie damit Zeit. Wenn diese Funktion aktiv ist, werden eingehende E-Mail-Anfragen analysiert, um für jede Anfrage die beste Kategorie zu ermitteln. In der Fußzeile einer Anfrage wird die vom System vorgeschlagene Kategorie angezeigt. Klicken Sie auf den Link für die Kategorie, um die Anfrage mit der vorgeschlagenen Kategorie zu aktualisieren.
