# Mit Antwortvorlagen arbeiten

Eine Antwortvorlage kann beispielsweise ein bestimmtes Layout für Antworten oder ein vorgefertigter Text sein, der in eine Nachricht eingefügt wird. Sie kann auch eine Anlage enthalten, die der Nachricht hinzugefügt wird. Bevor Sie die Antwort an den Kunden senden, können Sie den Text bearbeiten und Anlagen hinzufügen oder entfernen. Mit anderen Worten, eine Antwortvorlage bietet die Grundlage für eine individuell angepasste Kundenantwort. Die Verwendung von Antwortvorlagen spart Zeit und ermöglicht dennoch eine Anpassung an die jeweilige Situation.

## Antwortvorlagen suchen und anzeigen

Antwortvorlagen sind in einer Ordnerstruktur organisiert, die Sie selbst erstellen und verwalten können. Um eine Antwortvorlage zu finden und anzuzeigen:

1. Gehen Sie zum <i class="ph ph-list" aria-label="Hauptmenü"></i> in der oberen Leiste und wählen Sie **Wissensdatenbank** > **Antwortvorlagen**.
2. Navigieren Sie zum gewünschten Ordner.
3. Klicken Sie auf die Antwortvorlage, die Sie ansehen möchten. Die Ansicht **Antwortvorlagen anzeigen** wird geöffnet und zeigt:
    * Informationen zu Zugriffsebene, Sprache und Eigentümer.
    * Registerkarten mit dem eigentlichen Antwortvorlagentext und allen Anhängen (pro Sprachversion).

### Nach Antwortvorlagen suchen

In der Übersicht der Antwortvorlagen können Sie mit der Schaltfläche **Suchen** eine bestimmte Antwortvorlage finden:

1. Geben Sie den Text oder die Kriterien ein, nach denen Sie suchen möchten.
2. Klicken Sie auf **Suchen**. Eine Liste mit passenden Vorlagen wird angezeigt.
3. Klicken Sie auf eine Vorlage, um sie zu öffnen und anzuzeigen.

> [!TIP]
> Um die Suche nach Vorlagen in Zukunft zu erleichtern, sollten Sie sie in [Ordnern organisieren][1].
