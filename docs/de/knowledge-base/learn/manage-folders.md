# Ordner verwalten

Organisieren Sie Antwortvorlagen, FAQ-Einträge, E-Mail-Adressen und externe Dokumente in einer logischen Ordnerstruktur innerhalb der Wissensdatenbank, um Informationen leichter zu finden. Sie können auf jeder Ebene Unterordner erstellen.

## Ordner erstellen

So erstellen Sie einen neuen Ordner:

1. Wechseln Sie zu dem Ordner, unter dem Sie einen Unterordner erstellen möchten.
1. Klicken Sie oben in der Ansicht auf **Neuer Ordner**.
1. Das Feld **Ordner** zeigt automatisch den Namen des übergeordneten Ordners an. Sie können hier einen anderen Ordner auswählen.
1. Geben Sie einen aussagekräftigen Namen in das Feld **Name** ein.
1. Geben Sie im Feld **Beschreibung** eine kurze Beschreibung des Inhalts ein.

1. Zusätzliche Schritte für **FAQ-Ordner**:

    * Wählen Sie in dem Feld **Position** eine Nummer für die Position des Eintrags in der Sortierreihenfolge des entsprechenden Ordners. 1 bedeutet ganz oben in der Liste.
    * Wählen Sie in dem Feld **Zugriff**, wer Zugriff auf den betreffenden Ordner erhalten soll. Bei einem registrierten Kunden handelt es sich um einen Kunden, der Zugriff auf ein Kundenzentrum hat.
    * Im Feld **Zugriffskontrolle** können Sie ein Skript zur Kontrolle des Zugriffs externer Benutzer auf diesen Ordner eingeben. Wenn ein Skript vorhanden ist, wird es ausgewertet. Der Zugriff wird dann nur gestattet, wenn das Skript den Wert "1" zurückgibt.

1. Zusätzliche Schritte für **externe Dokumente**:

    * Im Listenfeld **Zugriff** können Sie wählen, wer Zugriffsrechte zum Lesen und Bearbeiten des Ordners erhalten soll.

1. Klicken Sie auf **OK**, um den Ordner zu erstellen.

Sie können nun Elemente zur Wissensdatenbank hinzufügen oder den Wert im Feld **Ordner** bestehender Elemente ändern.

## Ordner löschen

**Bevor Sie einen Ordner löschen, entfernen Sie alle enthaltenen Elemente** durch:

* Löschen der Elemente.
* Verschieben der Elemente in einen anderen Ordner (ändern Sie das Feld **Ordner** in der Bearbeitungsansicht jedes Elements).

> [!NOTE]
> Wenn Sie einen Ordner mit **externen Dokumenten** löschen, werden alle Dateien darin sofort gelöscht; Sie müssen ihn nicht zuerst leeren.

So löschen Sie einen Ordner:

1. Wechseln Sie zum gewünschten Ordner.
1. Klicken Sie auf **Ordner bearbeiten**.
1. Klicken Sie auf **Löschen**. Der Ordner wird sofort gelöscht.
