# Angebot erstellen

Befolgen Sie das GIF-Bild oder die folgenden Schritte, um zu sehen, wie Sie ein Angebot erstellen und senden:

![Nachdem Sie auf die Schaltfläche Angebot erstellen geklickt haben, können Sie damit beginnen, Ihre Produkte und Dienstleistungen zum Angebot hinzuzufügen -screenshot][img1]

## Schritte

1. Sie müssen einen [neuen Verkauf erstellen][1] oder einen vorhandenen Verkauf öffnen (einen, der nicht verkauft oder verloren ist).

2. Klicken Sie auf die Bereichsregisterkarte **Angebot**.

3. Klicken Sie auf **Ein neues Angebot erstellen** oder auf **Angebot aus einem anderen Verkauf kopieren**. Das Dialogfeld **Angebot bearbeiten** wird angezeigt.

    Sie können jetzt [Produkte zum Angebot hinzufügen][2].

## Wird das Dialogfeld ERP-Verbindung auswählen angezeigt?

Wenn das Dialogfeld **ERP-Verbindung auswählen** angezeigt wird, wenn Sie ein Angebot erstellen, kann dies folgende Gründe haben:

* Es sind keine Preislisten für die Währung des Verkaufs verfügbar. Sie müssen eine andere Währung für den Verkauf wählen oder eine Preisliste mit der gewünschten Währung in Einstellungen und Verwaltung erstellen.

    > [!NOTE]
    > Wenn die Liste der Währungen nicht neben dem Verkaufspreis angezeigt wird, müssen Sie vielleicht unterschiedliche Währungen zulassen aktivieren. Gehen Sie zu **Einstellungen** &gt; **Unterschiedliche Währungen zulassen** in Einstellungen und Verwaltung.

* SuperOffice CRM ist mit einem oder mehreren ERP-Systemen verbunden und es gibt mehrere relevante ERP-Verbindungen, aus denen gewählt werden kann. Die ERP-Verbindung bestimmt, von wo die Produktinformationen und die Preise bezogen werden.

Nachdem Sie eine ERP-Verbindung ausgewählt haben, können Sie mit der Erstellung eines Angebots fortfahren.
