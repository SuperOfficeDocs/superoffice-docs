Um die Selektion zu den Favoriten hinzuzufügen, klicken Sie auf das Sternsymbol ![Symbol][img1]. Der Stern wird daraufhin in Gelb angezeigt ![Symbol][img2] und die Selektion wird sofort zu den Favoriten hinzugefügt. Jetzt können Sie schnell auf die Selektion zugreifen, indem Sie im Navigationsmenü auf ![Symbol][img3] klicken.
