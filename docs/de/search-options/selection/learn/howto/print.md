# Selection drucken

1. [!include[Start](../includes/steps-start-task.md)]

1. Richten Sie die ersten fünf Spalten ein, die Sie einbeziehen möchten. Gruppierungen oder Spaltenberechnungen, die auf Ihre Selection angewendet werden, erscheinen im gedruckten Bericht.

1. Klicken Sie auf die **Aufgabe**-Schaltfläche und wählen Sie **Selection drucken**.

1. Im Fenster, das den zu druckenden Inhalt anzeigt, klicken Sie auf **Drucken**.
