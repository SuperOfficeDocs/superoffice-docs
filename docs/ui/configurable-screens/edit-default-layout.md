# Edit the default layout

[!include[Requirement](../../../common/includes/req-dev-tools.md)]

Modify the system layout of one or more screens for all users to fit the specific needs of your organization.

* No pixel layout in preview

"Tip: If you later want to edit the layout settings, double-click the layout in the list to open the Edit layout settings dialog. Please note that you cannot edit the settings for the default layout."
