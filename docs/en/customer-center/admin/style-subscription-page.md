# Style the subscription page

The subscription page allows contacts to manage which mailings they want to receive from your company. The page is linked from:

* The privacy confirmation email
* All marketing mailings

You can customize the design and text of the subscription page to align it with your brand identity.

## Where to find it

1. Go to **Settings and maintenance**.
1. Select <i class="ph ph-target" aria-hidden="true"></i> **Marketing**.
1. Open the **Subscription page styling** tab.
1. Make your changes in the **Styling** or **Options** section.
1. Click **Save**.

You can make changes in two sections: **Styling** and **Options**. A preview is shown on the right-hand side. Use the **Choose preview** dropdown to check how your changes look on the Manage subscriptions, Unsubscribe, Confirmation saved, and Confirmation cancelled pages.

![Subscription page styling in Settings and maintenance -screenshot][img1]

## What you can customize

### Styling

Use the **Styling** section to adjust the visual design of the subscription page.

* **Logo**
  * Upload a company logo.
  * Choose the logo position: left, center, or right.

* **Layout**
  * Set the page width.

* **Background**
  * Set a page background color and background image.
  * Set a content background color and background image.

* **Buttons and toggles**
  * Set the button color.
  * Set the subscription toggle color.

* **Typography**
  * Choose font, font size, and color for text.
  * Choose font, font size, and color for button text.

Changes are reflected immediately in the preview. Click **Save** to publish them.

### Options

Use the **Options** section to customize text for each supported language.

1. Select a language from the list.

1. Select the page you want to update. A preview appears automatically on the right.

1. Edit the available text fields.

![Subscription page Options tab in Settings and maintenance -screenshot][img2]

If you clear a field, the default SuperOffice text is restored automatically.

You can also remove languages that you do not use. This limits the languages available on the subscription page and reduces maintenance. You can add removed languages again later if needed.

Changes are not applied until you click **Save**.
