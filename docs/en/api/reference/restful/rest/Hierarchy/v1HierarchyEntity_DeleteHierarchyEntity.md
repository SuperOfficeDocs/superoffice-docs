# DEL Hierarchy/{id}

```http
DELETE /api/v1/Hierarchy/{id}
```

Deletes the HierarchyEntity


Calls the List agent service DeleteHierarchyEntity.





| Path Part | Type | Description |
|-----------|------|-------------|
| id | int32 | The HierarchyEntity to be deleted. **Required** |
