# PUT List/Rating/Items/{id}

```http
PUT /api/v1/List/Rating/Items/{id}
```

Updates the existing SaleStageEntity

| Path Part | Type | Description |
|-----------|------|-------------|
| id | int32 | The id of SaleStageEntity to be saved. **Required** |
