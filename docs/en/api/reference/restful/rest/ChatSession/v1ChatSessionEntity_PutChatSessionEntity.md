# PUT ChatSession/{id}

```http
PUT /api/v1/ChatSession/{id}
```

Updates the existing ChatSessionEntity






| Path Part | Type | Description |
|-----------|------|-------------|
| id | int32 | The ChatSessionEntity id to update. **Required** |
