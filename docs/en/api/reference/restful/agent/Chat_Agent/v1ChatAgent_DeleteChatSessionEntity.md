# POST Agents/Chat/DeleteChatSessionEntity

```http
POST /api/v1/Agents/Chat/DeleteChatSessionEntity
```

| Parameter Name | Type |  Description |
|----------------|------|--------------|
| ChatSessionEntityId | int32 | **Required** The id of the ChatSessionEntity to be deleted. |
