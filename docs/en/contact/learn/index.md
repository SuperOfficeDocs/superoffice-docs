# Contacts

[!include[Intro](includes/contacts-screen-intro.md)]

The bottom of the **Contact** screen contains five default section tabs:

* **Activities**
* **Sales**
* **Requests**
* **Projects**
* **Relations**

Everything you want to find out about a contact is available from a single screen.
