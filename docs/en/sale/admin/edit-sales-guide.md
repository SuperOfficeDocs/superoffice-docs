# Edit or deactivate a sales guide

[!include[Must be admin](../../learn/includes/req-admin.md)]

## Edit a sales guide

1. Select <i class="ph ph-arrows-left-right" aria-hidden="true"></i> **Workflow** in the Navigator.
1. Select the **Sales guide** tab.
1. Select the required type from the **Sale type** list.
1. Click **Edit**.
1. [Edit the individual stages][3].
1. Edit follow-ups and documents by double-clicking them in the lists at the bottom.

   * [Add a follow-up for a stage][1]
   * [Add a document for a stage][2]

   The changes are saved automatically.

## Deactivate a sales guide

To remove a sales guide from a [sale type][5], follow these steps:

1. Select <i class="ph ph-list-bullets" aria-hidden="true"></i> **Lists** in the Navigator.
1. Select **Sale - Sale type (for stages)** from the dropdown menu. The **Items** tab displays all existing sale types.
1. Double-click the sale type linked to the guide you want to deactivate. The **Edit list item** dialog opens.
1. Uncheck **This sale type is linked to a guide**.
1. (Optional) Update other fields as needed.
1. Click **Save** to apply your changes or **Cancel** to discard them.

> [!NOTE]
> If you later want to reuse the sales guide, recheck the option. Any follow-ups and documents previously added to the sales guide's stages will be automatically retrieved when reactivated. You can also add new follow-ups or documents, or remove any that are no longer relevant.
