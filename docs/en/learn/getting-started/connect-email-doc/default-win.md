# SuperOffice as your solution (Windows)

This is the default way to connect your email account, documents, and calendar to SuperOffice.

> [!NOTE]
> These instructions are for Windows only. Are you using a Mac? See the [instructions for macOS][1].

## Email ![icon][img1]

[!include[Why](includes/why-connect-email.md)]

Choose one of the following options:

* [WebTools][3] (default)

* [SuperOffice Inbox][2] - if you are unable to use WebTools.

    [!include[ALT](includes/set-up-inbox.md)]

* [!include[ALT](includes/why-superoffice-for-outlook.md)]

    [!include[Go to App Store](../../../email/superoffice-for-outlook/learn/includes/install.md)]

## Document ![icon][img2]

The default document integration is **SuperOffice document library** with [WebTools][3].

> [!NOTE]
> If you chose WebTools for email in the previous step, you are already done.
