# Close requests

To close a request means changing its [status][2] to *Closed*.

## Close one request

After [responding to the request][3], select **Closed** in the **Status** field in the sidebar properties.

## Close multiple request from a list

1. Select all requests that need to be closed.
1. Right click to get the context menu and select **Bulk update**.
1. Follow the guide for [Update multiple records (bulk update)][1] and select **Request - Status** as one of the fields to update.

## Use selection to close on multiple requests

1. Create a selection of all requests you want to close.
1. Click <i class="ph ph-list" aria-label="Task menu"></i> and select **Bulk update**.
1. Follow the guide for [Update multiple records (bulk update)][1] and select **Request - Status** as one of the fields to update.
