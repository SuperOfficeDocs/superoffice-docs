# Close requests

After responding to the request, you can set its status to **Closed** by selecting **Closed** in the **Status** field in the sidebar properties.

## Close multiple request

When you need to close multiple requests you have several options:

1. Select all requests that needs to be closed.
1. Right click to get the context menu and select **Bulk update**.
1. Follow the guide for [Update multiple records (bulk update)][1] and select **Request - Status** as one of the fields to update.

You can also:

1. Create a selection for all your requests you want to close.
1. Click ![icon][img1] **Task** button and select **Bulk update**.
1. Follow the guide for [Update multiple records (bulk update)][1] and select **Request - Status** as one of the fields to update.
