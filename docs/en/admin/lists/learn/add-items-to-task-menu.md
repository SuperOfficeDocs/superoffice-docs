# Add items to the Task menu list

[!include[Edit list item](includes/edit-list-item.md)] when you are adding an item to the **Project - Status** list:

1. Enter the name and description of the item in the **Name** and **Description** field.

    > [!include[Mandatory](includes/note-mandatory-field.md)]

2. Select the wanted item in the **Task menu** field.

3. Select between the types for URL, CRMScript or SoProtocol.

4. Add the different fields connected to the different types, like URL, CRMScript and SoProtocol.

5. In the URL encoding list box, you can select if the URL should be ANSI or Unicode (or no format).

6. Choose the different "Show xx" on where you would like to show the new task.

7. [!include[Save or cancel](includes/save-or-cancel.md)]
