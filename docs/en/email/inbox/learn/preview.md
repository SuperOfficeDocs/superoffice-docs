# View email messages

When you select a message in the inbox, a preview of the message is displayed in right part of the screen.

The **Preview** tab contains the following fields:

| Field | Description |
|---|---|
| Subject | Shows the subject of the email. |
| From | Shows the sender. If the sender is recorded in SuperOffice CRM, you can hold the mouse-pointer over the name to view the contact card. |
| Date | Shows the date and time of the message. |
| To | Shows the recipient(s) of the message. |
| Attachments | Displays any attachments to the message. |

> [!NOTE]
> Go to <i class="ph ph-user-circle" aria-hidden="true"></i> **Personal settings** > **Email options**. Open the **General** tab. If you check **Show inline images**, inline images are displayed in preview.
